## 一、转债和股票规则对比

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240311014340342.png)

|                  | 股票                        | 可转债                            |
| ---------------- | --------------------------- | --------------------------------- |
| 交易时间         | T+1（当天买下一工作日可卖） | T+0（当天买当天可卖）             |
| 涨跌幅限制       | 非首日±10%                  | 非首日±20%                        |
| 印花税           | 收（卖出时收取0.1%）        | 不收                              |
| 交易单位         | 1手（1手100股）             | 沪市为1手，深市为10张（1手=10张） |
| 交易价格最小单位 | 0.01元                      | 0.001元                           |

可转债最小交易单位为1手，1手=10张，每张的价格就是大家看到的价格。比如下面的正裕转债，1张的价格就是123.434元，1手的价格就是1234.34元。买入最少要花的资金就是债价*10元。

这和股票不一样，股票的1手是100股股票。买入最少要花的资金就是股价*100元。

另外新规中沪深两市的可转债最小价格变动单位都是0.001元。

## 二、交易权限开通

要开通可转债交易权限，得满足两大条件：**两年交易经验+20日日均10万资产。**

首先，你得有至少两年的交易经验，这个是从你第一次买股票开始算的，不管你是销户了还是换券商了，这个经验都不会清零。

其次，你还需要最近20天里，每天平均有10万块钱的资产，这个资产可以是股票、基金、余额理财，啥都行，不用非得买股票。但记住啊，这个资产在不同券商是不通用的。

如果你在2022年6月18日之前就开通了可转债权限，那就不用担心了，你还是能正常打新和交易可转债的。你要是不知道自己有没有这个权限，可以试试去打新或者交易一下，能操作就说明你有权限。

还有啊，你在A券商开通的权限，到B券商那儿可就不算数了。在B券商还得再满足一次那两个条件。交易经验是累加的，但那个20天日均10万资产得在B券商重新算。比如说，你在B券商放200万块钱一天，或者放10万块钱放20天，这样也行，也不用非得买股票。

## 三、可转债交易规则

可转债最新的交易规则如下：

1. **上市首日价格限制**：可转债上市首日，最多只能比发行价涨57.3%，或者跌43.3%。这样的设置旨在控制上市首日的价格波动，避免过度投机和市场异常波动。
2. **涨跌幅限制**：上市次日起，设置20%的涨跌幅价格限制。这一规定有助于维护市场的稳定，防止价格出现过大的波动。
3. **T+0交易制度**：可转债实行T+0交易制度，即买入的可转债在当天即可卖出。这种交易制度提高了市场的流动性，方便投资者更灵活地调整投资策略。
4. **交易时间和股票相同**：可转债的交易时间与股票市场的交易时间一致，方便投资者进行交易操作。
5. **T+1交收制度**：虽然交易是T+0，但交收是T+1，即交易达成的第二天才能完成资金和债券的交收，才能出金。
6. **强化风险提示**：在可转债最后交易日的证券简称前增加“Z”标识，以此向投资者充分提示风险，保护投资者的利益。
7. **交易成本**：可转债的交易成本通常只包括交易佣金，不需要支付印花税和过户费，这降低了投资者的交易成本。

对了，可转债的交易还设有临时停牌机制，用于应对市场异常波动情况。买卖时，最少要买10张，谁先下单、谁的价格合适，谁就先成交，即成交的顺序遵循时间优先和价格优先的原则。

这些规则可能会跟着市场情况和监管要求变，所以大家买卖可转债时，得多关注最新的规则和市场动态，这样投资才能更明智。
