## 1. 可转债估值星级系统

**定位**：基于全市场可转债价格中位数，判断转债整体估值高低，指导仓位管理和择时操作。

**核心指标**：可转债全市场价格中位数（元）

### 星级划分

| 星级 | 中位数区间 | 估值含义 | 仓位建议 |
|:---:|:---:|:---:|:---:|
| 一星 | >124元 | 高估（高危区） | 逐步减仓 |
| 二星 | 116~124元 | 中性偏高估 | ≤80% |
| 三星 | 108~116元 | 低估（性价比区） | 70~90% |
| 四星 | 100~108元 | 深度低估 | 80~100% |
| 五星 | ≤100元 | 极度低估 | 满仓 |

### 历史触发记录

- **一星（>124元）**：2022年7月（126元）、2022年8月（128元）、2025年8月（128元，历史百分位99.52%）
- **四星（100~108元）**：2024年2月2日（107.818元）、2024年9月（106.5元）
- **五星（≤100元）**：近5年未触发。历史最低2018年10月（93.132元）

### 操作原则

- 中位数突破124元进入一星高估区 → 减仓避险，浮盈多的优先减
- 中位数跌破116元进入三星低估区 → 入场/加仓
- 中位数跌破108元进入四星深度低估区 → 大幅加仓
- 2025年8月，中位数升至128元（历史94%分位），启四果断清仓转债轮动，切换至ETF轮动

**来源**：2022/05/03《中位数价格判断转债估值》、2024/02/02《转债正式进入4星级估值》、2025/03/02周报

## 2. 可转债双低策略

**定位**：通过溢价率和价格的双重约束选出"下跌空间有限、上涨空间无限"的转债，追求攻守兼备的收益。

**核心逻辑**：可转债有债底保护（到期保本），双低值越低意味着价格低+溢价率低，既有安全垫又有进攻性。

### 经典双低策略

- **公式**：双低值 = 可转债价格 + 溢价率×100
- **选债**：取双低值最小的前5只
- **轮动**：每周轮动
- **年化收益**：约47%（yyb凌波回测）

### 偏离策略（启四优化版，已暂停）

- **最终公式**：溢价率 + 34×(100 + 可转债近5天涨幅 - 正股近5天涨幅)，取前5名
- **排除**：上市不足3个月的转债
- **轮动**：每周轮动
- **回测**：年化70.76%~95.6%，最大回撤27.6%
- **暂停原因**（2023年9月）：波动太大实盘体验差、2022年8月新规后低溢价因子失效

### 建仓参考

- 双低均值 < 160：开始建仓
- 双低均值 < 155/150：逐步加仓
- 结合估值星级决定仓位上限

**来源**：2021/10/22《又发现一个比较好的策略》、2022/01/25《偏离策略优化》、2022/11/13《偏离策略名单》

## 3. 低波双低可转债策略

**定位**：在双低策略基础上叠加正股低波动率因子，追求更稳健的周轮动收益。

**核心逻辑**：低波动转债回撤更小、收益更高、名单变动小，更适合周频轮动。

### 具体规则

**排除条件**：
- 上市3天内的新债
- ST债
- 收盘价 < 100元
- 剩余年限 < 0.5年
- 剩余市值 > 10亿

**打分公式**：
```
评分 = 收盘价×2 - 转股价值×2 + 正股年化波动率
```
取评分最小的前5只。

**轮动**：每周轮动，周一对齐名单调仓。

### 回测数据

| 区间 | 年化收益 | 最大回撤 |
|:---|:---:|:---:|
| 2020.01~2023.09（周轮动） | 24.07% | 17.74% |
| 2020.01~2024.04（周轮动） | 19.33% | 17.74% |
| 2022.01~2023.09（日轮动） | 21.22% | 7.48% |

**来源**：2023/09/17《低波动双低策略》、2024/07/14周报第42期

## 4. 下修博弈策略

**定位**：提前低价埋伏有下修预期的可转债，博弈公司下修转股价带来的价格上涨。防守型策略。

**核心逻辑**：97%的可转债最终以强赎退出。当转股价远高于正股价时，公司需下修转股价促进转股。下修后转股价值提升，转债价格随之上涨。

### 下修三种情形

1. **避免回售**：临近回售期，公司为避免回售压力而下修
2. **促进转股**：公司主动下修为未来强赎做准备
3. **大股东减持**：大股东高比例配售+加杠杆，有强烈下修欲望

### 筛选标准（避免回售型）

- 剩余年限：0.6年 ≤ 剩余年限 ≤ 3年
- 价格 ≤ 115元
- 排除银行转债、大型国企
- 按双低值排名，越低越好
- 避开正股PB < 0.8的（PB太小稀释股权严重，下修动力弱）

### 筛选标准（大股东减持型）

- 上市时间 ≤ 0.6年
- 前十大持有人持有比例 > 30%
- 价格 ≤ 120元，排除银行转债、大型国企

### 建仓/卖出规则

| 价格 | 操作 |
|:---:|:---:|
| ≤115元 | 建仓 |
| ≤110元 | 加仓（"闭眼买"） |
| ≤105元 | 补仓，越跌越补 |

- **买入**：平均买入筛选后前10~15只，分散持仓
- **止盈**：每张赚10元卖出
- **回落条件单**：高于130元，回落1%，买三价卖出

### 优化排除条件

1. 排除大型国企（尤其银行转债）
2. 排除正股PB < 0.8
3. 排除正股总市值 > 100亿（历史统计：到期未强赎的转债，正股无一例外总市值超100亿）
4. 转债占正股流通市值比例较小的需谨慎

**来源**：2022/05/18《华兴下修》、2022/05/30《建仓迪贝》、2022/05/24《大跌来了，又可以买了》

## 5. ETF动量轮动策略

**定位**：基于动量效应，在不同风格ETF之间轮动，追涨杀跌、持强去弱的趋势跟踪策略。

**核心逻辑**：资金永远往更舒服的地方流，涨得多的往往还会继续涨。顺势而为，用数据替代直觉。

### 资产池

选择10个代表不同风格和风险因子的ETF，要求弱相关甚至负相关：
- 红利、成长、黄金、债券、转债、海外科技等
- 选择标准：费率低廉、跟踪效率高、大规模、流动性好
- 关键原则：相关性太高对动量策略是毒药

### 动量计算

- **绝对动量**：近20日涨幅 > 0.5%（自己有没有趋势）
- **相对动量**：20日、60日均线涨幅排名（跟其他资产比谁更强）
- **均线系统**：沪深300用5日&60日均线，创业板用2日&31日均线，金叉买、死叉卖

### 开仓/平仓

- **开仓**：近35日涨幅低点 > 1.5%（后优化收紧，避免短期涨幅过大时介入）
- **平仓**：信号转弱时止损，赚到一定程度先卖一半锁利
- **轮动频率**：每天评估信号，年均调仓17~25次

### 2026年优化

1. **溢价熔断**：场内溢价 > 5%时不进入纳指
2. **收紧高位开仓**：避免短期涨幅过大时介入
3. **最低持仓锁定期**：防止高位反复震荡加剧磨损
4. **平仓条件优化**：延迟平仓，少被日内抖动骗
5. **50%底仓法**：始终保留50%底仓，行情强再买50%，弱则卖50%

### 回测与实盘

| 版本 | 区间 | 年化收益 | 最大回撤 |
|:---|:---|:---:|:---:|
| 1.0回测 | 2016~2023（7年） | 36.96% | 22.96% |
| 2.0回测 | 2017~2025（8.6年） | 55.65% | 14.3% |
| 实盘3年 | 2022.11~2025.11 | 31% | 17% |

**来源**：2023/06/09《ETF轮动策略直播》、2025/10/15《新高51%！》、2026/06/07周报

## 6. 牛市三阶段投资框架

**定位**：A股牛市大多呈现"快速上涨→震荡盘整→持续上涨"的三段式节奏，不同阶段板块侧重不同。

### 第一阶段（初期）

- **特征**：情绪由谨慎转为乐观，大盘股先动，估值修复行情
- **涨幅**：创业板前期涨幅50%~156%
- **回调**：5%~10%，约两周
- **操作**：分批建仓，7-8成仓位
- **买什么**：沪深300、创业板50、科创板50等宽基；大盘蓝筹

### 第二阶段（中期）

- **特征**：热点轮动加速，板块风格切换，主力调仓换股
- **涨幅**：创业板中期涨幅66%~100%
- **回调**：10%~15%
- **操作**：关注盈利改善的板块，利用小回调加仓
- **买什么**：行业ETF、成长股、科技股、中小盘（中证500、中证1000）

### 第三阶段（后期）

- **特征**：投机性上涨，概念股爆炒，泡沫出现
- **涨幅**：明显收窄（17%~35%）
- **回调**：>15%，牛市结束回撤25%~40%
- **操作**：逐步获利了结，减仓止盈

### 牛市结束信号

1. 流动性收紧（核心标志）
2. 股债收益差百分位降至5%以下
3. 垃圾股飞天、成交量暴增
4. 居民存款/市值比降至0.8~1.1

### 牛市亏钱原因

追涨杀跌、急跌洗盘时割肉、末期追概念股、没有止盈纪律、心态膨胀重仓高风险股。

**来源**：2024/10/06《牛市涨幅规律 回调规律 买什么》、2024/10/27周报、2025/08/17周报

## 7. 四维轮动全天候策略

**定位**：利用四类低相关性资产的风险平价配置，消灭波动，让普通人也能重仓持有、穿越牛熊。

**核心逻辑**：源于达利欧全天候策略。将经济周期拆分为"经济好/差 × 通胀高/低"四种象限，每种象限都有对应资产能赚钱，保证任何时候都不全军覆没。

### 资产配比

| 资产 | 比例 | 角色 |
|:---|:---:|:---|
| 中长期国债 | 55% | 压舱石，稳定收益 |
| 红利低波 | 15% | 高分红防御 |
| 纳指100 | 15% | 经济向好时进攻 |
| 黄金 | 15% | 抗通胀，避险对冲 |

防御型资产合计85%，进攻型15%。

### 再平衡规则

- 任一资产偏离目标比例超过5%时，触发整体再平衡
- 低频调仓，不择时，不追热点

### 回测数据

| 指标 | 场外基金版 | 场内ETF版 |
|:---|:---:|:---:|
| 区间 | 2018.04~2025.01 | 2019.01~2026.01 |
| 年化收益 | 10.40% | 10.84% |
| 最大回撤 | 6.78% | 6.01% |
| 持有1年盈利概率 | 99.43% | — |
| 持有2年盈利概率 | 100% | — |
| 夏普比率 | 1.67 | — |

### 实盘表现

- 运作时间：2025年1月14日起
- 2025年4月：87%+跟投者盈利，最大回撤不超3%
- 2025年5月：94%+跟投者盈利，收益3.74%
- 2026年2月黄金暴跌16.76%，组合最大回撤仅3.62%

### 适用场景

- 一年以上不用的闲钱
- 追求低波动、稳稳增值
- 适合家庭资产"压舱石"
- 启四个人规划：攒够300万，年化10%，每月取2万消费，永续现金流

### 局限

- 纳指QDII时常限购
- 近年恰好中债+黄金+纳指三牛共振，存在过拟合质疑
- 55%中长债赌国内经济"L型"走势，利率上行期回撤可能翻倍

**来源**：2025/01/22《无脑躺赚40年》、2026/02/11《铁饭碗》、2025/04/21《87%的人盈利了！》
