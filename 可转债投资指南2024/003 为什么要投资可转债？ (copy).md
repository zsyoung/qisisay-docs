# #启四可转债 003 初识转债：为什么要投资可转债？

昨天讲了[什么是可转债？](https://mp.weixin.qq.com/s/7OZ8z3T5iFq16JfPRy8Mxw)今天继续讲为什么要投资可转债？

这个问题还可以这么问，投资品种这么多，为什么要选择可转债？

## 一、普通人可投资的品种

对于普通人来说，能投资的品种无非这六大类：**现金、固收、商品、另类、不动产、权益。**

> 1、现金，就是银行活期+、货币基金等。风险极低，收益极低；
>
> 2、固守，就是定期存款、债券、固收类的信托、私募等。风险低，收益适中；
>
> 3、商品，一般指黄金和原油。风险高，投资难度大；
>
> 4、另类，一般指邮币卡、艺术收藏品、比特币之类。风险高，投资难度大；
>
> 5、不动产，一般指房子、土地等。投资门槛高；
>
> 6、权益，就是指股票、基金等。投资门槛低，风险较高，收益较高。

梳理完之后，我们发现大众在市场上交易最多、认知最广泛的就是固收类的债券，和权益类的股票、基金这三个品种了。

而可转债就是兼具了固收类的债性和权益类的股性，它「进可攻，退可守」，操作得当，可以保证稳赚不赔，而且是大赚特赚。

并且，它的投资门槛极低，最低只需要1000元就可以投资。（2022年8月新规后，想交易可转债需要满足2年股票经验+近20交易日10万日均资产才可以）

## 二、可转债兼具低风险高收益属性

可转债兼具低风险高收益属性，这话怎么说呢？

一个骰子有六面，掷出1、2、3你赢一元，掷出4、5、6你不赢不输，你应不应该干？当然应该啊！稳赚不赔嘛！

一个骰子有六面，掷出1、2、3、4、5你赢一元，掷出6你输一元，你应不应该干？当然应该啊！只要掷的次数够多，一定是赚的嘛！

这种事不会发生在赌场，在证券市场却不少见，而可转债就是最鲜明的例子。

上篇文章我们说了：**可转债 = 转股权利 + 债券**

### 1、债券的债性保证了它的低风险属性

它风险低的原因主要有以下几点：

首先，可转债是归根到底是一张债券，当我们买入可转债之后，只要可转债的发行公司不破产退市，可转债都是要还本付息的。

其次，证监会制定了比较严格的可转债发行规则。比如“最近3个会计年度ROE平均不低6%”，“发行后累计公司债券余额不超过最近一期末净资产额的40%”，“可转债发行后，资产的负债率不能高于70%。”等对公司质地进行要求的条款，这些条款保证了整体上发行可转债的公司是相对优质的。

最后，就是证监会设计了一系列巧妙的条款来化解和解决可转债违约风险，相比股票，极大地保护了中小投资者，后面的进阶内容我们会重点讲解条款。

那么到底有没有企业到期之后，赖帐不还呢？

2023年之前尽管一直存在质地很差的转债，但转债从未发生过违约的情况，

但是0违约事件在2023年8月之后被打破了，我们先来看一组历史数据：

截止2024/03/06，历史上已退市的转债（337只）的平均收盘价为160.207元，其中只有6只的退市收盘价在100元以下，比例为1.78%。

这6只分别是：

| 名称     | 最后交易价格 | 退市原因  |
| -------- | ------------ | --------- |
| 搜特转债 | 18.002       | 正股退市  |
| 蓝盾转债 | 26.93        | 正股退市  |
| 正邦转债 | 84.813       | 正股重整  |
| 双良转债 | 93.82        | 低于3千万 |
| 全筑转债 | 97.032       | 正股重整  |
| 辉丰转债 | 99.999       | 低于3千万 |

其中搜特转债、蓝盾转债、正邦转债、全筑转债这4只均是因为正股退市或重整而提前退市了，但是这类违约的转债评级一般都在B级或者C级，并不是突然就会违约暴雷，完全可以提前判断规避。

所以，尽管存在违约的转债，但绝大部分转债的债性是有保护的，而且我们也可以提前规避违约转债。

### 2、转股权利的股性保证了它的高收益属性

转股就是把可转债转换为发行它的上市公司的股票，同时也不再拥有债券了。

这个特性带来的好处是，如果股票大幅上涨，那么我们也可以享受股票上涨带来的收益，因为可以债转股，即「进可攻」。

如我们之前举的例子，债王「英科转债」，正股英科医疗股价疯涨的同时，英科转债也随之不断上涨，在2021年1月29日这一天，最高价达到了3618.88元，相比发行价暴涨了36倍之多！

![iShot2021-08-11 23.25.14](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/iShot2021-08-11%2023.25.14.png)

而如果股票大幅下跌，我们可以不用承担股票下跌带来的全部损失，因为我们可以选择不进行债转股，到期拿回本息即可，即「退可守」。

## 三、可转债的历史投资收益

截止2024/03/06，统计已退市可转债的历史数据可知：可转债平均存续时间2.3年，97.63%的可转债退市前最高价都曾达到过130元以上。

转债自2018年进入大众视野以来，近5年涨幅56.22%，近3年涨幅23.76%，均大幅跑赢国内一众宽基指数。

## 四、总结

综上所述，可转债投资门槛低，风险和收益明显不对称，投资性价比极高，且历史投资收益可观，所以非常值得投资。
