yyb凌波（雪球ID）投资银行股的策略融合了可转债套利、正股价值挖掘和严密的仓位管理，其核心思路可拆解为以下四个维度，结合公开讨论和操作逻辑分析如下：

------

### 📊 **一、核心策略：银行转债/正股溢价率收敛套利**

1. **底层逻辑**：

   银行发行可转债的核心目的是补充资本金，**最终必然推动转股**（强赎或到期被动转股），极少选择还钱。

   **年化溢价率** = 转股溢价率 ÷ 剩余到期年限。该指标越高，意味着未来溢价收敛的空间越大，正股潜在收益越高。

2. **选股标准**：

   **高年化溢价率银行正股**：筛选剩余到期年限≤4年、年化溢价率＞10%的标的（如光大、中信、浦发等）。

   附加条件：

   市净率（PB）＜0.8（深度破净，安全边际高）；

   股息率＞5%（提供分红保护）；

   拨备覆盖率＞300%（资产质量优，风险缓冲足）。

3. **收益来源**：

   **溢价收敛收益**：若年化溢价率从15%降至5%，正股涨幅可能超额20%；

   **股息收益**：长期持有赚取稳定分红，抵减持仓成本；

   **强赎博弈**：银行可能拉升股价促成强赎（转股价的130%），带来超额收益。

------

### 🔍 **二、银行股筛选框架：四大类指标化繁为简**

yyb凌波在选股时聚焦四类核心指标，优先安全边际与股息保护：

1. 安全边际类（否决性指标）：

   **PB＜0.8**：破净幅度越深，股价下行风险越小；

   **不良贷款率＜1.5%**：资产质量稳健（如江阴银行拨备覆盖率高达450%）。

2. 股息回报类：

   **股息率＞5%**：提供现金流，如苏南城商行股息率普遍领先。

3. 盈利质量类：

   **ROE＞10%**：盈利能力持续（如招商银行ROE常年＞15%）；

   **净利润增速＞行业平均**：避免成长性断层。

4. 成长潜力类：

   **存贷款增速＞8%**：区域性银行（如成都银行）更易突破。

> **案例**：江阴银行被长期持有，因同时满足：
>
> PB=0.48（深度破净） + 股息率≈6% + ROE=10.2% + 拨备率＞450%。

------

### ⚖️ **三、仓位管理与风控机制**

1. 动态仓位调整：

   年化溢价率每下降2%，减仓10%；反之每上升2%，加仓10%；

   单只个股年化溢价率＞25%时，仓位上限25%。

2. 风险对冲设计：

   **转债替代正股**：当转债价格接近债底（如105元以下），直接持有转债替代正股，锁定下跌风险；

   **分散持仓**：持有6-8只银行股，避免单一个股信用风险。

3. 退出信号：

   年化溢价率降至0%（溢价完全收敛）；

   股价触及强赎价或转股价值＞130%（目标达成）。

------

### 💡 **四、投资理念精髓：坑中建仓，守正出奇**

1. 逆向布局：

   专注“被市场冷落的标的”，如银行股因地产风险暴露导致估值低谷时介入。

2. 长期视角：

   持股周期3-5年，等待溢价收敛与估值修复，不博弈短期波动。

3. 工具化思维：

   银行股作为“打新门票”：长期持有高股息银行股获取打新资格（如深圳市场）；

   可转债与正股灵活切换：根据债性/股性性价比动态调整。

------

### 📝 **总结：yyb凌波策略的实操要点**

| ------------ | ------------------------------------------------- |
| **选股**     | 年化溢价率＞10% + PB＜0.8 + 股息率＞5% + ROE＞10% |
| **仓位**     | 溢价率每波动2%调整10%仓位，单只上限25%            |

**一句话精髓**：

> 买的是银行“必转股”的确定性，赚的是溢价收敛+股息保护+估值修复的三重收益，用仓位管理控制时间成本。

当前时点（2025年6月），yyb凌波认为银行股处于“国家队与机构资金聚焦”的启动期，需耐心等待3-5年收获期，避免过早下车。
