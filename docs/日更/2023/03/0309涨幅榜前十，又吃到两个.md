# 涨幅榜前十，又吃到两个

文｜启四

大盘全天低位震荡，截至收盘，沪指跌0.22%，深成指跌0.16%，创业板指跌0.21%，科创50涨0.34%，北证50涨0.17%。两市全天成交额7567亿元，较昨日放量382亿元，北向资金净卖出40.77亿元。

## 二、今日观察

今天涨幅榜前十又吃到了两只，上一次吃肉仿佛就在前天：[涨幅榜前十吃到俩，这只转债跌出了性价比](https://mp.weixin.qq.com/s/drd6yJDLLUMTBlh3DKaC0g)

**1、天地转债**，昨天138.21买入，今天盘中最高涨幅11.67%，下午在146.5清仓，持仓2天，吃肉6%，是轮动策略抓到的第六条大鱼，前面五次分别是英联转债、台华转债、海波转债、英联转债、永吉转债。

得益于天地转债的出色表现，今天轮动账户日收益达到了1.4%，走出了5连阳。

关于轮动策略，在11-04的文章中[实盘新的轮动策略](https://mp.weixin.qq.com/s?__biz=MzkwOTI4NDAxOA==&mid=2247494485&idx=2&sn=fa8030e61c4a41658db91332c419c104&scene=21#wechat_redirect)有介绍，今年累计收益率7.31%，最大回撤3.45%。

虽然收益没有摊大饼高，但是考虑到回撤较小，又省心省力（每天只需要固定时间操作一次即可），性价比比较高，非常适合时间不充裕只想抄作业的选手。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230309205248440.png)



**2、永鼎转债**，这是摊大饼账户中的三低转债，昨晚各个群都在传播常温超导新闻，转债中它是唯一含超导概念的，因此遭到了炒作。不过这种炒作往往很难持久，今天趁冲高全部予以清仓。本次共持股20天，吃肉13.09%。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230309210905980.png)

永鼎套人能力强悍，后面不回到120元以下，暂时不考虑买入了，应该有一大批小伙伴就是在去年冲高回落后，在127-128的位置接手被套的。

**3、小规模妖债**，近两天反复活跃，如永吉、天地、永鼎、联泰、联得、花王、聚禾、飞鹿、起步、凌钢、维格等都或起飞或脉冲或蠢蠢欲动。
