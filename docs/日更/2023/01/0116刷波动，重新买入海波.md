# 刷波动，重新买入海波

A股指数全天放量上涨，截至收盘，沪指涨1.01%，重回3200点，深成指涨1.58%，北证50涨0.89%，创业板指涨1.86%，重回2500点，科创50涨1.88%。沪深两市全天成交额9171亿元。两市超3800股上涨，北向资金全天净买入158.43亿元。

可转债等权指数，今天涨幅0.83%，中位数121.107元，成交额430.83亿，继续缩量上涨。

## 一、今日操作

今天操作比较多，目标是利用好最近的一波大涨，刷刷波动，精简持仓数量，不做盲目的摊大饼。

1、精装转债，触发网格买卖，成本已经刷到110元附近了

2、高测转债，130.1减仓，129元建仓的，成本刷到了123元附近

3、洁特转债，108.653加仓

4、万孚转债，111.851加仓，洁特和万孚都是抗原概念火爆的时候买的，洁特是赚了没出，万孚是接的过早，下跌趋势中一直没接，目前基本已经调整到位，开始底部起涨了，加仓摊低成本

5、迪龙转债，再次起飞，128.017卖出2格

6、聚合转债，125.34清仓了，继续缩减持仓数量

7、轮动策略，147.476买入了海波转债，加仓了天地转债，清仓了永吉转债

##  二、涨跌幅榜

**1、涨幅榜**今天是普涨行情，行业并不集中。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230116200405905.png)

华亚转债是次新债中流通规模第二小的，仅次于蒙泰转债的1.31亿，高于1.5亿的崧盛转债。昨天我预测合理价格125-127元，可能冲击130元涨停。

看最近新债的表现，市场似乎又愿意给次新债更高的溢价了，接下来要上市的声讯转债和恒锋转债可以期待一下。

**2、跌幅榜**主要集中在妖债、汽车、煤炭等板块。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230116200334295.png)

跌幅榜第一的海波转债，和今天突然爆涨的崧盛转债，正股的涨跌都让人莫名其妙。

## 三、重点提示

下面是近期即将满足强赎条件的转债，要注意强赎杀溢价风险。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230116202609413.png)

1、拓尔转债，未来5天都不能满足强赎条件，风险暂时不大，如果价格回到130元附近，反而很具性价比；

2、瑞鹄转债，溢价率长期在15%-20%附近，如果不强赎的话，溢价率恢复带来的肉会不错，不过目前溢价率还有4.71%，风险较高我不打算参与博弈；

3、恩捷转债，周末突然发布了可能强赎提示性公告，之前多次满足条件沉默不语，这次突然发声，强赎可能性估计比较大，目前溢价率还有13.81%，风险很大。
