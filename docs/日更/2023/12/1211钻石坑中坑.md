# 钻石坑中坑

## 一、今日观察

### 1、姚记科技回本盈利

姚记科技今日再度大涨+5.69%，回本盈利。

它主要从事扑克牌的设计、生产和销售以及移动游戏的研发、发行和运营。今天大涨主要是源于游戏板块大涨。

当时买它主要有三个理由：

1）当时已经从年内高点下跌了32.5%+，距离下方支撑位31.73元不远；

2）游戏股龙头，活跃度很不错，每天振幅几乎都在3%上下；

3）上一次公告进展日是2023-05-24，证监会已经同意注册，如果进展顺利，短期内可能会发债。

看似风险不大，但这是A股，连沪深300指数都能一跌再跌，更何况个股，这几个月持有姚记科技的过程还是很波折的：07-25在32.5元买入，先大赚15%+，后大亏18%+，期间还经历过清仓，待企稳再接回。姚记科技由于临近截止日无法发债还申请了一次延长发债有效期。

不过好在传媒始终是今年的主线，通过连续的网格，结合姚记科技的大波动，今日终于再度回本盈利。目前成本27.602，相比当初买入价，成本降低了15%。仓位也不高了，还有2网就将清仓。

姚记科技带给启四的启示：

1）永远要对股票含有敬畏之心，严格控制仓位（我的配债股单只通常仓位在5%以内，再看好也不会超过10%，仓位控制好了，心态才能控制好，才不会操作变形）。

2）永远不要担心市场缺少赚钱的机会，要担心的是机会来临前是否有足够的资金。

3）估值短期会失效，长期必然回归有效。
