# 直线跌停，怎么办

文｜启四

## 一、今日观察

### 1、姚记科技

姚记科技午后直线跌停，是被游戏板块带崩的，今天那则重磅消息大家应该都知道了，国家新闻出版署发出一则意见征求稿：网络游戏不得设置每日登录、首次充值、连续充值等诱导性奖励。

此消息一出，国内最大的游戏公司腾讯控股、网易游戏等纷纷大跌，A股游戏ETF午后跌停，姚记科技自然不能幸免。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20231222195652255.png)

对于此消息，未来会不会导致游戏股一蹶不振，我觉得没那么悲观。原因有以下几点：

1. 目前监管还处于征求意见稿阶段，后续肯定还会针对征求意见稿与产业座谈，最终执行力度也许有缓和空间。
2. 限制诱导性奖励仅仅会**短期**影响游戏行业赚钱的速度，阻碍不了游戏行业赚钱的能力。就像每个烟盒上都有「吸烟有害健康」的标识，但是撼动不了烟草每年都是第一纳税大户，2022年烟草总公司纳税1.4万亿，第二名工商银行纳税仅1096亿，有着数量级差异。游戏和烟草一样，盈利都来源于人性的弱点。
3. 上有政策，下有对策，我天朝上国人民的智慧，永远能找到合理的钻漏子方法。不让我设置诱导性奖励，那我诱导性惩罚可不可以呢？
4. 盘后龙虎榜数据显示，游戏股紫天科技、奥飞娱乐等大幅净买入，一些机构资金抄底已经悄悄抄底了，腾讯盘后也公告斥资10亿元回购了359万股。机构都抄底了，

目前我继续持有姚记科技，没抄底也没割肉，网格条件单不修改也不暂停，总仓位2.1%，心态良好。

当前姚记科技17倍PE估值，同行业处于中游。没必要过度恐慌，平淡看待波动即可。

市场先生像个精神病，每天都会随机给出价格，有时候甚至天上一脚底下一脚，别瞅精神病出什么价，瞅手里的货，高估了就出，低估合理了就拿着，就这么简单。

### 2、次新债乱杀

今天大叶转债继续-20%，连续两天20%跌停，雅创转债-8.55%，3个交易日下跌23.67%

我在12-14的文章中曾提示过次新双高债的风险，并明确指出下个大跌的对象大概率就是大叶转债，仅仅过去了6个交易日，大叶转债已经从200元跌到145元了，它的合理价值就在120元左右，当前依然是高估的。

有朋友给我转发了某个投资群的截图，有人单纯地认为昨天跌停板了，今天一定不会，所以买入1000张就赌它开板，有人说自己150元的成本，希望活着出来……

正是见过太多不懂双高债的危险，眼一闭心一横就冲进去的韭菜，也听过太多的哭诉，所以我一直苦口婆心远离这些双高债（150+元、50%+溢价）。

还是那句话：在正确的方向上，用正确的方法，做正确的事情，这才是最可靠的赚钱方法。
