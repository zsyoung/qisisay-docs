# 快止跌了，如何选择

文｜启四

## 一、今日观察

### 1、当前的选择

今日转债平均跌幅-0.68%，对应正股平均跌幅-1.26%，转债整体规模从年初的8215.510亿，扩容到现在的8560.420亿，规模增加了345亿，转债成交量却基本一直在600亿以下徘徊，今日成交量只有343.27亿，换手率4.01%，和上周五基本持平，市场一直处于缩量下跌中，这通常意味着买方进入不活跃，卖方抛出股票的意愿也不强。

转债有到期收益率为正、回售价、面值100元几道关口，下跌到一定程度就会止跌，这也是比股票优越的地方。

当前转债中位数：116.665

所有历史范围内所处百分位：50.21%

近1年范围内所处百分位：3.28%

近2年范围内所处百分位：4.53%

近3年范围内所处百分位：23.46%

估值已经处于近一两年的绝地低位了，这个时候有钱抄底，没钱熬着是最佳选择。

### 2、金现转债上市

金现转债 123232（正股金现代 300830）

1）转债概况：发行规模2.025亿，行业：计算机-软件开发-垂直应用软件，转股价值93.08元。拟合曲线插值计算预测价格119.327元，转股期前可流通规模约1.76亿。

公司主营业务是电力行业信息化解决方案。质地一般。

2）对标转债：恒锋转债

3）上市价预测：给予46%-48%的溢价，**预估上市合理价值136-138元**。流通规模较小，130元开盘，尾盘极大概率冲击157.3元，考虑到近期新债上市较少，**有冲击180元潜力。**

4）操作建议：如果中签，建议捂一捂，首日不要贱卖，未来几天可以考虑从最高价跌10元卖出。也可以设置回落卖出条件单，价格高于150元，回落3%，即时买三价卖出。

### 4、福新转债

福新转债，上周五收盘折价-5.89%，今天正股开盘-7.12%，盘中虽然回正，但是尾盘-9.99%跌停。

如果上周参与折价套利的话，大概率是会亏损出局的（一般折价套利都是次日开盘直接卖出正股）。

大多数情况下，折价套利坑大于利。

弱市少博弈，这种不敢上大仓位的博弈套利，启四会尽量减少和规避。
