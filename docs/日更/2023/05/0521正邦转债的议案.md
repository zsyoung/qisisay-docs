# 正邦转债的议案

文｜启四

5月19日晚，正邦转债发布了《关于召开“正邦转债”2023年第一次债券持有人会议的通知》，公告称*ST正邦准备在6月5号召开转债持有人大会。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230521205343558.png)

会议打算审议下面几件事情：

1. 提议推选国信证券作为“正邦转债”全体未偿还债券持有人的受托管理人，并且根据《企业破产法》明确了法院若受理公司破产申请，则正邦转债提前到期；
2. 法院受理公司破产申请之日开始的15个自然日内，可转债持有人可以交易；
3. 法院受理公司破产申请之日开始的30个自然日内，可转债持有人可以转股。

目前还不能确定是利好还是利空，如果是按普通债权确定，可能存在打折或者延期兑付的可能，是利空，如果按110元到期兑付债权，是利好。

但是根据正邦这则公告的意思（没说要按照110元兑付)，可能是想按照普通债权走，待公司破产重整成功后按照比例进行偿付。

这样的话，转债人原本拥有的转股权利就被剥夺了，因为可转债=债权+看涨期权，如果未来都像正邦转债这样破产受理即到期，那么将严重损害转债持有人权益，白白剥夺掉了转债人的一大权利，毕竟可转债的利息本来就偏低。

正邦转债的募集说明书中规定了转股期限：

> 本次发行的可转换公司债券转股期限自发行结束之日起满六个月后的第一 个交易日起至可转换公司债券到期日止。

所以这个到期日就至关重要，究竟是按照普通债权那样破产即到期，还是保留可转债的转股权利直到强赎或者2026年6月到期，市场没有先例。

如果这次议案成功通过，那么将开创转债市场先例，其他申请重整的转债，包括全筑转债、花王转债、搜特转债（已经没救了）也将迎来重大打击。
