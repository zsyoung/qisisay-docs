## 二、今日观察

今天有两件事活久见。

### **1、搜特转债收盘后创下史低价58.3元**

目前正股*ST搜特股价已经连续9个交易日在1元以下，连续20个交易日股价低于1元将被强制退市。当前股价0.72元，需要连续7个5%涨停才行，再结合昨天的终止筹划控制权变更事项公告，搜特99%是救不回来了。

早上我在社群提示了暴雷债风险，如果在开盘前及时止损，都能避免更多的损失。

注册制下，该断臂求生的时候不要头铁，投资思路要与时俱进。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230505203850988.png)

### **2、嵘泰转债乌龙交易**

昨天已经是嵘泰转债赎回摘牌的最后交易日了，但是今天2023年5月5日早盘，它居然仍然可以挂牌交易，并且最高暴涨了19%，等上交所发现后，已经成交了接近1亿元。

上交所是这么做的：先于上午10时10分实施停牌，之后公告称是技术原因，嵘泰转债相关匹配成交取消，交易无效，不进行清算交收，并深表歉意。

有人追高参与了这次炒作，这种情况应该感到庆幸，不然又白白被割了韭菜。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20230505205120450.png)

至于那些参与了炒作并且赚钱了的老铁怎么办？上交所表示我都道歉了，你还要怎样？

### 3、两只新债

新港转债，流通规模0.9亿，如昨天文章所说，尾盘果然遭到了炒作，最高价157.3元，如果按照我建议的回落条件单卖出的话，卖出价应该是147.9元左右，比收盘价133.534元高出10.76%的利润。

百洋转债，刚好上市日碰到医药大跌，目前129元的价格在我看来是略低估的，合理溢价应该在32%-34%，对应133-135元的价格。

### 4、杭电转债提议下修

2023-05-19股东大会，2024-03-05到期，到期赎回价108元，正股pb1.48，接近到期，公司急了，大概率会下修到底。

根据当前20日均价预测，下修到底后转股价值大概率在96-100元左右，考虑到临期债，给予18%-22%的溢价率，合理估值在117-122元，当前价格114.193元，还有上涨区间。
