# 清仓三羊马，下周靠它吃肉？

文｜启四

截至收盘，沪指涨0.11%，深成指涨0.25%，北证50涨0.70%，创业板指涨0.43%，沪深两市成交额8266亿元。两市超3700只个股上涨。北向资金净卖出28.59亿元。

![](https://files.mdnice.com/user/7588/9a4b258a-9d05-4fab-ad30-edb46aab8a5b.png)

可转债等权指数，今天涨幅0.03%，中位数120.724元，位于二星级估值区，成交额509.94亿。

## 一、今日观察

**1、艾录转债**，下周一，2023-11-20上市，基本确定大肉，周日的文章我会发出预测分析和操作建议。

**2、东湖转债**，公告提前赎回。当前价格173.6元，溢价率-0.08%，经过连日的折价，当前剩余规模仅1.238亿，对正股基本形不成大的抛压了。最后交易日还未公布，接下来一段时间走势就跟随正股了。

**3、三羊马**，今天趁着冲高清仓了。第4次大肉获利了结。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20231117213345064.png)

10-19的文章：[提示2个套利机会](https://mp.weixin.qq.com/s/dIebBFnGZp7Nwnw49xcWwQ)中有写到本次36.69建仓。

三羊马流通规模小，股性活跃，今年一共做了3次。

第一次持股36天，经过网格做T后盈利5.2%清仓。

第二次持股37天，经过网格做T后盈利8.32%清仓。

第三次持股34天，经过网格做T后盈利9.31%清仓。

第四次持股22天，经过网格做T后盈利10.43%清仓。

四次博弈，赚的盆满钵满。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20231117203912630.png)

**4、远信工业**，发行可转债申请获得上市委通过。

我在11-14的文章：[新买一只配债股](https://mp.weixin.qq.com/s/cmWXnhqji-m6uml-Q1tAoA)中提到建仓了它，并给出了理由，当时预测本周五上市委会通过，果然成真。

接下来还需要经过证监会同意注册，和申购发行两个阶段，至少还有2个月的时间，可以慢慢玩。

也许是下一个三羊马？

**5、泰坦转债**，今天继续20%强势涨停，流通规模仅1亿，确实是近期难得一见的次新香饽饽，可以设置回落卖出条件单，大于200元回落5%卖出。这样就不会被小的波动甩下车。

三羊转债也不出意外尾盘冲到157.3元，尾盘抢筹的买单高达118.3万，下周上200元基本无悬念，甚至可以期待300元，它的流通规模比泰坦转债还小，可以参照泰坦转债的最高价卖。

**6、湖广转债**，今天正股涨停又开板，转债最近涨幅不小，应该是受抖音测试付费短视频消息刺激，但是需要提醒的一点是湖广转债还有半年左右就要到期了，临期债不配这么高的溢价，持有的话要注意风险。

**7、多伦转债**，昨天没参与折价套利，今天也没有，这种套利不敢上大仓位，就不浪费心神了，恭喜参与折价套利的小伙伴吃肉。
