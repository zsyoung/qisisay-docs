## 一、今日观察

### 1、泰坦转债上市

**泰坦转债 127096（正股泰坦股份 003036）**

1）转债概况：发行规模2.955亿，行业：机械设备-专用设备-纺织服装设备，转股价值93.48元。拟合曲线插值计算预测价格121.944元，转股期前可流通规模约1.02亿。

公司主营业务是纺织机械设备的研发、生产和销售。质地尚可。

2）对标转债：大叶转债

3）上市价预测：给予51%-53%的溢价，**预估上市合理价值141-143元**。流通规模很小，130元开盘，尾盘大概率冲击157.3元，有冲击200元潜力。

4）操作建议：如果中签，建议捂一捂，首日不要贱卖，未来几天可以考虑从最高价跌10元卖出。

### 2、下修情况

**1）岩土转债**，提议下修，曾经主动下修到底过一次，目前参数见下图。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20231114210348860.png)

虽然价格和规模都很低，但缺点是剩余年限仅0.334年，没几个月就要到期了，临期债往往溢价率会受到压制。

其实无论能否下修到底，价格都很难大涨了，除非正股突然有所表现。

当前正股PB1.778，不受下修条款中转股价不得低于每股净资产的限制，以正股现价和20日均价预估下修到底后转股价值为98-100元，**预估合理价值109-111元**。

我统计了一下数据：

从转债上市日至今，2018-04-25～2023-11-14，

正股起始收盘价：7.36，结束收盘价：2.63，区间涨幅：-64.26%；

转债起始收盘价：95.204，结束收盘价：109.951，区间涨跌幅：15.49%；

充分证明了转债投资的优越性，只要不乱追高价债，想大亏很难。

**2）天赐转债**，修正后转股价格：28.88/股，基本下修到底了。下修后转股价值96.57元，给予25%-27%的溢价，合理价值在120.5-122.5元，当前价115.25低估不少。

其实正股质地非常好，只是电池行业前几年太如日中天，涨的太猛太狠，如今估值回调，大部分公司股价杀起来也是如瀑布奔流，如长远锂科、利元亨、嘉元科技等等。这种大级别的调整往往要持续很久，比如1-3年，需要耐心等待业绩和情绪修复。

### 3、配债股

今天在12.90元清仓了**泰坦股份**，做T后盈利7.6%。

上次是在12.14元抄底的：[抄底泰坦股份！](https://mp.weixin.qq.com/s/O3qi-VUmZl3uEEwYpiQI4g)

腾出资金后，建仓了**远信工业**。

建仓理由：

1. 深交所审核委员会定于2023年11月17日（本周五）审议远信工业、龙星化工的可转债发行事项，如果没问题的话，本周五应该就能通过，形成利好。
2. 远信工业发债规模2.865亿，流通规模1.1亿，总市值22.03亿，流通市值仅6.27亿，符合启四的配债股选股标准——优先小市值。
3. 远信工业配债安全垫尚可，是昨晚的[20只妖债苗子](https://mp.weixin.qq.com/s/ag3LyXgCKB1RZfVaApmfGw)中之一，相比其他几只，股价涨幅尚可接受，还在股东大会阶段的配债股距离发债还很久，暂不考虑。
4. 远信工业当前振幅3.75%，波动性不错，如果价格下跌，未来更容易通过做T降低成本。
