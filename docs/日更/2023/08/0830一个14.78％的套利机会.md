# 一个14.78％的套利机会

## 一个14.78%的套利机会

### 前言

经纬纺机（SZ:000666），今晚发布了《关于撤回公司股票在深圳证券交易所交易的方案》。

公司拟以股东大会决议方式，主动在深交所退市，转到中小板交易。

背后的原因众说纷纭，大概率是因为中融信托暴雷引起的，央企为体现担当，保护中小股东利益，遂决定这么操作。

### 套利机会来源

经纬纺机退市公告中的【异议股东及其他股东保护机制】，给异议股东提供了9.24元的现金选择权。

大白话说就是，如果你持有经纬纺机的股票，那么在9月13日股权登记日之后，无论持有股票成本是多少，都可以以9.24元/股的价格卖回给公司。

目前经纬纺机收盘价是8.05，距离9.24有14.78%的涨幅空间，明日涨停价是8.86，今晚可以提前挂单抢筹。

### 风险点

**1、本次以股东大会决议方式主动终止上市事项相关的审批风险**

目前退市事项尚需公司股东大会审议通过，不过目前十大股东中，持股占比58.32%的前三大股东均同意主动退市，而这次退市本身就利好中小股东，所以接下来基本就是走一遍股东大会流程，极大概率是能通过的。

**2、现金选择权相关的风险** 

若股东大会不能通过退市议案，那大家自然不能获得现金选择权，也就不存在套利空间，但是上面第一点已经说了，大概率是能通过的。

若申报行使现金选择权时公司股价高于现金选择权价格9.24元，则股东申报行使现金选择权将可能使其利益受损，这时候不申报就是了，收益会更高。

### 总结

1、央企主动退市。

2、经营状况不错，持续稳定赚钱，现金充足，负债率低，持续稳定大额分红。

3、每股净资产14.19元，异议股东的现金选择价格为9.24元，截止目前，收盘价8.05，有14.78%的套利空间。
