# 15只三低转债名单，涨幅榜前2很危险

文｜启四

1、昨天有朋友想让我发一下最新的三低转债。

下面是按我的标准筛选出的15只三低转债名单。

> 1. 溢价率 ≤ 50%，剩余规模 ≤ 2.5亿的，价格 ≤ 125元
> 2. 溢价率 ≤ 80%，剩余规模 ≤ 1.5亿的，价格 ≤ 130元
> 3. 排除掉已发强赎公告和剩余年限小于0.5年的可转债

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240301215335548.png)

不出意外的话，3个月内，其中至少10只可以冲上130元。

小盘股的流动性风险未来也许还会存在，但是小盘债由于有下修和回售条款在，形成了比大盘债更明显的下跌有底上涨无限的局面。

转债由于有T+0的交易优势，日内波动就成为其超额收益的一大来源，而小盘债盘子小，天然存在流动性溢价，一旦市场转为平稳，溢价就会被逐渐抬升。

当前阶段，我认为小盘债今年依然会具有强劲的超额收益。

2、今日涨幅榜第一的中旗转债，之前我做过分析：[新妖债？能冲240+？](https://mp.weixin.qq.com/s/ZO8i1oMFy6LqfzWdYUOrlQ)，我的观点是未来它的归宿一定是暴跌，追高的散户被必杀，别看现在蹦的欢，小心将来拉清单。奉劝远离。

3、涨幅榜第二的福蓉转债，正股2天2板，转债两个交易日上涨30%，公司生产的铝制结构件材料用在手机、平板和笔记本电脑等终端产品的中高端机型上面，如三星品牌的S系列手机、折叠屏手机等。

上涨的主要原因是：华为P70有望发布；1月手机出货量大增；苹果等手机巨头加码生成式AI等，公司所属的手机产业链行业收益。

不过当前156元+51%的溢价率，已经进入双高债范畴，剩余规模6.4亿，又已经进入转股期，未来回撤杀溢价的风险不小。
