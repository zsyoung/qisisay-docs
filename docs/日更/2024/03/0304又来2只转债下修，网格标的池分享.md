# 又来2只转债下修，网格标的池分享

文｜启四

## 一、今日观察

### 下修

海环转债，未来转债，提议下修。

**海环转债**，现价110.650元，2023-11-10曾经主动下修到底过1次，剩余年限只有1.08年，公司是真的着急想强赎了，这次下修到底的概率也很大。如果下修到底，预估转股价值98-100元，预估合理价值112-114元左右。

环保行业热度一般，政策依赖性强，转债普遍没什么溢价和活跃度，一大堆高到期收益的。小票也很少波动，比如联泰转债、京源转债等。持有这类转债大多数只能熬时间，很少能吃到市场脉冲收益。

**未来转债**，现价118.410元，2023-09-12主动下修过一次，它的剩余年限也只有1.082年，本次大概率也会下修到底。如果下修到底，预估转股价值98-100元，预估合理价值120-122元左右。

定制家居这个行业在房地产的崩盘背景下一直在走下坡路，未来转债在其中还算是矬子里拔高个，5只转债里价格是最高的，原因就在于它的到期赎回价高达120元。但是谁让临期债不如狗呢，给不了太高的预估价。

### 强赎

招路转债，公告提前赎回，现价135.5元，溢价率-0.43%，剩余规模15.556亿，正股日成交量每天3-6亿左右，此次强赎规模较大，短期可能会形成股债双杀的局面，如果持有，建议及早走人。
