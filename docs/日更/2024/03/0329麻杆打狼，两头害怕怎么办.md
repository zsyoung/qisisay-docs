# 麻杆打狼，两头害怕怎么办

文｜启四

## 一、今日观察

### 短视思维

又是大涨的一天，原本计划在4月份暴雷高峰期到来前，在本周末清仓部分或全部转债，但是本周挑来挑去只清仓了2只大饼转债，2个配债股。剩余的一些转债要么是历史低价，远低于到期赎回价，要么是业绩没问题，溢价也不高的转债。

手上现在大仓位全是轮动转债，轮动策略今天收益0.7%，3月整体收益率5.45%，还有2.98%今年就能收益回正了。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/IMG_6488.JPEG)

从最大回撤-17%到现在的-2.98%，短短不到2个月的时间而已，当时的恐惧感再回想已经很模糊了，不知道是人类忘记痛苦的能力强，还是我适应了这种级别的回撤，也可能是心态比较坚定，善于给自己和他人按摩。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240329211217954.png)

现在市场这个位置很尴尬，有点麻杆打狼，两头害怕，清仓了怕踏空，持仓了又怕跌。

思来想去，还是短视思维惹的祸，总想在某个月达成超额收益，但是短期的行情又太难预测，做的越多，反而容易 错的越多。

索性就继续按照策略执行下去，假如4月份真的大跌，手上目前腾出的资金就继续低位加仓。反正现在中位数只有112元，距离中位数118元还有5.35%的距离（从2019年转债被大众熟知以来，每一年的中位数都达到过118元以上），考虑到轮动策略能跑赢等权指数，中位数真的来到118元的时候，收益估计有7%-8%了。

拉长视角看，今年取得正收益几乎是板上钉钉的，就是盈利多少的问题。接受短期的跑输，承认自己没有短期预测的能力，放过自己未尝不可。

后面的重心还是放在策略优化、因子回测和择时回测上，最近在回测中发现：新规后，双低+剩余市值+正股年化波动率的因子组合表现很不错，逻辑大概是新规后普通小规模债被大幅脉冲炒作的现象变少了，绝大部分的脉冲都是股债联动带来的，所以正股年化波动率这个因子会给小规模转债带来超额收益，大家选债的时候可以重视一下。

### 套利

标普500lof今日溢价只有1.37%了，而且今天是老外节日，暂停申购，我就没有继续申购了。

盘后标普500lof传来暂停申购的消息，下周连100元也不能申购了，下周溢价率大概率会再次抬升，基金又要大涨，不过已经跟我没关系了，这次我纯粹就是为套利而来。

总结起来，这次套利在溢价率5%以上进行申购，到账后集合竞价阶段卖出，每次都是盈利的，是一次成功的套利周期。
