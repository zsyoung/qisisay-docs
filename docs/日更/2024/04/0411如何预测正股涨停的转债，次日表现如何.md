# 如何预测正股涨停的转债，次日表现如何

文｜启四

## 学知识

### 如何预测正股涨停的转债次日表现如何？

启四给大家提供一个方法：使用同花顺的动态分组看当日涨停正股的“涨停封成比”。

**1、先解释一下“涨停封成比”是什么？**

它指股票涨停当日收盘时，盘口封单量和成交量的比例。
根据封成比的不同值，可以判断次日封板的概率：
1. 封成比大于10，次日以涨停开盘的几率大于70%。
2. 封成比小于10，大于3的，次日高开幅度多在6%至10%之间。
3. 封成比小于3，大于1的，次日高开幅度多在3%至6%之间。
4. 封成比小于1，大于0.5的，次日高开幅度多在1%至3%之间。
5. 封成比小于0.5的，次日开盘多在昨天收盘价左右，并不一定高开。

**2、如何在同花顺进行设置？**

首先，点击同花顺底部“首页”——应用商店——表头——搜索“涨停封成比”——添加表头。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/IMG_6615.JPEG)

其次，点击同花顺底部“自选”——左上角“三”——动态分组——新建动态分组——输入“今日涨停，含可转债”——查看“涨停封成比”。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/IMG_6612.JPEG)

从图中我们可以看出，这些正股的“涨停封成比”都小于0.5，次日开盘不一定会高开，那么就不适合追高转债。
