# 倒车接人，此时恐惧应战胜贪婪

文｜启四

## 一、今日观察

### 大跌

今天的大跌是什么级别？

转债平均跌幅-2.19%，对应正股平均跌幅-5.76%，乍一看很吓人，其实今天的转债跌幅只能排到历史第16位，也比不上今年01-22的跌幅-2.21%。

很多踏空急得抓耳挠腮的小伙伴，这不，上车的机会又来了。但是很多人又怕了，担心这波又会跌到天昏地暗，启四认为概率很小，更有可能是倒车接人，很快反弹。

一是，新官上任三把火，市场交投情绪已经起来，也有了主线，很难再跌破前低。

之前的文章中，我发过数据图，回顾历届证监会主席任期，进入2000年以后，除了担任时间最短的郭树清以外，其余主席均是在市场下跌时被换掉，任期内沪深300指数均走出了“A”字。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240207205129648.png)

二是，本次转债对应的正股反弹幅度足够大，和转债正股走势很接近的国证2000，从最低点已经反弹了33%+，市场有止盈需求，没有永远的下跌，也没有永远的上涨。

昨晚的文章：[接下来市场大概率会这样走](https://mp.weixin.qq.com/s/hqRh7RsF-iQq-fW-V_dqWg)我提出了市场的5个阶段，目前1、2已经吻合，第3阶段也80%吻合，今天微盘股的回调，就很像下图箭头里的第3阶段的回调，当时也是10%级别的，随后又上冲一波，属于倒车接人。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240228212852781.png)

三是，转债中位数再次来到110元区间，拉长时间看是绝对的低估位置，**这时候贪婪应该战胜恐惧。**

### 择债

目前加上今晚提议下修的岱美转债、迪贝转债，还在等待下修股东大会的转债多达18只。

其中不少还是到期正收益，既有保底安全性，又有正股底部反弹预期，和下修后的上涨预期。

相比博弈还未下修的转债，直接在这里选难度更小。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240228213624606.png)

今天上午我清仓了前几天博弈下修的神通转债，以及2024/07/01前都不会下修且溢价率高达109.74%的垒知转债，接下来会优先换仓待下修转债。

### 下修

1、岱美转债、迪贝转债，提议下修。

**岱美转债**，现价116.729元，它现在下修的目的主要是掩护大股东减持——十大持有人比例85.93%，有下修不到底的可能。如果下修到底，预估转股价值98-100元，预估合理价值117-120元左右。

**迪贝转债**，现价114.949元，没有过下修记录，剩余年限只有1.652年了，这次下修到底的概率很大。如果下修到底，预估转股价值95-97元，预估合理价值119-121元左右。

2、东南转债，华友转债，海波转债，未来6个月不下修。

荣23转债，未来2个月不下修。

泰福转债，2024/07/01之前不下修。

3、德尔转债，现价115元，修正后转股价格：15.88元/股，底价15.56元/股，未下修到底，修正后转股价值96.41元，考虑到剩余年限只有0.38年，到期赎回价112元，合理价值113-116左右。

4、明天有18只转债即将满足下修条件，分别是：

> **国城转债**、**亚泰转债**、宏微转债、**亚药转债**、**山鹰转债**、智能转债、龙星转债、丰山转债、**长集转债**、宏丰转债、火炬转债、**新港转债**、聚隆转债、特纸转债、**鹰19转债**、岩土转债、恒锋转债、设研转债。

其中鹰19转债、特纸转债、宏微转债、山鹰转债、亚泰转债都有过1次下修记录，亚药转债甚至下修过3次，上面字体颜色加重的，个人认为下修概率大一些。

**特别提醒：**集思录的下修和强赎数据经常出现问题，比如这次的岱美转债，今天已经发布提议下修公告了，网站上还显示下修进度为14/15。我查了一下，是因为2024-02-22的转股价值为84.99，却没有被计入下修满足日，总进程少了1天。东南转债也是同样的问题。我已经在网站提了相关建议，希望集思录及时修复，大家使用数据过程中也要注意。
