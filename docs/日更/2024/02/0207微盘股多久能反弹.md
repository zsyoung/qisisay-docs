# 微盘股多久能反弹

文｜启四

## 一、今日观察

### 1、证监会换帅

今天一则重磅消息公布，证监会主席易会满被免职，由吴清接任，吴清有个狠标签——号称“券商屠夫”。

2005年，吴清从机构部主任改任风险办主任，专事问题券商的风险处置，主持处理过南方证券、闽发证券、“德隆系”券商等31家违规证券公司，其中，推动26家公司进入司法破产程序，彼时，因其强硬作风被证券界称为“券商屠夫”。

从吴清以往履历来看，他作风强硬，严打老鼠仓，和易会满的宽松态度截然不同，给业界留下“敢碰硬敢亮剑”的印象。

我扒了一下数据，回顾历届证监会主席任期，进入2000年以后，除了担任时间最短的郭树清以外，其余主席均是在市场下跌时被换掉，任期内沪深300指数均走出了“A”字。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240207205129648.png)

市场一直流传一句话：“熊市末期标志是换帅”，参考以往沪深300指数走势和历届主席任期，似乎不无道理。

### 2、微盘股多久能反弹？

小市值开始回暖，今天中证1000大涨4.47%，但是微盘股依然持续暴跌，wind微盘股指数今天暴跌9.03%，自01/12以来，已经累计跌幅44.97%。

情绪转冷、估值较高和业绩预告不佳是微盘股调整的主要原因。

历史上，类似这种流动性高且迅速下跌的情况也曾经发生过，

在2015年6月18日至7月7日期间，万得微盘股指数下跌了47.37%，交易活跃，换手率保持在3%以上。特别是在7月6日和7日，指数的换手率分别为2.32%和3.52%，对应近年分位数分别为100%/37.8%，活跃度达到峰值后迅速下降。

7月8日，指数触底，并在接下来的14个交易日内上涨了30.6%，而同期沪深300指数上涨了4.5%。

而在2024年1月12日至2月2日期间，万得微盘股指数的换手率超过了一年中65%的分位数，2月2日更是达到了近一年最高水平的4.11%。

根据历史经验，一旦微盘股指数的流动性达到峰值后，很可能会迅速反弹。
