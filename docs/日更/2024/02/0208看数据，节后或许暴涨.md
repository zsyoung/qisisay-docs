# 看数据，节后或许暴涨

文｜启四

## 一、今日观察

最后一天了，微盘股终于气势如虹地开启了反攻，也印证了周二文章中的观点。

转债今日平均涨幅1.48%，对应正股平均涨幅6.79%，平均溢价率从92.76%下降到82.19%，直接下降了10个百分点。

昨晚的文章[微盘股什么时候能反弹？](https://mp.weixin.qq.com/s/3jPBaguEVYtP_UvpYW80-w)提到了：**根据历史经验，一旦微盘股指数的流动性达到峰值后，很可能会迅速反弹。**

历史上，类似这种流动性高且迅速下跌的情况也曾经发生过。

2014/12/31～2015/12/31，这一年微盘股指数涨幅232.37%，最大回撤55.12%。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/IMG_5792.PNG)

在2015/06/18～2015/07/07期间，万得微盘股指数下跌了47.37%，7月8日指数触底，7月9日探底成功开启反弹，并在当年达成新高。

这次，在2024/01/11～2024/02/07期间，万得微盘股指数下跌了45.98%，时长和跌幅都非常接近。

2015年指数探底成功时间附近的换手率如下：

- 7月6日，6.75%
- 7月7日，2.73%
- 7月8日，4.17%
- 7月9日，6.09%（探底回升）
- 7月10日，2.94%

2024年的换手率如下：

- 2月2日，3.97%
- 2月5日，3.92%
- 2月6日，5.81%
- 2月7日，6.26%
- 2月8日，7.93%（换手率新高）

近年来，微盘股换手率一般在3%附近，今天换手率创出了新高，如果年后换手率大幅下降，可以视作阶段性探底成功。

2015/07/08指数触底后，在接下来的14个交易日内上涨了30.6%，而同期沪深300指数上涨了4.5%，希望年后微盘股走出一波强势行情。

我去查了当时的市场新闻，和现在差不多，关键词是一样的流动性问题、高估值、股指期货、融资融券、恶意做空、gjd救市、千股涨停等等，一样的暴跌——救市——反弹流程。

如果把自己代入彼时彼刻，不到一个月的时间经历了比如今还要大的回撤，负面消息和悲观言论一定比现在还密集，信心和信仰会被冲击得比现在还凶猛，那时候的人们会相信小盘股有未来吗？恐怕更加悲观。可结果呢？年内再创新高。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240208210845939.png)

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240208203623283.png)

说白了，三根阳线改信仰，不是股市亘古不变的铁律，是人性的铁律。

迅速忘记疼痛是刻在人类基因中的本领，在原始社会，人类必须迅速忘掉疼痛，才有体力去觅食，存活下去。

届时涨起来了，“量化资金撤离，小盘未来完蛋了”的论调自然又会销声匿迹了。

涨久必跌，跌久必涨，靠情绪和消息投资不可靠，我们需要做的是分析数据，总结规律，形成策略，在执行中反思复盘，总结成功经验，规避历史教训。

这次微盘股大跌带崩转债，带给我的教训，一方面是配债股跌破60日均线一定要及时止损，一方面是转债要参考微盘股走势，如果出现放量大跌风险，要考虑清仓避险。

假期计划观察数据，总结经验，整理成可执行的方案，未来策略要进一步降低回撤风险。
