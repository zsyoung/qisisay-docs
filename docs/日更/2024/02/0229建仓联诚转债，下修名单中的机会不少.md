# 建仓联诚转债，下修名单中的机会不少

文｜启四

## 一、今日观察

今天用昨天清仓神通转债和垒知转债的资金，建仓了联诚转债，它距离我预估的下修后的合理价值120-124元尚有空间，而且剩余规模小，历史表现活跃，是我喜欢的三低转债。本来开盘就计划建仓，没想到一路上涨，尾盘才迎来了建仓机会。

### 下修

1、长集转债、岩土转债，提议下修。

长集转债，现价105.899元，剩余年限只有2.11年，接近回售起始日2024-04-08，公司应该是着急强赎了，本次大概率会下修到底。如果下修到底预估转股价值98-100元，预估合理价值116-118元左右。

岩土转债，现价107.791元，2024-03-15就到期了，到期赎回价108元，曾经两次提议下修到底也没能实现强赎的美梦，现在下修属于是孩子没了你来奶了，本次大概率会下修到底。如果下修到底预估转股价值97-99元，预估合理价值108元左右，不会有大波动。

2、山鹰转债、鹰19转债，宏丰转债，宏微转债，本次不下修。

特纸转债，聚隆转债，未来3个月不下修。

亚泰转债，新港转债，丰山转债，设研转债，未来6个月不下修。

3、明天有5只转债即将满足下修条件，分别是：

> 奇精转债、天源转债、火炬转债、开能转债、智能转债

其中奇精转债个人认为下修概率大一些，不过它剩余年限不足1年，到期赎回价仅113元，博弈下修的性价比不高。
