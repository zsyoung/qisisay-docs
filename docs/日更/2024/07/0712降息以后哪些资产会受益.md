# 降息以后哪些资产会受益

文｜启四

## 今日复盘

昨晚，美国6月份的CPI比预期降得更多，说明高通胀正在消退。很多机构预测，美联储可能在9月份降息。

研究表明，只要经济不衰退，美联储首次降息后6到12个月内，美国股市通常会上涨。

如果9月开始降息，国内的货币政策也会有更多空间，第四季度可能会降息降准。

**说说降息以后，哪些资产会受益：**

1. **长期国债**：无论是中国还是美国，降息会导致长期国债价格上涨。因为新发行的债券利率较低，已有的高利率债券更有吸引力，大家更愿意买这些高利率的债券，推高它们的价格。所以长期国债的波动大，适合波段操作。
2. **港股**：港股在内外资情绪改善下已经反弹，加上海外降息趋势确定，港股有望继续上涨。
3. **地产股**：利率降低，买房成本下降，买房的人多了，地产股自然受益。
4. **医药股**：对利率敏感，降息有利于这些股票的表现。
5. **美股小盘股**：利率低利好股市，但今年只有少数大公司表现好，接下来资金可能会流入小盘股。小盘股对利率更敏感，因为它们更依赖融资，低资金成本对它们有帮助。

**风险：**

我们现在的分析是基于假设美联储因通胀下降而降息，如果是这样，未来6到12个月股市可能表现良好。

但如果美联储因经济或市场问题被迫大幅降息，股市可能大跌。例如，标普500指数在宽松周期首次降息后的250天内平均下跌了20%。
