# 政策底基本确认，回本起飞

## 今日复盘

2800点，轮动策略再次出水了！套在3000点，回本在2800点。

今天早上的会议非常给力，降准+降息+降房贷+解决资金入市问题，全是大利好，**政策底基本确认**。

这有点像2014年底的配资加杠杆。当时由于央行在宏观环境压力下实施了全面宽松的货币政策，从2014年11月开始连续降准降息，很多投资者看到股市上涨的潜力，纷纷借助杠杆进场，导致市场快速上涨，形成了2015年初的牛市。

有些朋友担心三季报的业绩，但这种政策底在估值修复之前，市场一般不看业绩。2014年就是这样的，等到大家确认了业绩底时，牛市已经快结束了，指数已经涨了一倍。

10月以后，财政部、发改委和商务部可能还会有类似的会议，继续释放出财政加杠杆的决心。美联储已经降息了，咱们也没必要继续撑着了，因此，接下来的一个月大概率会政策频出。

此时即使担心回撤，也不宜大幅减仓。

如果加仓的话，基金优先配债宽基，沪深300、中证500、中证1000、创业板均可。

### 套利

H股LOF(160717)，今天盘中溢价率超过3%，有群友问能不能套利。

要知道，基金套利不仅要看溢价率，也要看成交量。

这只基金今天成交量急剧放大，也只有不到170万，平常的成交量只有2-20万左右。

此时买进去套利，一旦几天后基金回撤，很容易卖不掉，所以不建议套利。
