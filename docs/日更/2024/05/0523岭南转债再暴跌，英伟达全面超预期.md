# 岭南转债再暴跌，英伟达全面超预期

文｜启四

## 一、今日复盘

### 英伟达全面超预期

全球AI龙头英伟达公布了2025财年第一季度财报，各项数据全面超出市场预期。盘后股价首次突破1000美元（相当于7250元一股……）。

得益于英伟达的暴涨，最近持仓的全球芯片LOF又开始走出上涨趋势了。

今天英伟达还公布了一则消息——打算拆股，1股拆10股，这样散户也更容易买到了。

参考类似的案例：特斯拉，第一次拆股前，特斯拉股价同样是大幅飙升，一年时间涨幅超1000%（英伟达700%）， 第一次拆股后的一年多时间里，公司股价再度上涨超145%。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240523203254712.png)

英伟达长期看大概率还会有所表现，还远没到顶点，人工智能已经在以势不可挡的速度进入到我们的日常生活中，AI手机，AI PC，AI翻译，AI视觉助手，这些未来都会成为标配。

英伟达创始人黄仁勋表示：“下一场工业革命已经开始。企业和国家正与英伟达合作，将价值数万亿美元的传统数据中心转向加速计算，并建立一种新型数据中心——人工智能工厂，生产一种新商品：人工智能。”

但也要提醒一下，想立马梭哈纳指科技或者全球芯片的小伙伴，不要太上头，长期上涨的股票短期可能套死一大批人。

### 岭南转债继续暴跌

去年的10月15日，鸿达转债暴跌12.61%，今天的岭南转债暴跌14.18%，三天跌去了31%+。正股再度跌停，只需要再来4个跌停股价就跌破1元了。

历史又一次昨日重现，去年写的这段文字，改个名字就能再发一遍，道理几乎一模一样。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240523203240713.png)

昨天亏了近100万的博主，今天又在继续买入，并称等着50、40、30元的岭南转债，目前已经累计在岭南转债上亏了几百万。这样算下来，他至少持有了上千万的岭南转债，几乎压上了1/3仓位甚至半仓。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20240523210728657.png)

这让我有种炸金花拿了一把235，然后杀红眼赌上全部筹码的感觉，虽然无法理解，但我大受震撼。

希望岭南能走出困境吧，岭南违约，对所有转债人都不是好事儿。

### 下修

1、**纵横转债**，公告了下修结果，新的转股价值103.757元，当前价格125.199元，合理价值124-126元，价格已体现下修到底预期。

纵横转债虽然规模不大，但是波动一直很小，相对比较惰性，年化波动率才9.86%，在通信行业中它是垫底的，最高的中贝转债年化波动率82.59%。

**维尔转债**，公告了下修结果，新的转股价值64.466元，未下修到底，当前价格96.7元，个人认为合理价值88-92元。

公司剩余年限1.89年，4月15日进入回售期，最后两个计息年度，如果任意30个交易日转股价值低于70元，转债持有人就有权将转债回售给公司，公司必须以100元+当期利息回购。

目前博弈维尔转债的投资者，都在赌公司会将转债拉上100元避免回售。

其实如果公司不想触发回售条款，除了拉升股价之外还有2个解决办法：一是不断下修拖时间，因为一旦转股价改变，回售周期就要重新计算；二是更改下修条款，支持破净下修（听说公司有这个打算）。

除非第2个办法实现，否则很长一段时间，维尔都难上100元。

2、台21转债，未来6个月不下修。

3、明天又有一大波转债将满足下修条件，其中大参转债、福立转债、尚荣转债、旗滨转债、科顺转债、科数转债这6只转债下修动机更强一些。
