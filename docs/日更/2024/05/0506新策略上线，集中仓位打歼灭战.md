# 新策略上线，集中仓位打歼灭战

文｜启四

## 一、今日复盘

五一假期，在社群直播讲解了新的轮动策略，今天把三低轮动策略的仓位进行了替换，另外也把大饼仓位清仓了4只，目前除了轮动策略的转债，只剩下6只转债：3只大饼转债+1只短期博弈债+2只网格债，本周也计划清仓掉。全部转移到轮动策略。

注意：我的清仓并不是不看好大饼转债的**未来**，而是我觉得**当前**新策略选出的转债性价比更高，要集中仓位收缩战线打歼灭战。

### 为什么要替换新策略？

1、新策略无论是历史还是新规后的表现都优于三低轮动，更加攻防兼备，且更容易抓到脉冲收益。

- 新策略：

  历史总年化收益65.74%，最大回撤-13.96%。 

  新规后年化收益43.27%，最大回撤-13.96%。

- 三低轮动： 

  历史总年化收益59.16%，最大回撤-17.08%。 

  新规后年化收益16.23%，最大回撤-16.62%。

2、国九条新规后，对小微盘股形成打击，三低轮动对应正股均是小微盘，回撤会较大，新策略通过增加防守因子降低了回撤，通过增加波动率因子让策略更容易获取脉冲收益。

3、三低转债被越来越多的人所熟知，买的人多了，策略会逐渐失效。一个策略的高光有效期就是1-1.5年，要么接受策略短暂的失效，要么不断根据市场去调整优化自己的策略。

从2022-04-26开始做三低转债以来，一共分为三个阶段：

**第一阶段：2022-04-26～2022-11-06**

2022-04-26我们开始使用三低策略摊大饼，只要进入名单前15就买入，设置回落条件单达到135元就卖出。但是2022-08-01之后，转债8月新规打击转债炒作行为，转债中位数创新高进入一星级高估区，再加上股市下跌，三大利空导致，8月份之后策略回撤严重。

这一阶段收益率30%，最大回撤7.4%。等权指数涨幅6.73%，超额收益23.27%。

**第二阶段：2022-11-06～2023-09-07**

2022-11-06，第一版三低轮动开始，三低因子日轮动，选择了回测表现最均衡的150元阈值。从摊大饼改为轮动的目的，一是为了优化策略，提高收益降低回撤；二是为了解决大部分群友跟不上摊大饼的问题。

摊大饼问题包含两方面，一方面是摊大饼要求更多的资金，资金利用率更低；一方面是摊大饼存在止损难题，大饼会越摊越多，最终手里只剩下惰性债。

同时，我调查了社群成员资金情况，接近一半的群友资金在5万元以下，所以开放了5万元实盘展示账户，方便大家跟单。

这一阶段收益率17.84%，最大回撤5%。等权指数涨幅-1.23%，超额收益19.07%。

**第三阶段：2023-09-07～2024-04-30**

2023-09-07，第二版三低轮动优化，这次优化的目的同样是为了提高收益降低回撤，同时解决轮动中的惰性债问题。

经过大量回测后，优化的方案是：去掉了沪市转债，增加了日内止盈。

这一阶段收益率4.89%，最大回撤17%。等权指数涨幅-5.39%，超额收益10.28%。

**三个阶段累计超额收益52.62%。**

有些人喜欢说：“回测没输过，实盘没赢过。回测都是看历史，对未来没有什么参考意义。”

其实自己择股本质上也是在做回测，看历史表现去预测未来，否则自己为什么不丢飞镖随便乱选呢？

事实证明，我不仅回测赢了，实盘更是大赢。
