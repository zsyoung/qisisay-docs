# 又接回来这2只转债

## 今日复盘

1、股市今天热点还是集中在昨晚说的两个方向上：AI眼镜和大基建，前者代表股瀛通通讯，后者代表股西藏天路今天继续涨停，但是转债前期透支了涨幅，今天均大跌，我在尾盘根据[轮动策略](https://mp.weixin.qq.com/s/sCWtk03GSxs2GhDRV6U6rg)信号又接回了瀛通转债，昨天的止盈恰到好处。

2、转债今天又是妖债的天下，伟24转债、晶瑞转债、严牌转债、正裕转债、溢利转债盘中都有脉冲。

次新妖债赛龙转债、泰瑞转债盘中先急跌再拉，游资意欲洗出一波散户。

做短线的话，优先关注流通规模2亿规模以下的妖债或次新债。

做长线的话，也建议优先关注小规模转债，小规模绝大部分时候都天然具有流动性溢价。

3、转债中位数今日再度下跌，来到了106.624元，**距离年内最低点2月5日的106.05元一步之遥**。

如果没有死抱着低价债（100以下）不放的话，现在的收益大概率不会比年内那波低点更低。

当时我的策略累计收益率-16.83%，现在是-1.35%。

这也是我一直没有放弃转债的原因，再恶劣的环境，再困苦的条件，转债总有能赚钱的方法：埋伏下修、博弈下修后估值回归、博弈回售、埋伏配债股、抢权配债、热点转债轮动、三低转债轮动、短线博弈高价妖债、长线埋伏低价妖债、博弈临期债等等等等……

只是行情好的时候赚得快赚得多，行情差时赚得慢亏得少，掌握方法做好止损想要大亏真的很难。

现在的转债市场，很明显还是极度低估的，不过低价债缺乏机构资金的注入，情绪依然极寒，大家还是需要预防低价债回撤，此时不宜左侧重仓抄底。

我的做法是除非确定性较强，否则不碰100元以下的转债。

我的轮动策略从7月份开始，也保留了一份现金，9成仓位轮动，等待情绪回暖。

万物之中，希望至美。假如所有人都抛弃了转债，那就让我做最后的坚守者吧……
