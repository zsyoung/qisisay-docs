# 黄金又创新高，这两类ETF可以关注

## 今日复盘

上周五，伦敦的现货黄金又创了历史新高。今天黄金ETF大涨，主力资金大幅流入。

现在有三个主要原因在推动金价上涨。第一是地缘冲突的不确定性，全球避险情绪升温；第二是美国通胀压力加剧，大家预期美联储会降息；第三是全球央行继续大量买入黄金。综合来看，黄金作为避险资产的价值又开始显现，黄金价格应该还有继续上涨的空间。

**黄金影响因素有哪些？**

1. **美联储的动作**：如果美联储继续降息或者保持低利率，金价可能会继续上涨，反之则可能承压。
2. **美元强弱**：黄金和美元通常是反向的，美元弱，黄金涨。
3. **全球局势**：地缘政治风险，比如说战争、贸易争端，都会增加市场对黄金的避险需求。
4. **矿业公司的表现**：黄金股ETF的表现还取决于矿业公司的运营情况，比如生产成本和管理效率。

如果关注黄金的话，可以分别看看黄金ETF和黄金股ETF。

比如黄金ETF华夏（518850）跟踪的是上海黄金交易所黄金现货价格，密切跟踪黄金现货价格，随着金价的波动，ETF也稳步上涨。

而黄金股ETF（159562）跟踪的是中证沪港深黄金产业股指数，主要持有黄金矿业公司的股票，这些公司在金价上涨的环境下盈利增加，股价也随之上涨，推动了ETF的整体表现，俗称**金价放大器**，波动更大。

总的来说，如果你更关注黄金价格本身的变动，并希望减少股票市场带来的额外风险，那么黄金ETF华夏（518850）可能更适合你。相反，如果你认为黄金行业的股票有较好的投资机会，并且愿意承担相应的市场风险，那么黄金股ETF（159562）可能是更好的选择。
