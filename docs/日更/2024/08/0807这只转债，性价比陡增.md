# 这只转债，性价比陡增

## 今日复盘

1、今天两市继续缩量上涨，市场开始涌现出一些老热点：

1）**卫星互联网** +4.81%，理由：“千帆星座”首批组网卫星“一箭18星”顺利升空入轨，代表债：盟升转债、沪工转债、华体转债

2）**光纤概念**+5.34%，理由：海外光纤公司LumenTechnologies连续大涨。代表债：通光转债、永鼎转债

3）**智能电网**+2.33%，理由：三部委印发《加快构建新型电力系统行动方案（2024—2027年）》。代表债：通光转债、九洲转2

九洲集团：公司生产兆瓦级光伏逆变器，成本同特斯拉相当；拥有化学储能、物理储热等一系列电力和能源的变换与控制的核心关键技术，可为客户量身定做发电侧和用户侧电化学储能解决方案等在内的智能电网系统解决方案

2、**新星转债**，正股深圳新星3天2板，原因是子公司拟剥离部分六氟磷酸锂项目资产，公告给出的解释是：“近年来，由于六氟磷酸锂产品价格大幅下跌，六氟磷酸锂业务板块持续处于亏损状态，且短期内经营业绩很难有较大的改善。为提高公司整体经营效率，进一步优化资产结构，聚焦优质主营业务，改善财务状况，公司全资子公司拟剥离部分六氟磷酸锂项目资产。”

深圳新星已经连续2年亏损，这是股价大幅下跌的主因。我查了财报，2023亏损1.41亿，主要是财务费用增加和六氟磷酸锂业务板块亏损导致的，光六氟磷酸锂业务板块就占了1.17亿（毛利同比下降9,296.28万元及计提六氟磷酸锂资产减值准备2,414.12万元），财务费用同比增加2,019.65万（因2023年度短期借款及长期借款增加导致）。

剥离了这个亏钱的业务板块，主营业务铝晶粒细化剂是赚钱的，2023毛利7,748.04万，公司2024年大概率扭亏为盈，这对投资者来说是个好消息，而且铝晶粒细化剂可应用于航空航天，刚好是近期的热门概念，所以近期股价大幅上涨。

如果能扭亏，新星转债直接从绩差债变成了盈利债，到期赎回价120元，目前110元+6%的溢价，性价比就太高了。
