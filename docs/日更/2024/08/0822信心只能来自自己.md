# 信心只能来自自己

## 今日复盘

1、又是下跌的一天，当前市场抛压主要来自三个方向：

> 1，刚解除封闭期的基金的被动赎回
>
> 2，被限制无法执行原策略的量化资金持续退场
>
> 3，为了避险或故意做空的主动卖出砸盘的外资

雪球上有人统计了近期解禁或即将解禁的封闭基金，无一只收益为正，且亏的最狠的是规模最大的，也就是说3年前最被看好的基金如今表现是最差的，绝大多数投资者都看走眼了，这也从另外一个角度说明了：**随大流往往是错的。**

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/19178975d71baaf3fd8ba469.png!raw.jpg)

现在的大流是什么？90%的投资者都在看空A股，这时候骂A股明显比做心理按摩更容易引起共鸣。

很多人一边骂着A股，一边又留在A股，或者一边骂着A股，一边希望A股来一场暴跌好让自己骨折价抄底。

理性的投资者不应该被这种情绪带偏，首先要看手里的资产有没有风险和问题，其次要“三知道”——知道涨多少，知道跌多少，知道期限多长；如果一个都不知道，那就必须抛出了。最后不要轻易加仓，加仓的标准只有一个，大盘向好，个股企稳反弹，突破下行趋势后企稳，重新开始走强。当你想加仓时，不妨先问问自己，如果是新开仓，自己会不会进场，不会，那就不要轻易加仓。

大家都知道现在一定是普跌低估的区域，很多优质股票的市盈率都跌到了令人发指的地步，但是又不知道底在哪里，割了怕涨，拿着怕跌，这是现在市场情绪极其低落的原因。

这个时候，希望和信心来自哪里？只能来自自己。

如果自己有给资金判“无期徒刑”的能力，自然可以熬可以从容等待，如果本身就是借钱投资，这时候恐怕真的熬不起。

4、岭南股份发布了《[关于岭南转债持有人参与收购方案网络投票通知](https://static.cninfo.com.cn/finalpage/2024-08-23/1220952524.PDF)》，如果还持有岭南转债，且愿意被中山国资收购的话，**必须通过网络投票系统投出“赞成”票**。收购价格为100.127元/张，收购数量为每人每身份证1000张及以下，多账户合并计算。信用账户中的岭南转债可能存在权利受限情形，不被收购。

**投票方式**：一、券商手机或电脑客户端（一般路径：交易——更多——网络投票，找不到的话问客户经理）；二、通过以下网址投票：http://wltp.cninfo.com.cn（还需要身份认证，比较麻烦）

**投票时间：**方式一：2024年8月23日至2024年9月9日的交易时间；方式二：2024年8月23日上午9:15至2024年9月9日下午15:00，非交易时段也可以。
