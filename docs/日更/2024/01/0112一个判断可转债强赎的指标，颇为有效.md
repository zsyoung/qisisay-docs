# 一个判断可转债强赎的指标，颇为有效

文｜启四

## 一、今日观察

今天突然发现了一个判断可转债强赎的指标，颇为有效，那就是看其账上的**货币资金余量**，是否接近或大于可转债的剩余规模。

还是老规矩，数据说话！

我分别统计了近期强赎和不强赎的可转债，他们发布公告时的剩余规模和正股货币资金余量数据。

近期强赎的转债：

- 转债名称  剩余规模  货币资金余量
- **川投转债**  21.299亿  21亿
- 大业转债  1.388亿  8亿
- 深科转债  2.395亿  3亿
- 法本转债  3.709亿  4亿
- 胜达转债  2.657亿  16亿
- 多伦转债  2.051亿  3亿

- 东湖转债  8.956亿  39亿
- 亚康转债  1.518亿  2亿

近期不强赎的转债：

- 转债名称  剩余规模  货币资金余量
- **巨星转债**  9.155亿  6亿
- 耐普转债  3.997亿  5亿
- 新致转债  2.914亿  4亿
- 润达转债  4.028亿  9亿
- 北方转债  5.338亿  43亿
- 联得转债  1.472亿  6亿
- **锦鸡转债**  2.626亿  1亿
- 爱迪转债  15.698亿  23亿
- 精测转债  3.084亿  11亿

可以发现，上述转债中，除了川投转债剩余规模接近货币资金余量，巨星转债、锦鸡转债剩余规模小于货币资金余量以外，其余可转债货币资金余量均大于剩余规模。

所以，**当货币资金余量大于或接近剩余规模时，强赎意图不好判断，但是当货币资金余量远小于剩余规模时，则很难强赎。**

因为可转债公告强赎时，公司需要缴纳全额保证金，也就是强赎价格对应的转债余额，比如大业转债的强赎价格是101.6元，转债剩余规模是1.388亿，则转债余额是1.388*（101.6/100）=1.41亿。

这意味着公司需要准备一大笔保证金，来保障投资人的权益，避免出现违约情况。

虽然这个规定并没有写在发行文件中，但是实际操作时一般是这样执行的。

至于怎么快速查找对应公司的货币资金，还是我们的老朋友——同花顺i问财。
