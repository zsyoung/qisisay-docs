# 才发现！至少50％的人都不会这个投资技巧？！

## 今日复盘

A股三大指数今日涨跌不一，截至收盘，沪指涨0.29%，深成指涨0.33%，创业板指跌0.11%，北证50指数跌1.41%，全市场成交额17984亿元，较上日缩量4295亿元。全市场超3800只个股上涨。

昨天市场高开低走，今天又恢复了平淡节奏。不过小票挺争气，像1000、2000指数，几乎反包了昨天回落的涨幅。看这架势，游资果然是最有冲劲的玩家，做多的热情一骑绝尘。

大票就没这么“上进”了。红利指数、沪深300、A500全都表现平平，这些是机构偏爱的领域，但今天明显熄火了。其实想想也不意外，年底了，基金经理们大多选择按兵不动。毕竟，随便动一动就可能影响年终绩效，没人愿意在这个节骨眼冒险。所以干脆放平心态，等着市场风格自己切换。

再看看成交量，缩到1.7万亿，比昨天冷清不少。昨天那股情绪高涨的热闹劲儿一过，资金又开始观望，眼睛都盯着即将召开的经济工作会议，等一个明确的信号和方向。

不过，整体来看，市场情绪还不算差。每次下跌，总有资金进场抄底，修复得也挺快。这说明，市场里还是有愿意尝试的力量在活跃。但要说对经济基本面的信心，那还真是差点意思。政策虽然已经在发力，但力度似乎还不足以让机构们完全相信“经济问题很快能解决”。这里头就形成了一个预期差。

当然，正如昨天提到的，**该涨的终究会涨，只是时间问题。**回顾2008-2009、2014-2015和2019年，股债双牛的窗口期往往出现在政策强刺激的“非常规时期”。这次“双宽周期”（货币宽松+财政宽松）已经在酝酿，大概率会再次催生股债齐飞的局面。
