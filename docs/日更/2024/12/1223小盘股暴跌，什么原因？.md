# 小盘股暴跌，什么原因？

## 今日复盘

A股三大指数今日集体下跌，截至收盘，沪指跌0.50%，深成指跌1.03%，创业板指跌0.98%，北证50指数跌2.64%，微盘股指数跌7.13%，超300只个股跌超9%。全市场成交额15457亿元，较上日放量129亿元。

![](https://files.mdnice.com/user/7588/4d7389d2-0d32-4f80-a63a-a540bc13276b.png)

今天市场“小盘切大盘”的迹象非常明显。大盘股还算稳得住，比如银行、石油石化、公用事业这些股息率高的板块收盘竟然是红的。但小盘股就惨了，1000指数跌了2.8%，2000指数更狠，直接崩了4.88%。微盘股更是领跌冠军，指数跌了7%，看着有点像去年初那波小盘股“爆仓”的感觉。

**为啥小盘股今天崩得这么惨？**主要是几个原因：

一是周末在各大投资群疯传的“退市风险名单”，直接把市场情绪吓崩了。

二是马上就到2025年1月1日了，史上最严的“退市新规”即将实施，资金赶紧提前撤离，而这些公司偏偏又是小盘股里的“业绩差生”。

三是小盘股从7月低点到现在，涨幅已经快翻倍了，我跟踪的小盘股组合甚至涨幅140%+，获利盘砸出来止盈也是正常操作。

不过，今天债市也跟着跌了，这说明资金并没有跑去债市避险，主要还是市场情绪引发的杀跌。成交量依然达到1.5万亿，说明市场里的钱还挺活跃的。短期虽然跌得猛，但抄底资金很可能随时杀回来。所以，也没必要太悲观。

还有种说法是，每年的微盘股1、4月份必跌，因此转债也应该清仓，这种说法是比较片面的。大跌一般都是熊市年，牛市年要么不跌，要么是大涨的。我记得今年某大V在4月就因为日历效应提前做了规避，最终的结果也证明是错误的。所以，大跌不完全是规律，得分情况。

我的转债策略，23年1月涨幅5.24%，24年1月杀的比较狠，下跌了12.47%，不过24年是量化踩踏+新国九条利空微盘股+违约信用潮导致的，今年在牛市化债的背景下，再来一波40%-50%的跌幅可能性不大。而那些风险股，比如灵康、声迅、起步、普利这些，早就被市场扒拉干净了，它们大面积带崩其他转债的可能性也不大。

具体到个债层面，短期只要没跌破30日或60日均线这种关键支撑，就不用太慌。

它刚刚上市不到1年，它的上一任卡倍转债强赎的也很快，只存续了1.2年。

据说卡倍亿正在筹划发行卡倍转03，大股东明显是尝到甜头了，发债-炒作-强赎玩得风生水起。
