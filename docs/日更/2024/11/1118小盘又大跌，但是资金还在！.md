# 小盘又大跌，但是资金还在！

## 今日复盘

A股三大指数今日集体下跌，截至收盘，沪指跌0.21%，深成指跌1.91%，创业板指跌2.35%，北证50指数跌3.64%，全市场成交额17960亿元，较上日缩量673亿元。全市场超4100只个股飘绿。

![](https://files.mdnice.com/user/7588/6b0ba00d-c7d6-4cd0-a983-aeaf60409102.png)

最近的市场就像每天都在演一出“上午还有点希望，下午就撑不住了”的戏码，这种走势已经持续好几天了。

今天的表现可以用“大小盘冰火两重天”来形容。大盘股还算坚挺，比如红利指数和上证50都收红了，但小盘股就不太行了。像1000指数和2000指数，一天就跌了2.46%和2.95%，北交所更惨，直接跌了3.6%。放到一周的时间轴来看，小盘股已经跌了快10%了。

这种下跌速度确实有点狠，特别是我们转债党，感受可能更强烈，因为转债的正股基本上都是小盘股，受这波行情拖累比较明显。

现在，市场整体处在一个“调整期”，有点像暂停按钮被按下了。这两周没有新的政策，也没啥关键数据出炉，属于典型的“空窗期”。市场缺乏兴奋点，投资情绪自然提不起来。不过不用太担心，月底会有PMI数据公布，下个月初还有政治局会议，这些都会给市场带来新消息，有机会重新点燃做多热情。也就是说，现在的低迷只是暂时的，大家可以稍微耐心点。

为什么说要耐心？因为目前**资金的动作告诉我们，它们并没有离开市场避险，而是忙着调仓换股**。今天30年期国债期货下跌了1.1%，这说明钱并没有跑去债券市场，而是继续在股市内部寻找机会。尤其是像中证全指这样的指数，已经快接近长期支撑位了，跌下去的空间可能不大了。一旦资金调整完毕，市场就有可能慢慢企稳，甚至再度回暖。

从国外的情况看，美元和美债的短期波动差不多已经到头了，对国内成长股的压力也快消化完了。这等于说，外部的“紧箍咒”在松动，给国内市场反弹创造了更多空间。

展望后市，大小盘都有表现机会。大盘股靠稳健的基本面，资金会继续青睐它们；而小盘股经历这一波调整后，反而可能成为成长亮点，尤其是科技股的上涨潜力依旧很大。不过，这种行情下，最好的策略是保持仓位均衡，不要太偏向某一类股票。

最后提醒一下，别因为短期调整就频繁买卖，比如看到小盘跌了就卖掉追大盘，或者看大盘涨了又杀回来。这种追涨杀跌的操作，不仅心累，还可能赚不到钱。不如稳住心态，耐心等待市场恢复。
