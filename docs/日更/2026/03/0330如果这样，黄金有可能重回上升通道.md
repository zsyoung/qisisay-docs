# 如果这样，黄金有可能重回上升通道

## 今日复盘

1、如昨晚文章所说，今天市场果然没有如大部分人预期的全天大跌，反而低开高走，中证全指收盘还是红的。相比美日韩的大跌，A 股一枝独秀。长债今天也全天收红，主要得益于央行释放了流动性，政策在极力呵护，市场的牛市预期还在。

2、成交额方面，两市成交额1.93万亿元，虽然较上周五放量约638亿元，但整体仍处于缩量反弹的范畴（**重回2万亿以上才是强信号**）。意味着场外资金还在观望，并没有大举进场抢筹，目前的上涨更多是存量资金的博弈。如果明天不能补量，今天的反弹可能只是“下跌中继”。

3、情绪面方面，全市场超2800只个股上涨，虽然涨多跌少，但空头阵营依然庞大，今天又新增了3只空头。资金正在从“防守型红利”（电力）流向“通胀型资源”（有色、油）和“超跌成长”（医药）。

4、微盘股经过短暂的流动性冲击，也连续2天站上60日均线。科技小盘一般是对外部风险最敏感的品种，受流动性冲击最大，它们不继续调整，正说明市场对外部的冲击正在钝化。接下来4月财报季将检验各板块业绩成色，市场定价将**回归基本面。**

5、黄金前段时间在经历了一轮快速下跌后，这两天开始企稳反弹了。我们也看到了一种罕见的现象——黄金和原油同步上涨。战争如果这么持续下去，油价控制不住，黄金可能又会回到上升通道。

- 过去的逻辑：油价上涨 → 通胀预期升温 → 美联储可能加息或维持高利率 → 利空黄金（因为持有无息资产黄金的机会成本增加）。

- 现在/未来的逻辑：油价因地缘冲突暴涨 → 被视为对全球经济的“强制性征税” → 市场担忧经济将陷入“滞胀”甚至“硬着陆”衰退 → 预期美联储必须通过降息来救市 → 利好黄金（持有黄金的机会成本下降）
