# 大反攻！但明天可能不太妙

## 今日复盘

1、今天市场大反弹，情况刚好和昨天来了个颠倒。红利和上证 50 跌，科技成长和小盘涨。不过中证全指成交量继续缩量到了 2.01 万亿，还不能过于兴奋，不能做出调整到此为止的判断。

2、其实反弹最大的原因，我认为还是**油价-股市的跷跷板**逻辑。A 股开盘时间刚好油价大幅下跌，油价下跌利好科技成长和小盘，油价上涨资金避险就会冲向红利和大盘。

风险偏好提高之后，资金自然还是看**业绩确定性高**的品种。像今天涨幅最大的AI算力链（光模块、铜箔等）的一季度订单饱满，涨价逻辑清晰，自然成为首选。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260318204348021.png)

3、不过晚上这会，伊朗据称加大袭击中东能源设施，油价又开始大幅上冲了。夜盘黄金大跌，明天早上如果油价不能回落，股市应该不妙。

4、明天是周四，又到了每周定投四维轮动 5000 元的日子，目标是投够 300 万，实现永续现金流。

这是长期策略，设计目标年化 8%-12%，最大回撤 7%以内，思路是复刻达利欧的全天候策略配置——55%的中长债+15%黄金+15%科技+15%红利。

这个配置在中美市场都经过了长期的回测验证，证明了其有效性。长期来看，这些资产都会不断新高，而这样的配置方法，利用了资产间的不相关性对冲波动，从而实现重仓拿住吃到各类资产长期收益的目标。

所以短期的波动无所畏惧，下跌刚好是加仓的好时机。
