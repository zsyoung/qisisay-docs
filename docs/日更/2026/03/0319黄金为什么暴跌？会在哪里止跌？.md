# 黄金为什么暴跌？会在哪里止跌？

## 今日复盘

1、今天还是原油-股市跷跷板逻辑引起的大跌，该死的鱿鱼轰炸了朗子的油气设施，而且没有通知老美，摆明了就是想进一步拖老美下水，把事情继续闹大。

朗子一看你这么狠是吧，想断我活路，那大家都别过了，他也以牙还牙，炸了老美在中东的油气设施。

这么一来，停战预期又得往后拖了，油价短期之内也不可能下来了。

**“高油价=高通胀=紧缩预期”**的逻辑链条一打通，全球股市都跟着遭殃，尤其是高估值的科技股和小盘股。  



2、值得注意的是黄金。

黄金遭遇史诗级双杀，两天时间，从5000美元上方狂泻，昨晚伦敦金跌破了4800美元，今晚更是跌至盘中最低4500美元，累计跌幅近10%，

避险属性好像失效了？其实这段时间一直在失效。

这次黄金大跌，核心原因是两点：

**一是美联储突然变“鹰”**，昨晚的会议维持利率不变，而且直接泼冷水说今年可能只降一次息，甚至不降，导致美元美债收益率飙升。

众所周知，黄金是不生息的，现在这样就导致持有黄金的成本激增（放在美债里可以躺赚利息），所以市场就赶紧抛售黄金换美元。打仗带来的避险买盘反而是疯狂涌入美元避险，而美元走强直接对金价形成了极限施压。

**二是杠杆踩踏**。之前涨得太猛了，获利盘巨大。一旦跌破5000心理关口，触发大量多头爆仓和追加保证金。机构为了凑钱救急，不得不疯狂抛售流动性最好的黄金，形成非理性的“多杀多”。简单说，不是黄金不值钱，是大家急着卖金补窟窿。  



3、那么接下来会跌到哪里企稳呢？

从目前的市场格局和历史数据来看，黄金的支撑位可以看三个层次：

**第一支撑位：**4700-4850美元。这是2026年以来上涨通道的下轨，属于“第一道防火墙”。不过今晚已经失守了，就会变成强阻力位。

**第二支撑位：**4550-4600美元。这是2025年行情爆发前的双重顶转底位置，技术面颈线位，有极强的技术支撑，大概率先在此喘口气。

**第三支撑位：**4280-4300美元。这是2025年底金价突破前的历史大顶。“顶底转换”原则下，昔日的压力位如今就是铁底。重合长期趋势线，也是长线资金的极限心理防线，可以看作终极铁底。跌到这里，大概率会引发抄底资金入场，形成强力反弹。

我的判断是：只要不有效跌破4280美元，黄金的长期牛市逻辑就未破坏。美国债务危机和地缘乱局这两个底层驱动力没变，全球央行仍是这个市场最大的“多头死忠粉”。**现在的暴跌只是杠杆出清，跌出来的全是给长线资金的“黄金坑”。**
