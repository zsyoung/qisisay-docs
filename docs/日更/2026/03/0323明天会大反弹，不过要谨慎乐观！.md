# 明天会大反弹，不过要谨慎乐观！

## 今日复盘

1、今天太惨烈了，可以用尸横遍野来形容，已经很久没有看到过这种场景了。亚太股市全线崩盘，韩国直接熔断，A股五千多只股票下跌。原因就是三个字：**流动性**。

川普周末威胁给伊朗的48小时最后通牒即将到期，而且美方态度从“逐步结束”突变为“准备地面部队登陆”和“封锁霍尔木兹海峡”。伊朗当然也没有要投降的意思。后果就是原油继续拉升，全球担心石油危机引起“滞胀”（高通胀+经济衰退）。

所以今天市场就一个逻辑：别问什么资产好，只要是能卖的，全卖了换现金！这就是典型的流动性危机，恐慌到了极点，资本全都在逃命。

2、让我没想到的是黄金居然在4500美元关口不带喘气的，今天继续疯狂下杀，盘中最低跌破了4100美元，这太不正常了！

3月19日的文章，启四提到黄金有三道防线，分别是4700-4850美元、4550-4600美元、4280-4300美元。两天时间而已，三道防线全破了。

不过最后一道防线还不能算有效跌破，因为盘后川普再一次服软了，声称和伊朗进行了你侬我侬的友好交流，并推迟5天对伊朗的军事行动。

黄金应声大涨，瞬间就回到了4400美元以上。恒生科技、纳指期货全部大幅反弹。论股神，谁能比得过川普？真乃古往今来第一人也！

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260323220215258.png)

3、这时候，更要冷静一点，启四问了自己两个问题。

**第一，这消息靠谱吗？**川普那边说得眉飞色舞，但是很快伊朗那边直接打脸：“根本没谈过！”以色列也一声不吭。这明显是特朗普惯用的“交易式外交”——先吓唬你，再假装缓和捞点政治资本。这种单方面的口头利好，根基非常虚，随时可能因为对方的一句硬话而破灭。地缘政治的火药桶还在，而且短期很难灭火，各方利益诉求很难达成一致。

**第二，为什么反弹这么猛？**不是因为世界和平了，是因为这几天跌得太惨了！黄金、纳指之前被错杀得太狠，现在只要有一点好消息，空头就赶紧回补，加上抄底资金进来，自然会暴力反弹。但是这只能叫“技术性修复”，不叫“趋势反转”。

明天大概率会有一个超跌反弹，不过我觉得还是**不能过于乐观。**尤其是小盘和科技方向。

关税战是川普一个人发动的，他想停就可以停。但是战争一开打，各方利益交织，诉求不一，真不是他一两句话就能按下来的。

4、上周五我的小盘股策略清仓了9成，今天**完全清仓**了。

本来按照策略应该在尾盘接回，4月份再空仓的，但最终还是没接回。

因为盘中趋势已坏，微盘放量下杀，下跌动能太强，微盘股跌幅超过国证2000，说明市场已经进入“多杀多”的流动性崩溃阶段。

如果今天接回，再坚持到4月清仓，其实赌的是这最后一周会有个“超跌反弹”可以卖在高点。

但问题是：现在有谁会进场接盘小市值？ 避险资金后面更可能去黄金和红利，投机资金被美债吸走，小市值马上又到了财报季，很难吸引大规模资金入场。

所以与其赌最后几天的反弹，不如提前止损。

5、说到红利，今天红利指数和红利低波也没能起到防守作用。还是因为流动性挤兑。

在**系统性崩盘**面前，没有任何板块能独善其身。

平时它是机构的“提款机”，里面挤满了机构重仓盘（公募、保险、理财子）。现在市场崩了，成长股跌停卖不掉，机构为了换现金救急，只能砸这种流动性好、之前涨得多的红利股。

再加上打仗引发通胀恐慌，大家觉得以后利息要涨，红利股的“类债”吸引力就弱了。

不过这部分不用过于担心，它的基本面（高分红）长期仍在，流动性危机缓解后，还会不断向上，可以一直持有。

6、至于四维轮动和闲钱罐，都是长期策略，如果打算长期拿，不用过于担心。

闲钱罐是固收+策略，80%短债+20%增强，历史回测数据显示，任意时点买入，持有满1年，盈利概率100%。

四维轮动是全天候策略，历经了两次石油危机、欧债危机、次贷崩盘以及互联网泡沫的洗礼，却始终屹立不倒，穿越牛熊，稳健向上。在我看来，这已经是普通人能**重仓拿住的最优解**。换成其他策略想要博取同样的收益，恐怕得承受剧烈得多的波动。
