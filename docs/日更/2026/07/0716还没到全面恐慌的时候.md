# 还没到全面恐慌的时候

## 今日复盘

1、今天科创50冲高回落，又大跌4.02%，5天跌掉15.51%。中证全指大跌1.96%，再次回到年线下方。但市场中位数只跌0.18%，近2500只股票是涨的，还是科技权重拖指数，不是所有票一起跌。

成交2.36万亿继续缩，融资余额从2.93万亿降到2.87万亿，杠杆开始撤了，但这个幅度还谈不上出清。科技没止住，金融权重也开始转弱，调整范围没有缩小，这个状态持续下去的话，盘中的反弹就走不远。

2、科技前期涨幅斜率太大，所以回调起来也狠。科创50这轮回撤了近18%，创业板50跌了近17%。韩国今天盘后又宣布要继续收紧杠杆ETF门槛，他们的2倍杠杆没消化完，后面再恐慌杀跌一波不意外，A股科技情绪难免也会被带一下。

但我暂时没那么悲观。存储、光通信确实在赚钱，产业趋势不会因为几根阴线消失。等估值和拥挤度降下来，资金转一圈大概率还会回科技。双创和韩国综指不能简单类比，A股内部还有红利、消费、医药这些方向可以分流。

3、有技术派用波浪理论算，创业板50和科创板50可能在牛市中途的四浪调整，常见回调范围15%-25%。韩国综指是完整五浪全部结束，上涨周期终结，走牛转熊ABC大调整。但这东西主观性太强，拿来估算空间还行，不能直接判断底部。大家可以参考一下心里有个数。

4、定量指标方面，中证全指缩量下跌，但还没连续3天跌破年线。定性指标方面，估值不便宜，但还没到离谱的位置，低利率环境下股债收益差还是有吸引力。净利润还在底部往上抬。政策还是宽松的，美联储加息压力也暂时缓解了。

我的判断没变，问题不大，不用被科技连跌吓住。中证全指大概率围绕年线磨2周左右。涨一天别急着乐观，跌一天也别马上看熊。没放量连续破年线，就按牛市中途调整处理。
