# 近200股跌停，但这还不是最坏的信号

## 今日复盘

1、今天这根阴线够狠。沪指跌3.05%失守3800点，创业板指和科创50双双跌超7%，中证全指也跌了4.78%。全市场超过5000只股票下跌，接近200只跌停，成交额放大到2.67万亿。恐慌盘和止损盘一起涌出来，科技杀跌很快扩散到大部分宽基。

但有个细节不能忽略。中证2000跌了6.18%，微盘股只跌4.44%，红利指数只跌0.74%，红利低波甚至涨了0.25%，国债ETF也在走强。资金没有全部离场，只是从高波动方向撤到了防守资产。今天更准确地说，是科技成长集中补跌，风险开始扩散，但还没到所有资产一起失控的程度。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260717184315630.png)

2、这次下跌，第一个原因还是科技前面涨太快。存储、算力、AI硬件吸走了大量资金，估值和仓位都很拥挤。海外芯片股一转弱，最先被卖的自然是前期涨幅最大的方向。

第二个原因来自韩国。这轮科技崩盘从韩国开始，他们刚爆完3倍杠杆，2倍杠杆还没出清，昨天又开始限制三星、海力士这种单只股票的杠杆ETF。去杠杆很难一天结束，后面肯定还有反复，对A股半导体情绪也会形成拖累。

第三个原因是买盘接不住。近期成交额连续萎缩，市场从前面的增量进攻，转成存量资金互相腾仓。加上长鑫科技大型IPO分流资金的担忧，买盘一弱，卖压就集中爆了出来。

3、现在也不能直接确认熊市。一天暴跌只能说明原来的上升结构被打坏了，接下来要进入**牛熊判断区。**

判断区主要观察四个信号。一是中证全指和沪深300能不能尽快收回年线，连续三五天压在下面站不回去，牛市调整的说法就悬了。二是成交量，放量下跌比缩量下跌危险得多。三是科技以外板块能不能稳住，红利银行如果也放量跌，就接近全面收缩。四是海外科技去杠杆什么时候结束，现在还在扩散，不能指望马上企稳。

4、我的态度是，暂时不确认熊市，但风险等级已经上调。现在不适合急着抄科技底，也没必要在暴跌后把仓位一次砍光，尤其是非科技方向的仓位。先等最拥挤的方向把杠杆和情绪消化完。短期看，连续暴跌后，下周大概率有技术性反抽，沪指3800点、创业板指5日线是观察位，但反抽不等于企稳，右侧信号没出来之前先别急着上。

5、中期我仍然偏向牛市中场休息。缓涨急跌是牛市常见走法，当前估值和杠杆没到失控的位置，政策宽松和业绩修复的方向也没变。这轮下跌更多是估值和情绪出清，杀的是融资盘，AI产业长期趋势没有根本逆转，如果中报能验证真成长，调整反而给了入场机会。
