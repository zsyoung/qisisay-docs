# 牛市才是散户亏大钱的时候

## 今日复盘

4681只股票下跌，跌停187家，全A中位数跌4.18%，创业板跌3%，深证成指跌3.5%，上证指数跌2%。今天太惨了！

所有指数都被打到十日均线下方，中证全指也干到了年线以下，上一次这么惨还是今年的3月23日，如果3天之内不能收回年线，就比较危险了。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260713210325307.png)

今天大跌的原因，主要还是韩国的海力士业绩不及预期，韩国一熔断，科技跟着倒，A股也很难独善其身，而且港股科技也在跌，这波是全球性的避险情绪释放。

比科技崩盘更难受的是中小盘大面积溃散，沪深300跌1.8%，中证2000跌5.7%，小盘股跌了权重股的三倍。银行今天还在扛，但谁都看得出来是在托指数。说明风险已经不是某个板块的问题了，高波动方向基本都在挨打。

成交额2.8万亿，比昨天少了快6000亿。缩量下跌，看起来好像抛压在减弱，但跌停数还在扩张，这就不是好事。缩量只能说明买盘也不愿意进场了，不是卖盘累了。换手没完成之前，还不能说止跌。

除了四维轮动这种全天候配置思路，今天什么策略都得跌。

总的来说，现在还不能判断风险出清，盘后也没有超预期的利好出现。

虽然从中线看，极端缩量加上这种级别的恐慌，往往是卖方快被榨干的信号，但是也别急着抄底，因为拐点不是猜的，是等出来的。

先关注海外情况，韩国和港股半导体不喘口气的话，A股想独善其身太难。

然后明天看A股跌停数，如果继续增加，那风险出清就还没完；如果开始回落，再看A50和沪深300能不能站稳。

最后还是想感叹一句：牛市才是散户亏大钱的时候，波动越大，投资越难做。
