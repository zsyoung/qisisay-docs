# 港股打新，决定胜率的2个核心指标

## 今日复盘

上周港股打新中了一手曦智科技，上市首日涨幅高达383.62%，我暗盘提前卖到了次高位，一签赚了9500多港币。

明天又有一只超高热度的天星医疗要打，认购倍数目前已经接近4000倍了，很有希望超越曦智科技的5784倍。

即使超越不了，也已经锁定了超高热度，参考可比公司【威高骨科】的上市首日涨幅111.72%，它上市首日大涨基本没跑。

港股打新分析说难也难，说简单也简单，我最重视的核心指标就两个：**认购倍数、基石阵容**。

### 一、认购倍数

认购倍数几乎决定胜率，是港股打新的第一判断维度。

1、大盘股 + 认购500倍以上，胜率可达90%。

2、小盘股 + 认购1000倍以上，也能稳稳赚钱。

3、认购200倍以下的项目，基本可以放弃，或者纯娱乐。

4、市值100亿港元以上为大盘股，是安全边界线。

### 二、基石投资人判断

从基石阵容能看出项目成色，老板裙带关系多为“自导自演”，真正的含金量在机构背景。

1、首选：**中信系 + 高瓴 / 红杉 / 淡马锡 / GIC** 等重量级基石。

“中信系 + 高质量基石” = 几乎稳赚组合。

“高瓴 + 淡马锡 + 中信建投” 三组合，可直接闭眼打（前提是定价合理）。

2、题材不是重点，定价才是灵魂——越低估越安全。

3、基石比例和锁定期越长越稳，尤其是锁6个月以上的。“FA类小投行 + 无锁定期基石”十有八九要破发，这类直接放弃。

4、保荐人靠谱最重要：中信、摩根等一线体系，质量有保障。

### 三、一些经验法则

1、**铁律**：发行价相对存量市场折价 ≥15%时才出手，否则一律当骗子。

2、**二婚股原则**：二次上市除非折价极高（≥30%），否则直接忽略。
