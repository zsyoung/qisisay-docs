# 明天打算建仓

## 今日复盘

明天打算建仓小盘股策略了，理由有以下几点：

1、**风险出清。**现在采用的日历空仓法，躲开每年12月、1月、4月这三个小盘股胜率低的月份。4月马上就结束了，策略的持仓股年报、一季报已全部披露完毕，风险已充分出清，具备布局条件。

2、**流动性宽松。**小盘股策略核心中的核心就是**流动性**，看流动性最主要是对货币政策进行定性分析，现在货币政策依然宽松，没有打压股市的迹象。我最担心的是创业板股指期货正式上市，应该会成为牛熊拐点，大概还有半年到一年的时间，现在还可以继续嗨。

3、**估值不危险。**随着A股持续扩容，2010年中期以后，大盘股的市盈率一直呈现窄幅波动，在8-16倍波动，之前在12-48倍之间波动，所以现在从估值角度去判断大盘股的高低，参考价值不是很高。
