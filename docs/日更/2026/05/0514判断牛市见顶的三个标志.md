# 判断牛市见顶的三个标志

## 今日复盘

今天群友梦想哥发了一张抖音文字截图，问是不是牛市见顶了？

截图里的大意是：某散户说现在市场上5437支股票从924以来几乎全部翻倍，已经高估的不能再高估了，能撤就撤吧，赶紧跑，别怪我没提醒你之类的话。

这类纯情绪发泄，逻辑和事实都站不住脚的内容，居然还收获了一片点赞。

且不说事实错误——我查了一下，截至2026年5月14日，自924行情启动以来，翻倍股票1989只，占比不到37%。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260514215334287.png)

就单说这个逻辑——单纯靠股价翻倍、估值高低来判断牛市是否见顶，本身就很容易证伪。

**我个人判断牛市到顶的标志，主要看三个：**

第一，流动性&政策收紧。所有大牛市终结，根源都是：钱不再变多，反而开始收水。中美当前都是：宽松周期余温、没有任何主动收水杀市场的动作。

第二，大型做空工具落地，机构大规模对冲砸盘。牛市中后期，一旦配套完备做空工具上市，机构从“只能做多”变成“多空双杀”。目前创业板的股指期货还没出来。

第三，不可控超级黑天鹅，打碎全球经济预期。比如全球疫情那样的不可控事件。美伊战争其实也可以算作黑天鹅突发事件，但是现在看已经缓和了。

其实还可以补充两个：

第四，情绪估值极致泡沫。全市场散户疯狂进场、新股无脑爆炒、垃圾股遍地翻倍、公募基金单日爆卖千亿、人人谈股赚钱。现在情绪远远没到疯狂泡沫期。

第五，产业主线彻底透支。一轮牛市主线（比如AI、科技）所有逻辑全部兑现，利好出尽，没有未来叙事空间。现在AI、算力、半导体还在业绩兑现路上，远没透支。

**但是这两个指标不容易跟踪和量化，有滞后性。**

……

以上是基于全面大水牛的情况下的判断，另外还有一种可能，就是未来不会有全面性的大牛市，会呈现结构性行情，就是一边使劲涨，一边使劲跌或者震荡横盘。

比如美股就是这样的，科技一直在涨，但是传统工业、夕阳板块常年不涨甚至阴跌。

**这种情况下，散户有两条路：**

要么去做动量趋势投资；

要么定投如A500、沪深300、创业板、科创50这类核心宽基指数，不用选股，靠指数定期换血、淘汰垃圾、纳入新龙头，长期震荡走高。

关于后一条路，我有两个例子想讲。

第一个例子，我记得924牛市来的时候，有人跟我打赌，创业板绝对不会创新高的，因为里面都是医疗股和光伏股，医疗股受到集采的影响，光伏和电池内卷严重，永远不会再回归历史高点了，现在再看呢？创业板不是依旧新高了。

可能某些个股永远不会再回归高点，但指数的成分股是不断新陈代谢的，靠着科技股的强势崛起，创业板最终还是能创新高。

第二个例子，当初寒武纪纳入上证50的时候，很多人都在喊已经高位了，这时候纳入就是骗散户去接盘的。现在再看呢？寒武纪又翻了一倍多。

2023年12月，纳入沪深300，价格约170元；

2024年12月，纳入上证50，价格约568元（+234%）；

现在2026年5月，1260元，累计涨了大约6.41倍……

长期来看，指数比人“聪明”的多。
