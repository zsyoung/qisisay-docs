# 到底是谁“陷害”我买了恒生科技？

## 今日复盘

1、先说件开心的事。节前根据ETF轮动策略信号买入的黄金ETF，节后给了一个大红包。从1月27号到现在，经过连续6次操作，4次亏损2次盈利。

虽然中间过程十分狰狞，就像在天堂和地狱之间坐了一趟过山车，但结局不错，总算是盈利的。黄金这一波大涨大跌，既极大锻炼了我的心态，也让我对策略有了更大的信心。有极其优秀的历史回测数据打底，有明确的买卖指标，我需要做的无非是坚定执行，承认自己不如量化即可。

2、相比之下，恒生科技就是个反面教材。看着账户里的浮亏，我经常陷入沉思：我当初到底是怎么看上它的？是朋友介绍、平台推荐、偶然发现，还是仇人陷害？现在想想，当初那个拍脑门进去的自己，可能就是最大的“仇人”。买之前做过“**三知道**”评测吗？知道顶、知道底、知道时间吗？

实话实说，当初就是看到它在模型里超跌了，又恰好跌到强支撑，再被外界一波“AI叙事”洗脑，就鬼使神差进场了。虽然有过短暂盈利，但现在吃的一嘴泥。亏损额度对我来说并不高，但主观买入的标的最累人，天天要为他人解释为什么涨、为什么跌，不胜其烦。没有明确指标的每一步操作，未来会形成什么效果，心里其实是虚的。

3、恒生科技这种磨底阴跌法，之前没有遇到过，确实让人恶心。最近我也在反思，我到底是做左侧配置还是右侧趋势？如果是左侧，那就长持，如果是右侧，早就该走人了。之前算的那些精细点位，实战中除了磨心态没什么用。想来想去，我决定不折腾了，要走就走，要留就留，在几百个点里反复被抽耳光太低级。

4、既然不折腾了，那就要分析到底恒生科技还有没有救。现在综合各方材料来看，逻辑还没崩。目前的下跌不是基本面垮了，而是**流动性错配**。股价在跌，汇率在涨，离岸人民币顶着美元高压强拉到 6.88，这就是典型的股汇背离。这种割裂说明有人“不得不卖”，可能是海外某些基金触发了强平，或者是仓位再平衡的被动调仓。这种因为流动性枯竭导致的下杀，往往就是最后最疼的那一哆嗦。

5、不过还是要客观看待“**技术性熊市**”的事实，我认怂，但不投降。现在我有12%的仓位，如果恒生科技指数跌到 5250以下，就补回4%，重新加到16%。这4%不谈信仰，涨三五个点就撤，因为大概率在重要均线处，恒生科技还要经历很多次阻力。

6、关于**止损**，我定死两点：一是人民币汇率别贬破 7.0，那是宏观底线；二是指数别连续三天收在 5200 下方，那是技术底线。不满足这两条，盘中所有波动都是噪音。

恒生科技真正的**命门**是：政策底、业绩底、流动性真正转向。与其在震荡里被洗出来，不如把判定权交给更底层的汇率。
