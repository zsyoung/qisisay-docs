# 隐藏羊毛

## 今日复盘

今天策略差点止损了，弥补了一把没有体验过死刑前夜的遗憾，太刺激了！

全球资产最近都被美联储加息预期折腾得不轻，股债商罕见三杀。尤其是黄金和原油，近20日都跌了13%左右。

归根结底还是三个字——**流动性**。几乎所有资产涨跌、行情分化，都可以归结到这三个字上。

**1、为什么美联储加息预期影响这么大呢？**

因为美元是全球“货币之锚”，美债利率是全球无风险定价基准。

美联储加息意味着全球流动性抽水、资产全面重定价。

这就导致新兴市场包括中国，面临资本外流、汇率承压、股市债市缺钱的困境，21年的牛市就是这么结束的。

黄金是无息资产，持有的成本就是美债利率上行的机会成本，既然放在美债里更赚钱，钱就会从黄金里流出，所以它经常和美元呈现反向走势。

而科技成长股依靠远期现金流估值，高度依赖无风险利率。美国AI公司现在9成依赖外部融资，利率一高，融资成本爆炸，还没赚钱呢，就先被利息拖死了，所以加息极易引发估值泡沫收缩，甚至泡沫破裂（参考2000科网泡沫、2008次贷）。

**2、那美联储为什么要加息呢？**

因为通胀反弹+就业过热。

首先，美联储法定目标是2%通胀，现在中东冲突，油价大涨，能源价格传导到全社会物价，5月数据已经到了4.2%，翻倍了，收紧通胀的动机很强。

其次，5月非农就业人口新增17.2万人（预期8.5万，几乎翻倍），失业率4.3%低位。就业过热会带动薪资上涨，形成薪资与通胀互相强化的恶性循环。

**那不管通胀数据行不行呢？不行。**如果通胀彻底失控，就会出现东西越来越贵，但赚钱越来越难的滞胀环境，属于最难调节的经济坏局面。这就是美联储宁可加息杀估值、忍受市场波动，也要摁住通胀的根本原因。

**3、黄金未来怎么看？**

今年金价从高位回落了差不多28%，但我坚持**短期调整不改黄金长期牛市**的判断。

疫情后，全球量化宽松，各大经济体央行货币超发严重，为了确保资产不缩水，必须大量囤积黄金应对货币贬值的风险。而货币贬值的压力短期内没办法解除，叠加应对战争的考量，所以央行囤积黄金的步伐可能有时会放慢，但不会轻易停止，更不会轻易抛售。这是黄金价格上涨的主要驱动力，并对黄金价格形成长期支撑。

还有一个原因，就是由于资源枯竭和开采成本大幅上升，现在的储量只够挖19年的，供给弹性几乎为零，黄金的稀缺性还在提升。

但是我们也不能忽略短期回调的风险，逻辑再硬的资产也没有一步涨到位不带回调的。

参考机构的预测，年内美联储有55%概率加息25pb，金价大概会下探3800–4200区间；45%概率维持利率不变，金价将震荡回升至4200–4800。极端滞胀场景下，金价短期恐慌会砸到3600，但参考70年代行情，滞胀周期中黄金是最优对冲资产，短期恐慌下跌后将开启长期大涨，当时10年翻3-4倍。

大家可以参考这个预测，做好自己的资产配置和投资应对。
