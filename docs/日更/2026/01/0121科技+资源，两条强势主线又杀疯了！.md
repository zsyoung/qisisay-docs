# 科技+资源，两条强势主线又杀疯了！

## 今日复盘

1、今天A股三大指数集体上涨。在科创、贵金属、小盘的带领下，市场又开始了高歌猛进。

昨晚美股大跌，原以为今天大A会跟一下，没想到这么强硬。东升西落，东稳西乱的意味越来越明显了。

成交额2.6万亿，比昨天缩量1805亿，活跃度依然很高，但市场情绪已经开始从“狂热”转向了“精挑细选”。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260121195951915.png)

2、说到资源。黄金今天又杀疯了！谁能想到国际金价能这么快就突破4800美元/盎司。

现在贵金属的逻辑彻底变了：大家不再只盯着美元看，看的是**全球对抗和美债信任危机。**只要地缘摩擦没个消停，黄金就自带一种“确定性溢价”，短期想崩盘都难。

回头看，去年10月以来行情真的撕裂：美股跌了-6.25%， 国内红利低波也回调了-3.51%。 如果你只赌单边资产，这段时间心态肯定崩了。

但这就是“四维轮动”策略（15%黄金+15%红利+15%港美科技+55%中长债）的魅力：靠着黄金近20%的暴力拉升，完美对冲了权益类的调整，**东方不亮西方亮。**组合继续创新高！

目前各资产动态平衡，明天我会按继续计划定投，稳步攒份额。

3、说到科技。最近雪球有篇文章《一个老登的科技转型经历》挺火的，我也认真看了。

作者很坦诚，承认自己曾是“老登股”的坚定拥护者。2014到2021年，老登们靠着传统蓝筹赚得盆满钵满，他甚至在2022年还靠央企地产股成功避险。但2021年后风向确实变了，白酒、银行这些传统板块持续颓势，作为职业投资者，作者感叹不转型就没钱赚、没饭吃。

2023年起他转战科技，但他有个观点我特别认同：**个人投资者很难跟上科技行业的迭代节奏，看好赛道后，布局ETF才是上策。**既能重仓拿住大行情，又能规避掉个股莫名其妙“掉队”的风险。

在文末，他提到了目前重点关注的两个方向：大数据和软件。还专门提到了**创业板软件ETF富国（159107）**，指出其成分股在2025年已经完成了从“讲故事”到“交业绩”的华丽进化。

我也顺着这个思路复盘了一下，逻辑确实挺硬。159107本质上押注的是企业数字化和国产软件替代的长期叙事。不同于纯情绪驱动的炒作，软件公司的变化是看得见摸得着的，往往先体现在订单和收入，再慢慢反映到利润上。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260120180710073.png)

从数据看，它跟踪的指数在2025年三季报中营收同比增长了33.51%，业绩回升的趋势很明显，不再是前几年那种“纯画饼”阶段了。

说到业绩，如果说软件是稳扎稳打，那**创业板人工智能ETF富国（159246）**就更具进攻性。它跟踪的指数2025年三季度净利润同比增长竟然达到了90.55%！

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20260120182930824.png)

相比之下，159246更偏向AI全产业链，既覆盖了CPO光模块这种硬科技，也兼顾了国产应用标的，爆发力更强，也值得加入自选观察。

最近这两只ETF在开年冲高后都调整了不少。从过往走势看，这类高成长赛道在回调10%-15%处往往容易企稳。

既然2026年是公认的AI应用大年，科技主线的位置就还在。希望这波回调能多砸出点坑，砸出更大的机会吧。
