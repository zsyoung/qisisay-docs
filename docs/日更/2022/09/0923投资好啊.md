# 投资好啊

文｜启四

投资好啊，好就好在它好个锤子。

事实证明，上周做出的本周减仓决定是对的，但即使已经很小心地减仓，大大缩小了亏损幅度，还是挡不住一路向下的市场。

## 一、东风转债

东风转债今天盘中短暂冲高，万绿丛中一点红，迅速减仓不格局。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20220923213331731.png)

## 二、正邦转债

正邦转债自上次下修成功之后，开启一路阴跌，毫无下限。

虽然大家都很清楚它的正股正邦科技财务状况和公司运行情况相当糟糕，即使猪周期到来，正邦科技可能也吃不到红利——坊间传闻其在困难时期卖了猪仔，目前已经没有足够的猪可以卖。看年报流动负债远高于流动资产，明显的资不抵债。

但考虑到转债的下跌有底，以及正邦科技闪转腾挪的能力——经过几个月的骚操作，净资产已经不为负了。

目前还是坚定持有正邦转债，慢慢熬，但是不再频繁加仓，如果有跌破100元的机会，再考虑加仓。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20220923213958121.png)

## 三、博下修转债

有几只博下修转债目前价格已经跌的差不多了，看得出来跌不太动了。

- 亚药转债，103.41元，剩余规模9.619亿，剩余年限2.526，正股pb7.09
- 大族转债，109.156元，剩余规模22.993亿，剩余年限1.373，正股pb2.02
- 海澜转债，108.969元，剩余规模29.495亿，剩余年限1.805，正股pb1.3

刚刚查了一下中签记录，博实转债、兴发转债也是颗粒无收，没有一丝丝安慰～你们呢？
