# 券商羊毛

之前在免费星球内测了一个薅零花钱的计划，感兴趣的小伙伴还挺多，再简单讲下吧。

简单来说，就是通过不断开hu+入金去薅券shang的羊毛，一家薅100-300块。

**别看单次好像不多，但每个月都能薅1-2家，一年就可以有2-5000左右的收益。积少成多嘛~**

为什么能薅？因为券shang大多都有一个开hu量的指标需要完成。后面我就不细说了，懂的都懂。

说到这里有的同学可能要说，那不就是不停开hu吗？一人只能开3个啊，我都开满了。要是参与的话那我不是老要去销户，多麻烦啊？

**nonono，如果每次都要去销户这么麻烦，我还会拿出来讲吗？当然不会！**

这个计划的前提就是，绝大部分薅的对象，都是“不限三”的。也就是哪怕你开满了3个也能参与的，而且操作很方便~

首先跟大家科普下限三是什么？简单来说，限三针对的是沪A账号，每个人只能有3个。而深A账号是没有这个限制的，最高能开到20个。

这些不限三的券shang都是只需要开深A即可，我们只要在薅的时候选择“转户”，就可以完成开户操作了，根本不需要去注销之前的账户。（我今天已经亲测过，10分钟不到就搞定了）

而且这种不限三的券shang，大部分可以直接网上销户，不需要去网点现场的。相当于我们薅一年，然后在这一年抽一天出来，可能花一两个小时就能清空重来了。

**别跟我说一两小时都好多、好麻烦。咱们低头算算自己的时薪，一小时能赚500+吗？**（ps：时薪500对应日薪4000，对应月薪9万左右）

讲到这里，应该只剩下感兴趣的同学了，那下面就来说说细节~

----

**Q：门槛是多少？**

A：一般都是要求入金1W左右+放30天左右+交易一次（购买基金/股票/或指定类货基理财产品）。理论上拿1W出来一年可以滚11-12次。

**Q：单次收益多少？**

A：根据不同券shang、不同要求、不同参与金额会不一样，但一般至少也有100，多的400+也有。

**Q：安全吗？**

A：都是正规券商，拿牌照甚至上市的那种。跟你平时开户是一样的。

**Q：什么时候能拿到羊毛？**

A：由于渠道结算需要时间，当月20号前做任务的，次月20-25号左右可以拿到。（所以建议大家都在20号前完成当月任务）

理论上来说只要你每月都参与，那就每月都可以领到money~
