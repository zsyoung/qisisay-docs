# 生猪

文｜启四

猪价企稳回升，22Q3有望维持强势；种企阵型名单发布，头部种企有望获政策倾斜。重点推荐生猪和种植板块。

　　生猪：猪价企稳回升，22Q3或维持强势。本周猪价环比上涨1.4%，南方猪价涨幅明显，北方猪价窄幅调整。2021年8-11月母猪产能加速去化，按照生猪产能传导周期推算，22Q3生猪供给或明显收缩，同时出栏体重仍处合理区间，或难对短期猪价形成冲击；而需求端受季节性因素影响或边际向好。预计22Q3供需结构持续改善有望支撑猪价维持强势，成本改善加持下，猪企22Q3有望实现扭亏。建议继续加配生猪养殖板块。标的选择方面：1）首选已经多轮周期验证过成本优势及扩张能力的[牧原股份](http://sys.hibor.com.cn/dayline_a_new.asp?code=002714&abc=YWCXuNsOpOtRmQrMsNmOqN&def=mOpPpNmOlOrRyQjMnNrO7NmOpOMYnQmRxNpPnQ&xyz=mOrPmNwOnOoRmQ&vidd=24&keyy=DLJOU45LJO&op=1)；2）经营已现拐点的[温氏股份](http://sys.hibor.com.cn/dayline_a_new.asp?code=300498&abc=YWCXuNsOpOtRmQrMsNmOqN&def=mOpPpNmOlOrRyQjMnNrO7NmOpOMYnQmRxNpPnQ&xyz=mOrPmNwOnOoRmQ&vidd=24&keyy=DLJOU45LJO&op=1)等；3）扩张确定性高的性价比小猪[巨星农牧](http://sys.hibor.com.cn/dayline_a_new.asp?code=603477&abc=YWCXuNsOpOtRmQrMsNmOqN&def=mOpPpNmOlOrRyQjMnNrO7NmOpOMYnQmRxNpPnQ&xyz=mOrPmNwOnOoRmQ&vidd=24&keyy=DLJOU45LJO&op=1)等。
