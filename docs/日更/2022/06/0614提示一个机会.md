# 提示一个机会

文｜启四

昨晚的文章中，我分享了一些[配债潜力不错的标的](https://mp.weixin.qq.com/s/NCLhp4q3mRMVKiaf5YVkvA)。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20220614211134657.png)

今天重点说一下**京源环保**。

### 1、基本信息

目前价位百元含权22.28，发行规模3.33亿，沪市科创板股票。总结为：**高含权，小规模，可1手党**（使用精确算法）。

### 2、预估收益

使用曲线拟合插值预测价格142.272元，但是考虑到京源环保规模如此之小，首日开盘90%以上的概率可以达到150元（沪市首日开盘最高价），尾盘有冲击200+元的潜力（不要低估妖债的爆发力）。

那么**配一手的预估收益在50%-100+%。**

### 3、计算安全垫

沪市安全垫 = (预测价格-100)*10/一手党配售10张所需股数/正股价

- 预测价格：150-200

- 一手党配售10张所需股数为200股
  - 集思录查到配售10张需要323股，因为沪市一手党配售10张所需股数 = 配售10张所需股数*0.5，向上百位取整
  - 所以323\*0.5=161.5，向上取整为200股

- 正股价13.91，配一手需要200*13.91=2782元

代入公式：

- 首日价格为150元时，安全垫=(150-100)*10/200/13.91=17.97%

- 首日价格为200元时，安全垫=(200-100)*10/200/13.91=35.95%

### 4、股票跌幅判断

提前埋伏配债的聪明人很多，**申购日当天抛压很大自不必说**，那么具体跌幅有多少呢？

科创板涨跌幅最大为20%，那么悲观估计，当天最大跌幅为20%，

以近期最低价11.10元估计，更悲观地估计，最大跌幅为：(13.91-11.1)/-1=25.32%

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20220614205804025.png)

股票亏25.32%，则2782*25.32%=704.4024，对应转债要涨到170.44才行。

股票亏20%，则2782*20%=556.4，对应转债要涨到155.64才行。

### 5、总结

1）悲观估计，如果转债上市涨超170.44元，稳赚，否则可能亏损。

2）上面预测股票下跌的幅度是超级悲观的看法，大概率跌不到这么多，总体安全垫应该在15%以上，**值得配债**。

3）持有200股即可配债。
