## 三、哪些值得配债？

原本在经历过几次配债亏损之后，我已经放弃了配债的念想，但是考虑到

1. 近期即将发行转债的上市公司越来越多
2. 目前配债安全垫高的转债很多
3. 大盘相比年初也有不少起色

我又起了歹念～

**那么哪些值得配债呢？**

### 1）安全垫15%以上的：

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20220613213804122.png)

发行流程：董事会预案 → 股东大会批准 → 证监会受理 → 发审委通过 → 证监会核准/同意注册 → 发行公告

发行流程就差临门一脚的是：**新疆天业、丰山集团、京源环保**。

安全垫最高的是：**长沙银行、瑞丰银行、齐鲁银行、厦门银行**。

*注意：安全垫数据为粗略计算，使用曲线拟合插值预测的转债价值，有一定误差。*
