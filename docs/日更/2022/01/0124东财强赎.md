# 东财强赎

大家好，我是启四，长期分享套利机会。

我在2022-01-13的文章：[新的潜伏机会来了！](https://mp.weixin.qq.com/s/tFGBLK-Vr8A-SABBrgGdhg)提到了绿色动力的潜伏逻辑。

不出意外，明天应该会涨一波。

## 二、赎回潮来袭

今天有大量转债满足赎回条件。

1、其中**宣布强赎**的有：东财转3、星帅转债、比音转债、银河转债、中鼎转2。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20220124214446047.png)

其中东财转3我在之前的文章：[东财转3的博弈机会](https://mp.weixin.qq.com/s/idrXRqK6Yryq1x6qcCR5kQ)中提示过博弈机会，目前价格140.01，溢价率0.31%，很安全，大家不要恐慌。

卖出或者继续持有到强赎前都可以，溢价这么低很难暴跌的。当然也可以转股，每年的一季度也是券商行情好的阶段。

当初选择博弈它的原因，就是因为赌输了小亏，赌赢了大赚，明显是风险收益不对称的机会。

持有银河转债的同学比较惨，目前溢价13.51%，明天大跌杀溢价不可避免，建议尽早卖出吧。

2、**宣布不强赎**的更多：恩捷转债、齐翔转2、金禾转债、万顺转债、万顺转2、联创转债、高澜转债。

这些转债又可以愉快玩耍了。

我看了下，基本上大家都预期这些转债不会强赎，杀溢价的幅度都很小，没有什么套利机会。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20220124214654907.png)

3、明天**即将满足强赎**的转债：湖广转债、伯特转债。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20220124220136855.png)

湖广转债溢价5.86%，伯特转债溢价8.86%，如果宣布强赎将会杀溢价，建议明天卖出防止强赎杀溢价。

白电转债的正股白云电器今天拉升了一波，但是截止收盘还是未满足强赎触发价11.78，继续持有，博弈其在2022-01-28收盘之前，会将股价拉升到11.78以上2次；

胜达转债的正股大胜达今天突然跌停，也有了类似于白云电器强赎博弈机会，明天我会关注。
