# 洪涛转债有戏？妖债穷途末路

文｜启四

## 一、洪涛转债大涨

洪涛今日正股涨停，转债随之大涨，最高价115.888，上周在108以下买入了洪涛转债，还真的博弈成功了。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20220512121433014.png)

众所周知，**可转债=债券+转股权利=债券+看涨期权**，如今洪涛转债马上就要到期了，时间期权已经接近于0，所以只要有溢价就可以先出掉。

后面如果还有机会回到108以下，可以再考虑薅羊毛。

我是不会再寄希望于洪涛股份还能完成强赎了。

## 二、小规模转债继续疯狂

今天涨幅5%以上的转债，绝大多数又是这些熟悉的次新妖债。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20220512123553579.png)

另外今天的三低转债又是全部上涨：

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20220512123737905.png)

它们都有一个共同特征——**可交易规模小**。

三低转债是已经进入转股期的，剩余规模小。

次新妖债是还未进入转股期的，可交易规模小。

但是归根结底，都是因为市场上流通的份额小了，所以更容易炒作，活性更强。

今年普跌的环境下，这个特征愈发明显，因为游资和大多数散户的**刚需**是高频交易，如果买一些大规模转债，可能长期处于不温不火的境地，钱无处可去，就只好炒作活性强的。

什么转债活性强呢？就是上述两种转债。

**我的刚需是赚钱**，所以我不会冒险选择次新转债，顶多在其上市的时候抢一些，做确定性套利，后续就会把资金挪到低风险转债上面去。

**你的刚需是什么呢？**是寻求高频交易的快感还是赚钱？反思过吗？
