# 永吉巨星转债上市

文｜启四

## 一、转债上市

2022-05-17周二，永吉转债、巨星转债上市。

### 1）永吉转债 113646 沪市

**上市价预测**：发行规模1.46亿，行业：轻工制造-包装印刷-纸包装，转股价值82.76，拟合曲线插值计算预测价格127.644元。

中签率0.0004%，网上规模0.50亿，上市可流通规模0.85亿。

这三个数据和胜蓝转债、聚合转债、泰林转债极其相似。

现在这3只转债的价格是这样的：

- 胜蓝转债，价格：283.60元，溢价率：230.70%
- 聚合转债，价格：234.01元，溢价率：194.37%
- 泰林转债，价格：383.00元，溢价率：359.71%

前三个都发达了，老四没理由穷着吧？

由于沪市转债首日开盘价最高150元，所以永吉转债明天早上开盘价铁定是150元，至于后市价格能到多少，**我觉得至少200元**。（非投资建议，据此买卖盈亏自负）

说一下我的理由：

- 为什么大家都叫它们妖债？——因为高价且高溢价，脱离了其应有的价值。

- 那为什么会脱离其应有价值呢？——剩余规模小，转债流动性强。
- 未来这种情况是否会持续？——很有可能，转债被越来越多的人熟知，参与的人越多，资金涌入的越多。以前只有游资才能炒妖债，现在资金只要超过一千万，买入一只妖债，就能让其价格剧烈波动。
- 当参与的人越来越多，大家公认妖债就应该超过200元的时候，它就拥有了200元的价值。
- 这种现象也可以类比到比特币、NFT等，引用大佬的一句名言：**傻x的共识也是共识。**

**对标转债**：胜蓝转债、聚合转债、泰林转债

**操作建议**：

- 如果中签，首日千万别傻傻卖出；

- 如果未中签，今晚参与夜市委托抢筹，或者挂涨停买入条件单，150元抢筹永吉转债。

- 大概率是抢不到的，尾盘集合竞价还可以继续挂更高的价格，理论上可以挂无限高，但是超过200元就不推荐再去抢了，短期风险太高。

### 2）巨星转债 113648 沪市

**上市价预测**：发行规模10亿，行业：农林牧渔-养殖业-生猪养殖，转股价值71.91，拟合曲线插值计算预测价格120.454元。

给予53%-56%的溢价率，预估上市价110-112元。

**对标转债**：希望转债
