# 大涨，加仓了！

## 今日复盘

A股三大指数今日集体反弹，截至收盘，沪指涨0.71%，深成指涨1.14%，创业板指涨0.70%，北证50指数涨2.07%。全市场成交额10891亿元，较上日放量179亿元。全市场超4000只个股上涨。

![](https://files.mdnice.com/user/7588/88bec6d1-eb87-43ab-a67a-343feaff9700.png)

昨晚刚提到“[止跌信号](https://mp.weixin.qq.com/s/JANRx93lxbbfhZH923ROxA)”可能来了，今天市场就给了回应——直接反弹了。除了国债和红利指数，其他全线飘红，几乎都收出了光头阳线。更妙的是，除了创业板，其他主要指数都走出了底分型。

先说说**底分型**是啥。简单点理解，就是三根K线的组合，中间那根的低点比两边更低，高点也最低。这种形态通常被视为底部信号，意味着市场可能准备反弹或者反转。

其实，这种反弹没啥意外。股市总是涨涨跌跌，没有只跌不涨的道理。这几天我耐心等着，今天下午趁机加了一些仓。这里提个**思路**：追涨补仓比追跌更有效。因为追涨意味着趋势在变好，而追跌容易陷入“越补越跌”的坑，这符合有效市场假说，也更适合普通投资者的操作逻辑。

短期来看，技术面支撑市场可能还有反弹的空间。尽管今天成交量只有1万亿，但形态上依然看涨。不过，想要彻底扭转局面，还需要更大的力量。现在市场连60日均线都跌破了，要修复恐怕需要时间和新变量。以当前的市场情绪和资金推动力，这事儿不容易。

从更大的时间维度看，影响市场的还有政策和外围环境。1月20日，川普就职，这是一个重要时间节点。在此之前，我们大概率会留一些“牌”来应对新的国际局势，比如潜在的关税政策压力。这段时间，市场或许会以震荡为主。

回到A股的历史节奏，大多数牛市并不是一蹴而就的。常见的是“**快涨—震荡—主升浪**”三段式，比如1999-2001年、2013-2015年、2019-2021年。每次快速上涨后，都会有一段调整期，市场用来消化过高的预期，同时等待盈利改善的验证。

休整期并不可怕。事实上，它往往是未来主线行情酝酿的阶段。比如2013年的TMT、2019年的半导体和新能源，都是震荡阶段崭露头角的。当前的调整也类似，市场在等经济复苏和企业业绩修复。只不过这次的恢复速度不够快，可能要多等一阵子。

我的**建议**是：策略上保持均衡，减少操作频率，尤其不要随意空仓满仓，避免来回打脸。这个阶段，心态比操作更重要。牛市主升浪需要时间酝酿，耐心点，等到二级火箭起飞，真正的行情自然会来。

再聊聊转债方面。

1、**迪贝转债**，公告强赎了。现价150.777元，溢价率-2.45%，剩余规模1.656亿，虽然规模不大，溢价率很低，但是正股已经三天三板，转债今天也大涨9.4%，接下来还是有回调的风险。

2、跌幅榜第一的**普利转债**，因为触及重大违法强制退市条款，今天正股转债双双20cm跌停，且封单很大。如果要博转债刚兑，我觉得跌到30元附近才有博弈价值（非投资建议）。

3、跌幅榜第二的**伟隆转债**，今天下午闪崩，据说有人在284元的高点买入了50万，按收盘价算亏损接近26%，不到一个小时就跌没了13万。

它闪崩是必然的，现在距离转股期只有1个月多点点，溢价率还高达102.41%，等进入转股期后，大股东的份额解禁，流通规模会从现在的1.192亿变成2.697亿，炒作优势大大降低，也就维持不了现在的高溢价了。即使今天跌停，也依然是非常高估，未来还会继续大跌。

通常距离转股期仅剩1-2月时，游资就会撤离。最近的案例可以参考赛龙转债、泰瑞转债，两只转债从最高点到现在都跌了45%+。

现在比较**危险**的还有松原转债（187元+82%溢价率）、远信转债（252元+210%溢价率），随时断头杀。
