# 成交额跌破2万亿了

## 今日复盘

今天市场又开始反弹上涨了，但是沪深两市成交额已经不足2万亿，今日之前曾连续20个交易日突破2万亿。这就说明热度有，但资金并没有真心实意地“all in”。

从技术面看，中证全指在短期均线一带还算有支撑，所以指数暂时不会轻易掉头，可能会通过震荡来慢慢消耗情绪，等待长期均线逐渐追上来。

债市这边，今天跌得很猛。十年期国债利率直接冲到1.8%以上。连续大跌的原因，一方面是股市吸走了资金，另一方面是宏观流动性并不宽裕，央行也暂时没有加大投放的打算。说白了，要等经济数据真正掉下来，央行转向宽松，债市才可能迎来拐点。在此之前，股债依然是冰火两重天，任何模棱两可的数据，市场都会解读成利好股市、不利债市。

至于国债，虽然看上去跌出了机会，但30年期国债波动太大，并不适合大多数人去博弈。国债的本意是用来平衡股市风险，长期持有不至于绝对亏损，但如果你带着“买了就要立马赚钱”的心态，不断加仓，结果扛不住2%-4%的波动半途割肉，那就是实打实的亏损了。所以，已经拿的就别动，没买的也别轻易上。

我自己还有20%的资金被套利的ST天茂锁着，等资金来回，看情况计划布局一些国债。
