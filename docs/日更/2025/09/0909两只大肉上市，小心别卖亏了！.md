# 两只大肉上市，小心别卖亏了！

## 今日复盘

1、今天市场又缩量下跌，4000多只票绿了，只剩1300多只飘红。之前冲在最前面的电子、计算机、通信、医药、军工、电力设备全线趴下，反倒是银行、地产、有色这些“稳健型选手”顶上来。风格切得很明显，资金在往保守防守的方向挪。

不过虽然成交量缩到2.1万亿，但昨天两融资金还在创新高，这说明投资者**只是开始犹豫，还没彻底转空。**

2、盘面走势上看，最近这波节奏确实折磨人：跌三天、涨两天、再跌，一直在撩拨你的情绪。看多有理，看空也有理，让人摇摆不定。其实这就是震荡整固的本质——就像行军打仗，打下一片新阵地，一定要安营扎寨、休整补充一下，才能开始进攻下一片阵地。一味地猛冲猛打，孤军深入，很容易被反打崩盘。

今天除了红利和黄金，其他指数基本全灭。之前就说过，这俩就是震荡期的压舱石，能稳住心态，现在看确实挺对。

3、那短期为啥会调整？核心原因还是9月3日后资金逻辑变了：之前大家都在追动量、抱团冲高，现在更看重性价比和基本面。抱团一旦散了，就容易急跌，毕竟很多人是追高进的，一旦跌破成本，割肉砸盘很正常。所以短期大概率还要在震荡里耗一段时间，等新催化、新共识。

但中长期逻辑没变。两个点：

**第一，基本面。**制造业已经有企稳迹象：二季报数据在改善，很多企业缩减投入，供给在加速出清。简单说，竞争越来越少，存活下来的公司未来会更有话语权。按照一些机构的预测，等到2026年前后，制造业的供需关系可能彻底好转，企业利润有望迎来一波像样的反弹。

**第二，资金。**居民存款搬家才刚开始。未来2-3年正好是定期存款到期高峰，但利率这么低，钱继续放银行划不来。股市自然会成为承接地。现在居民多余存款有30万亿的规模，潜力巨大。而且不少20-21年发行的公募基金产品快接近回本，如果市场维持赚钱效应，资金回流和市场上行就能形成正反馈。

所以总结一句：短期震荡别慌，当成休整；中长期，我还是看好A股的机会。
