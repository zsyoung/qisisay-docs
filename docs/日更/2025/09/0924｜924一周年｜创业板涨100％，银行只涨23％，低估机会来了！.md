# 924一周年｜创业板涨100％，银行只涨23％，低估机会来了！

今天各大指数集体低开高走，盘面上，科创、微盘股、创业板领涨，长期国债因为跷跷板效应大跌。成交额方面，沪深两市全天成交额2.33万亿，较上一个交易日缩量1676亿。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250924194148293.png)

挺有意思的，今天正好是 924 行情一周年。

回头看这一年，创业板、科创板这些成长板块涨得飞快，创业板涨了100%，科创50涨了 120% 左右，真的是主线中的主线。

反观银行板块，这一年先涨后跌，只拿到23%的涨幅，基本没什么存在感，很多人都说“银行没戏了”。但我觉得，恰恰在这种时候，**银行可能会有一些被低估的配置机会。**

为什么这么说呢？

先说大环境。今年的行情走得很典型：成长在前，分红在后。小盘股、科技股冲在前面，赚足了眼球，但凡是防御属性强的，比如银行/红利/长债，就显得“不性感”。

其实历史上也经常这样。2014 年大牛市加速的时候，电子板块一度被抛弃，结果半年后又卷土重来。银行现在的处境有点像那时候。

再看逻辑。过去两年，银行是稳稳的“宠儿”，红利风格火的时候，它是资金最爱去的地方。但进入牛市加速期，资金自然会往成长板块扎堆，银行被边缘化是正常的。问题在于：这是不是意味着银行彻底没机会了？我觉得未必。

### 几个理由：

1. **估值和股息率**：银行现在整体估值还是在底部区间，PE分位40%，PB分位24%，股息率4.21%对比国债收益率依然有明显优势。对长线资金来说，这是躲不开的选择。![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250924211248774.png)
2. **资金风格切换**：四季度开始，三季报会逐步披露，叠加政策预期升温，市场可能从“疯狂追小票”转向“大盘价值”。这种时候，银行就容易被资金重新盯上。
3. **政策环境**：今年宏观环境比较微妙，经济恢复还需要稳预期。银行作为“压舱石”，天生带有一点政策底色。
4. **技术面：**银行ETF已经跌到了年线附近的强支撑位，而且人气还没散。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250924210713031.png)

市场总是这样：当大家都觉得银行没意思的时候，它往往**悄悄蓄力**。等到资金风格一切换，银行的补涨往往来得又快又猛。

当然了，银行也不是过去那种“稳稳的幸福”了。牛市后期波动会更大，可能上涨很快，但回调也狠。换句话说，银行更像是波段性机会，而不是长期抱住不放的板块。

总结一句：关注银行，不是说它能替代科技去当主线，而是说在当前这个阶段，它的配置价值正在被低估。对稳健型投资者来说，银行可能不是最性感的选择，但一定是最不能忽视的一个。

最后，关于**红利低波**，我的看法和银行一样，也到了配置价值突出的位置。
