# 又新高了，躺着就好了

## 今日复盘

今天市场整体还行，除了创业板和国债，其他指数都涨了。小盘股和北证50走在前头，一方面说明风险偏好又回来了，另一方面前期大涨的通信板块今天领跌，也能看出资金在做轮动切换。

成交额2.4万亿，比不上前几天的3万亿，但也比8月中旬之前热闹多了。人气被上周五的反包点燃后延续下来，这就是个好现象。

说回到我自己的操作。从清掉转债以后，现在就主做两个策略：**ETF轮动和小盘股**。转债策略切换到ETF轮动策略后微亏0.36%，小盘股策略盈利2%。这类策略本来就是低频的，所以绝大多数时候，我都是呆坐不动，不需要天天盯着市场。

这也带来了一个挺微妙的状态：投资上很舒服，生活上也轻松了——我有大把时间可以看书、陪家人、研究AI、折腾社群网站。但对公众号日更来说，这就是麻烦了：市场没什么特别的事，我也就没什么特别的可写。

不过转念一想，这种“无事可做”的感觉，不就是投资里最理想的状态吗？该上就上，该空就空，不必时时刻刻和市场死磕。长期投资的意义，也许就在于它能让人把精力和自由还给自己。

和两年前那个常常焦虑的自己比，现在的我明显感觉进步了。市场的波动几乎撼动不了我执行策略的决心。从一开始我就知道，只要走在对的方向上，盈利迟早会属于我——因为时间站在我这边。
