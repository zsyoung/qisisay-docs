# 今晚吃面笑着吃

## 今日复盘

1、A股三大指数今日集体下跌，截至收盘，沪指跌1.98%，深成指跌2.89%，创业板指跌3.82%，北证50指数跌4.32%。全市场成交额19055亿元，较上日缩量1364亿元。全市场超4700只个股飘绿。

![](https://files.mdnice.com/user/7588/aaade69f-ae40-4404-b614-ebe422d8956b.png)

2、除了30年国债ETF，今天所有指数全军覆没，包括红利。昨天我还在纠结将减仓腾出的资金挪到红利还是长债上，长债有顾虑是因为机构普遍预期降准降息的窗口可能推迟到一季度末至二季度，最近可能还会继续调整；红利有顾虑是因为回溯了一下历史数据，发现2019年的科技牛市，3月份回调的时候，红利也没大涨，而是跟着波动，索性就继续持有现金吧。

昨天刚说完新手总想抓最后一只螃蟹，老手则要多关注逃生通道畅通性。今天就应声大跌了，且指数是低开低走，畅通无阻，一路绿灯，没有任何像样的反弹。

其实这就是牛市的常态，缓涨急跌，用一次又一次的“反包行情”让越来越多的散户自我催眠“这次不一样”，最终在某一天被顶部套牢。

你以为的牛市 vs 现实中的牛市，这种图的含金量还在上升。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250228190855476.png)

3、今天这一跌，基本上将指数打回了2月7号的位置，估计很多人都是一个月白干。下跌的原因其实没必要纠结，最主要还是近期涨得太多太快了，已经涨得双脚离地了，这时候来一波急跌，反而能释放风险，为未来的上涨打下基础。

春节后宽松政策一直没有出台，其实也是股市上涨带来了底气。不过光靠AI炒作不足以解决所有经济问题，要真正解决通缩，还需要更多的财政刺激，两会前跌一跌也有利于推动政策。

4、当前大盘的长期均线并没有破，整体的上涨趋势还在，不用过于担心。这次的回调，或许会让短期的炒作降温，接下来可能会有一段风格均衡调整期。比如涨红利、长债、消费、医药，跌科技、小票。所以手上有追高的标的，要准备做好风控止损。

启四今天暂时还没有抄底，继续耐心等待企稳信号和抄底机会。

### 转债方面

1、即使减了半仓，今天也一样被锤的很痛。不过**幸福感来自于比较**，大家都跌不算跌，跌得少了心里还会暗暗窃喜，晚上吃面都是笑着吃的。

2月份总收益7.7%，今年累计收益7.91%。暂时计划等中位数跌到120以下再恢复满仓状态。

2、神码转债，公告强赎。现价150.956元，剩余规模8.784亿还是挺大的，有股债双杀的风险。

3、涨都是风险，跌才有机会。大跌的时候，往往配债股和提议下修的转债才会出现配置的性价比和机会，可以多向这两个方向挖掘。
