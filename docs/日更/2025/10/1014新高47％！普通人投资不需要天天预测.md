这就是问题的根。**人们亏钱，不是因为市场太难，而是因为太想预测。**
预测能给人安全感，给人一种自以为掌控一切的幻觉，但市场最擅长的事，就是让“看懂的人”最先交学费。
想想自己，从国庆节后到今天，能躲过科创的大跌，又吃到黄金的大涨，里外里多赚了15%，正是因为紧紧跟随策略信号，强迫自己不预测市场。
### 真正的高手，从不预测
传奇的桥水基金创始人达利欧有句名言：
> “你不需要预测天气，只需要穿对衣服。”
这是全世界最聪明的投资逻辑。
![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/Ray_Dalio_Sept_23_2017_NYC.jpg)
他在几十年的交易中发现，市场的涨跌，其实只由两个变量决定：经济增长和通胀水平。
于是他画出了市场的“天气图”：
1. **繁荣 + 通胀上升** → 大宗、黄金涨
2. **繁荣 + 通胀下降** → 股票、债券涨
3. **衰退 + 通胀上升** → 黄金、通胀债避险
4. **衰退 + 通胀下降** → 国债、现金护底
这就是后来家喻户晓的——全天候策略（All Weather Portfolio）。
你不需要预测哪一天风向变，只要在所有天气里都能活下去。
几十年来，这套策略在美股市场经历了无数危机，依然屹立不倒：
- **2008年金融危机**：全球股市暴跌40%，全天候组合仍实现正回报。
- **2020年新冠疫情**：市场大跌，全天候策略迅速回稳，波动远低于普通组合。
- **40年回测数据**：年均回报9.5%，最大回撤仅14.59%，远低于股票市场的50%以上！
![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250121181437742.png)
从石油危机、次贷危机，到互联网泡沫，全天候策略都能穿越周期，稳健增长。
