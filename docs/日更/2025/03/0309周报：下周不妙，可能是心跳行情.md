# 周报：下周不妙，可能是心跳行情

## 下周观察

梳理一下周末利好利空消息：

**利好消息**

1. **央行狂买黄金不手软**
   中国央行连续第四个月增持黄金，2月末黄金储备冲到7361万盎司（约2289.53吨），单月又加了16万盎司。这波操作明摆着在给避险资产站台，全球动荡下咱家“屯粮”动作很实在。
2. **地方政策挺给力**
   - **上海国企动真格**：国资委发话要盘活老资产、推进改革，上海本地国资股（比如老牌商贸、基建企业）直接受益。
   - **北京学生全员学AI**：2025年秋季起，北京中小学每学年至少上8节AI课，教育类AI公司（比如智能教具、课程平台）要乐开花了。
3. **大资金悄悄抄底**
   上交所数据爆猛料——沪市ETF规模飙到2.7万亿，同比暴增77%。说明机构和大户正闷头买买买，给市场吃颗定心丸。
4. **政策给新赛道撑腰**
   政府工作报告点名要搞人工智能、数字经济立法，相当于给这些行业发“长期饭票”，合规发展的公司躺赢。

**利空消息**

1. **中加贸易要干架**
   2025年3月起，加拿大菜籽油、猪肉等商品关税最高加到100%，贸易摩擦火药味浓了，外资可能瑟瑟发抖。
2. **经济数据有点垮**
   - **CPI突然转负**：2月CPI同比跌0.7%（上月还涨0.5%），PPI也跌2.2%。主要怪春节错位、菜价便宜、车企疯狂降价，消费复苏还得再等等。
3. **短线小心黑天鹅**
   重要会议3月11日闭幕，叠加关税、数据利空连环锤，接下来两周大盘可能上蹿下跳，玩心跳行情。

**综合影响**

- **利好撑场子**：上海国企改革、AI教育普及这些政策红包，加上真金白银的ETF资金流入，能托住市场情绪。
- **利空砸场子**：贸易摩擦+经济数据拉胯，可能引发短线恐慌砸盘，尤其得防着游资借着利好出货。

**一句话总结**：政策暖风对抗外部寒流，短期震荡难免，但AI、国企改革这些主线跌多了反而是机会。

目前仓位依然是5成多。

下周如果大涨，不打算追涨，保持谨慎；如果大跌，耐心等待抄底机会，逐步收集筹码。

上周已经缩减了持仓宽基数量，国债耐心等待右侧机会，关注30分钟/60分钟线。

若转债中位数来到120元区间，恢复转债满仓运行。

#### 已经公告强赎/临近到期

现在有16只转债已经公告强赎或者临近到期了，如果持有，记得关注最后交易日。

#### 即将满足强赎

未来5个交易日内，将有13只转债最快可以满足强赎条件。

其中转股价值接近130元，且规模大于3亿或溢价率大于3%的转债，需要注意强赎后杀溢价的风险。

比如**振华转债、春23转债、濮耐转债、会通转债、精锻转债**就比较危险，尤其是振华转债，非常危险。

利扬转债，华锋转债，亿田转债，已经公告未来3个月不强赎。

对后两者来说算利好，华锋的溢价率预估可以恢复到5%-10%，亿田的溢价率可以恢复到1%-3%。

对利扬的持有人来说，只能算惊险躲过一劫，溢价率已经体现了不强赎预期。
