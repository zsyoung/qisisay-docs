# 好机会，越跌越加

## 今日复盘

A股三大指数今日窄幅震荡，截至收盘，沪指涨0.19%，深成指跌0.19%，创业板指跌0.52%，北证50指数涨1.85%。全市场成交额16209亿元，较上日缩量2200亿元。全市场超3100只个股上涨。

![](https://files.mdnice.com/user/7588/5d6a75ae-1308-4268-a0d9-2da2308411a8.png)

今天市场还是顶住了压力，在促消费政策的带动下继续反弹，算是把信贷数据不及预期的影响压了下去。领涨的板块主要集中在房地产、家电、汽车这些和消费相关的方向，但这波行情的持续性存疑，毕竟下午的发布会并没有太多新内容，市场的热情很可能很快消退。说到底，想要真正拉动这些强周期行业，在当前的宏观环境下还是挺难的。

债市就没这么幸运了，降准降息的预期落空，市场直接用脚投票，继续下跌。这种情绪和去年9月24日之前的股市很像，现在换成了债市情绪高度敏感，风险偏好低；稍有风吹草动就容易形成负反馈，导致理财资金进一步赎回，短期内压力不小。而股市这边，情绪相对较好，资金也在积极运作，对政·策的敏感度反而没那么高。

现在市场有点进入“左右为难”的阶段。前期大涨的计算机、机器人、TMT等高景气板块开始犹豫，4月就要迎来业绩考验，资金不敢贸然冲高。而资源、地产这些方向，依然受经济通缩拖累，复苏之路还远着。再往后看，制造业可能因为出口走弱，拖累PMI数据，市场风格也会随之调整，极端的小票和成长股行情可能难以为继。

整体来看，现在市场没有明确的主线，外部环境魔幻，内部政·策则是见招拆招，确定性都不强。在这样的行情里，盲目激进不如攻守兼备，保持灵活性才是更稳妥的选择。

### 30年国债

说一下前段时间布局的30年国债ETF，这两天已经从入场位置回撤了4.6%了，对纯债来说，确实挺肉痛。

目前30年国债收益率已经回到去年12月9日左右的水平了，基本上把去年抢跑降息的预期修正过来了。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250317202011397.png)

现在整个债市出现了绝对机会，从资金净流入方面来看，30年国债ETF近6天合计“吸金”27.31亿元，日均净流入达4.55亿元，机构资金一直在抄底；而且现在经济数据也不好（2月CPI-0.7%），想破通缩，不降息降准不行，未来大概率会降息，那时就是上涨时刻了。

即便央行今年不降息，他每年也有约2.1%的利息收益通过分红的方式分配。大不了熬着，反正长期持有，最终都能回本并盈利，无非时间早晚而已。

结论就是：好机会，越跌越加。我已经加到了8%，最大目标仓位10%。

不过30年国债ETF有两个缺点：一个是贵，所以加仓不能太频繁，不适合小资金；另一个是久期长，波动大，基本上是普通债券波动率的2-3倍。所以，心态不好也容易拿不住。
