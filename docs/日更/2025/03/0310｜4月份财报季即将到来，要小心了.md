# 4月份财报季即将到来，要小心了

## 今日复盘

A股三大指数今日震荡调整，截至收盘，沪指跌0.19%，深成指跌0.17%，创业板指跌0.25%，北证50指数涨1.89%。全市场成交额15440亿元，较上日缩量3176亿元。全市场超3200只个股上涨。

![](https://files.mdnice.com/user/7588/f78ccdc7-2f88-489b-8f09-ecf406b27fe3.png)

今天市场进一步缩量到1.5亿，虽然尾盘市场还能拉一把，不过从指数涨幅来看，只是小票的炒作情绪在支撑，并且热度明显下降了。如果没有新的热点接力，资金很难撑住这个局面。

4月份是财报季，对小盘股不利，所以往往市场从3月中下旬就开始交易预期。游资慢慢撤退，小盘股一直跌到4月中旬，等待利空消化完，才开始转涨。如果碰上熊市，干脆会直接跌到4月末，大家可以去翻一下中证2000的K线，这个历史规律大部分年份都会得到验证。哪怕是2019年的科技牛，市场从2月初一直涨到4月中旬，但随后也连续下跌了5周，跌幅达15%。

所以做小盘股策略最好避开1月和4月，往往能大幅降低回撤，提升收益。

整体来看，现在市场处在**“预期很高、现实一般”**的状态，政策端推动力度有限。短期内，科技股的热度让大家忽略了经济数据的低迷，但如果没有新的故事出现，市场情绪迟早会回归现实，风格也可能会重新切换，所以谨慎为上。
