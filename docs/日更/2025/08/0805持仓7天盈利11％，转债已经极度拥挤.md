## 今日复盘
今天市场这走势，说实话有点超预期。完全没给空头机会，全天平稳上行，红利低波领涨，上证指数比中证2000涨得还多，两市超3900股上涨，成交量也明显放出来了，回到1.6万亿，确实像点样子了。
昨天说：“如果接下来三天市场能放量到1.8万亿以上，尤其是中小票能放量活跃，那说明资金真回来了，风格切换也有实质进展，到时候我会考虑调仓往成长那边走一走。”我们静观其变。
现在市场的表现看起来情绪已经拐到“牛市思维”了。前脚刚调整，后脚又被拉起来，基本没给人缓冲时间，只要一跌，资金立马扑上去抢，生怕错过车。这种“越涨越怕踏空”的情绪，是最容易让人盲目乐观的时候。
但说到底，不管是不是牛市，涨多了就要喘口气，中证全指现在都已经离60日均线拉开了挺大一段。6月20到现在已经涨了10%+，这意味着大多数散户都已经赚钱了，都有落袋为安的需求。所以后面哪怕真是牛市，也不可能一路直线冲。
时刻记住：**牛市多暴跌，越是牛市越是散户亏钱的开始。**

简单说就是：情绪转牛但别头铁，涨太快反而要冷静点。
### 转债方面
1、转债最近也涨得非常好，今天盘中中位数一度突破了130元，今天收盘后中位数129.9，这是什么概念呢？
这个数值指的是**当前全市场可转债价格的中位值为129.9元**，也就是说，一半的转债价格高于129.9元，一半低于它。
要知道，可转债的面值是100块，现在中位数快130了，说明大部分转债都涨了30%。而且这不是个别涨，是“普涨”。在小市值股票一路高歌猛进的带动下，转债市场情绪也热得发烫，大量资金涌进来博弹性，想搏一波牛市红利。
再看看这个中位数处在什么位置？
| 时间范围 | 所处百分位 |
| -------- | ---------- |
| 所有历史 | 99.08%     |
| 近1年    | 99.18%     |
| 近2年    | 99.59%     |
| 近3年    | 99.17%     |
这意味着：当前可转债中位数价格**已经比过去99%以上的时间都高**，已经是一种“极度拥挤”的状态。
有同学说，如果牛市来了，中位数能一直涨到140。
我的判断是：就算牛市真的来了，可转债中位数也**不太可能一直涨**，尤其是在现在这种接近历史极致高位的背景下，**风险已经非常高了**。不是说完全不能涨，而是再往上，性价比会迅速变差，**波动大、回撤重、吃肉难、挨打快**。再加上一旦市场关注点从“博弹性”切回“看基本面”，小盘切大盘，转债就会成为“先涨透、后被抛”的品种。
所以现在不是闭眼买转债的阶段，如果要买，就要非常挑，控制仓位，只参与少数正股趋势向好、溢价合理的个债。
明天我会继续监控中位数情况，如果轮动的时候，中位数突破130元，我会清仓。
