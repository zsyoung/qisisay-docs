## 今日复盘

今天持仓的科创50大涨，创业板50走了一个V型反转，红利低波也好起来了，从指数表现上看，大小盘和成长价值都在涨，风格分化不大，节奏很乱，多头依然强势，国债本来已经反弹了，尾盘又被压了下去，股债跷跷板效应依然明显。

成交额2.4万亿，看着还是吓人，但其实已经连续缩量了，量价也开始背离。不过量少≠马上跌，关键还是看存量资金什么时候愿意卖。如果没人抛，少量资金也能推着指数往上走。

这波上涨，坦白说超出了我的预期，典型的资金推动型行情：流动性宽松、风险偏好上来了，钱愿意往股市里跑。

经济数据还那么弱，为什么股市却一直涨？搜了一些资料，发现其实这背后有点“底部逻辑”：经济下行到尾声的时候，往下的空间有限，但往上的弹性反而大；再加上利率低，债券没啥吸引力，资金自然会逐渐转向股市。简单来说，就是即使基本面没改善，股市也可能先走一步。

那错了怎么办呢？及时修正观点，做策略应对呗。

策略上，我没去反向操作，而是用更适合牛市的ETF轮动策略，买了创业板50。防守端我配了红利低波和黄金，所以国债还没入场。国债我还是打算右侧交易，等市场情绪释放完、趋势站稳再考虑进。

债市最近调整明显，主要是股市火热、通胀预期上升压制了债市情绪。但大部分机构认为基本面偏弱格局没变，所以跌幅有限，配置价值在慢慢积累。久期越长的债波动肯定越大，你要是实在受不了3%-4%的波动，要么买短久期的债（比如10年期国债），要么干脆别折腾，钱存银行最保险。

### 转债方面

1、**中装转2**，今天停牌了，很多人很慌，不知道该怎么办。

先梳理一下事件：2025年8月19日，ST中装被法院裁定要走重整，股票要被戴上“*ST”的帽子。如果重整失败而被法院宣告破产，公司股票将面临退市风险。

股票8月20日停牌，21日复牌，还是能交易，但每天只有5%的涨跌幅，基本就是在一个退市倒计时的氛围里晃荡。

更关键的是转债，中装转2。交易所已经定了，9月3日停牌，9月18日是最后的转股日。过了这一天，你要是还没转，剩下的就只能去债权申报，排队等清偿。

重整里的债权清偿比例一般都不高，能拿回多少钱完全没谱。运气好的，有5-10万元以内的小额刚兑，运气不好的，直接一毛钱都拿不回来。

总结一下，过去那些正股退市或重整的转债结局：

| 名称     | 转债结局                       | 退市原因 |
| -------- | ------------------------------ | -------- |
| 搜特转债 | 实质违约→接近归零              | 正股退市 |
| 蓝盾转债 | 实质违约→接近归零              | 正股退市 |
| 鸿达转债 | 实质违约→接近归零              | 正股退市 |
| 普利转债 | 尚未兑付→接近归零              | 正股退市 |
| 全筑转债 | 5万以内小额刚兑                | 正股重整 |
| 正邦转债 | 10万以内小额刚兑               | 正股重整 |
| 岭南转债 | 10万以内小额刚兑，国资债务重组 | 转债到期 |

可以看到，涉及到正股重整或者国资债务重组的，还有小额刚兑的可能性，但若是正股直接退市，那就可能“分文没有”了。

这事儿的本质就是：公司能不能翻身，全看重整能不能顺利推进。要是引入新资金、重整成功，还能保得住上市，届时转债可能有个5万的小额刚兑，或者跟随正股大涨；要是失败，那就是直接退市，股票、转债都凉透。

中装转2，9月3日停牌，在此之前，多空双方估计会激烈博弈。

参考正邦转债案例，可能先涨到100元附近，临近停牌前又重新跌回90元附近。最佳的剧本是正股大涨带动转债负溢价转股。不管怎么样，这段时间就是情绪主导价格，没有合理估值可言。

这种债我把握不住，个人不打算参与。
