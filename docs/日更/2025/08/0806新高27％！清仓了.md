## 今日复盘
今天市场又是一路稳步高走，小票再创新高，可转债中位数也冲到了130.565。按照计划，我在盘中已经清掉了轮动策略里的所有转债仓位。
清仓的理由其实昨天文章已经讲过了，怕有人没看到，今天再简单说一遍。
中位数这个指标，代表的是全市场转债价格的中间值。现在是130.565，意味着有一半的转债价格高于这个数，一半低于它。
那它目前处在什么位置呢？
| 时间范围 | 所处百分位 |
| -------- | ---------- |
| 所有历史 | 99.46%     |
| 近1年    | 99.18%     |
| 近2年    | 99.59%     |
| 近3年    | 99.45%     |
简单说：**这已经是历史极限高位了，比过去99%以上的时间都高估。**
下图是转债等权指数和中位数的走势对比（蓝线是等权指数，彩色线是中位数），几乎是同涨同跌，高位和低位这些关键节点几乎完全同步。
![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250806185454707.png)
上一次中位数的历史高点，是2022年1月12日的131.501元，现在已经几乎打平了。所以说，行情虽然好，但估值已经不便宜了，在这个位置清仓其实不难，反倒是“继续留着”才是更难的决定。
毕竟，人声鼎沸的时候不走，等踩踏的时候想跑，往往就来不及了。
