# 周报：下周估计是科技、军工唱戏

## 周末利好利空消息

**利好消息：**

1. 重大资产重组新规落地：证监会修订《上市公司重大资产重组管办法》，首次允许私募基金“反向挂钩”，简化审核程序，激活并购重组需求，利好券商、创投及壳资源板块。  
2. 科技金融政策加码：七部门设立“国家创业投资引导基金”，聚焦科技创新和未来产业，同时央行降准释放万亿流动性，科技企业融资环境优化。  
3. 军工板块催化：央视报道歼-10CE实战表现超预期，哥伦比亚拟采购24架，叠加巴黎航展临近，军工出口预期升温。  
4. 中美关税调整带动出口：中美暂停部分关税后，集装箱运价指数上涨，外贸企业订单激增，航运、跨境电商受益。  

**利空消息：**

1. 证监会严打市场操纵：开出3张超亿元罚单，打击高标股炒作，短期抑制投机情绪（如化工、连板题材股）。  
2. 减持潮压力：多家公司发布减持公告（如联特科技、光华股份等），高位股抛压加剧。  
3. 外围市场风险：美国或加征新关税，穆迪下调美债评级，黄金价格跳水，外部不确定性增加。  

**总结：**

本周末政策面暖风频吹，重组新规和科技金融政策给市场打了强心针，军工、出口链也有故事可讲。但高标股被监管“点名”，减持潮和外围幺蛾子也得防着点。下周估计是科技、军工唱戏，炒差的得悠着点，别接最后一棒。
