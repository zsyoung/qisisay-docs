# 大跌，但没看出大风险

## 今日复盘

A股三大指数今日集体调整，截至收盘，沪指跌0.22%，深成指跌0.72%，创业板指跌0.96%，北证50指数跌6.15%。全市场成交额11397亿元，较上日缩量746亿元。全市场超4400只个股飘绿。

![](https://files.mdnice.com/user/7588/571aa903-78dc-4d99-90f6-15154152ce89.png)

今天市场风格继续分化，依然是涨红利和大盘，跌科技和小盘。成交量依然维持在1.1万亿，市场情绪有点冷。

红利低波和上证50分别涨0.46%和0.19%，科创50和中证2000分别跌0.48%和1.23%，北证50更夸张，暴跌6.15%，一夜回到解放前，相当于两周白涨。

小盘的下跌主要是流动性不足引起的，毕竟当前位置向上有压力，又没有利好事件催化指数突破压力位，时间越久，多头的压力就越大，所以横久必跌。大盘跌得不深，主要是靠银行、红利这种防御派在撑着。

目前看，这两天的调整都是正常调整，没看出来太大的风险，符合之前的判断，也没有出现绝对机会，继续按兵不动。

今晚小米YU7发布会，看完以后觉得实车真帅，配置真高，如果价格能对标特斯拉modelY，毫无疑问又将是一大爆。如果我再年轻几岁，可能还有冲动购买的欲望，不过现在有了孩子，目光已经不自觉移到6座大SUV上了。

### 转债方面

1、润达转债，未来3个月不强赎。现价161.67，溢价率19.01%。溢价已经透支了不强赎预期，其实我有点不理解它的持有者从哪里来的信心和勇气，能笃定它不强赎。要知道它一旦强赎，次日必然是15%+的跌幅。我认为它的合理溢价率就是7%-15%，现在已经略高估。

2、九洲转2，未来3个月不强赎。现价145，溢价率7.48%。溢价已经体现了不强赎预期。预估合理溢价率5%-10%。

3、豪24转债，未来3个月不强赎。现价247.394，溢价率0.26%。利好。价格较高，预估合理溢价率2%-5%。

4、虽然九洲转2、润达转债不强赎，但是不代表赌它们不强赎的投资者是对的，这种风险收益明显不对称的事情，做的次数越多亏的概率越大。接下来要警惕银轮转债，它还有2天满足强赎条件，溢价率还有8.16%，非常危险。

5、涨幅榜第一的精装转债，正股有资产重组概念，公司中天精装间接持有科睿斯33.3784%的股权，公司实控人为东阳国资，科睿斯有东阳国资入股，或存在科睿斯资产注入上市公司预期。
