## 周末利好利空消息
上周最后一天最后半小时市场大幅回调，不过大概率是国·家·队控盘，风险不大，市场情绪平稳。
当前这个区间大概率就是10%的震荡空间，可以做T，也可以等出现极端机会抄底，我觉得其实是很舒服的。
**利好消息：**  
1. 并购重组新规落地：证·监·会缩短审核周期，鼓励私募参与并购重组，利好券商及科技板块（如半导体、生物医药）。 
2. 绿色产业升级：国常会通过《制造业绿色低碳发展行动案》，重点支持新能源、环保技术等绿色产业，利好清洁能源和传统行业改造。  
3. 央行释放流动性：开展5000亿元MLF操作，缓解银行资金压力，利好金融、地产等板块。
4. 人民币汇率上涨：离岸人民币涨至7.17，创去年11月以来新高，增强外资流入信心。
5. 科技领域突破：华为昇腾芯片国产替代加速，全国首个开源鸿蒙机器人操作系统发布，全固态电池技术进入预量产阶段，推动科技板块活跃。  
6. 医药板块走强：创新药企业海外合作扩大，主力资金连续6周净流入医药生物板块，抗癌药研发进展提振行业预期。  
**利空消息：** 
1. 贸易摩擦升级：特朗普威胁对欧盟加征50%关税，并拟对非美制造iPhone征收25%关税，加剧全球贸易不确定性，拖累消费电子和出口链板块。  
2. 解禁潮压力：下周A股超160亿元市值解禁，万达轴承、炬芯科技等个股面临抛压。
3. 外围市场拖累：美股因贸易摩擦风险下跌，纳指跌幅超1%，或影响A股开盘情绪。 
**总结：**  
这个周末政·策·利好不少，并购重组和绿色转型是主旋律，外资和险资也来撑场子，科技、医药这些有硬货的板块算是有了底气。但坏消息也不少，川普又搞关税威胁，加上技术图形走坏，市场情绪有点绷不住。
下周估计得先消化利空，指数可能探底回升，医药、黄金这些防御板块能扛事，消费电子和苹果链得小心点。
