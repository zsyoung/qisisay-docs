# 这个板块技术面高位了，要小心！

## 今日复盘

今天市场继续延续燥热情绪，成交量又上到1.5万亿，指数是一路慢慢爬，氛围挺热闹。

可转债这边也不差，等权指数涨了1.11%，中位数直接蹿到了126.8元，距离130元的清仓价只有一步之遥了。

板块轮动挺快，热点有好几个。

PCB是今天的热点之一。龙头债景23转债直接带头冲，明电双子、东材转债、宏丰转债也不甘落后，轮着涨。这波上涨主要是因为英伟达获批，将开始向中国市场销售H20芯片。

然后鸿蒙又刷存在感了。原因是坊间那波“9月30前强制适配”小作文开始疯传，说明年不适配就约谈+查税，挺狠。但这消息真假难辨，有的分析师站队“基本属实”，有的论坛大佬直接说“老早就知道了”。但说句实话，资金不管真假，看到关键词就扫货。延华智能、润和软件这些涨得飞起。转债方面，科蓝转债和恒锋转债搭了点顺风车，不过我个人觉得这条线风险不小，消息真伪迟早要分出胜负，最好谨慎点。

再说物理AI，黄仁勋随口一句“下个浪潮是物理AI”，就硬生生炒起来了。按我的理解，这玩意就是AI+机器人那一套，给AI加上“动手能力”。但尴尬的是，今天反而“编码”概念更强，金现代直接20CM涨停……多少有点离谱。

医药股今天也很强，尤其是创新药那块。主要是国家集采那边明确了个“反内卷”原则，市场直接理解成“暂时不杀新药”，利好情绪直接拉满。很多创新药个股涨十几个点。转债这边，塞力转债、博瑞转债，妥妥的龙头债，也都是之前轮动的老面孔了。**不过创新药的技术面日、周、月线都到高位了，短期风险不小，要小心。**

你要说这些股债的基本面很好吗？并没有。很明显最近的好行情都是资金炒作带来的，情绪才是短期定价权。

群里现在也分成了两波情绪，一波是拿不住的已经清仓了，另一波是觉得踏空了想满仓追涨的。

我觉得在看不清未来的当下，半仓就最合适，进退有据。

毕竟韭菜永远不缺机会，但是缺本金。钱是赚不完的，但能亏完。
