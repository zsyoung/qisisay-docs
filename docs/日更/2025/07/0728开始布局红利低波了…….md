# 开始布局红利低波了……

## 今日复盘

今天又是熟悉的剧情：盘中跳水、尾盘拉回，成交量1.76万亿还在高位，说明市场内外的换手依旧很激烈。一边是老资金想见好就收，另一边是场外资金等着低吸，小盘股又一次跑赢红利板块，风险偏好明显还是在的。

不过要注意，**融资融券余额已经突破3月的高点**，上周五冲到了1.96万亿，基本算是满负荷了。也就是说，虽然市场看着还挺嗨，但新增的“弹药”其实差不多耗完了。

这段时间的成交量基本稳定在1.7-1.9万亿上下，背后其实就是资金在持续换防：涨多的在撤，新进的在追。但这些**新资金成本更高**，一旦行情不给正反馈，很容易从“接力”变成“砸盘”。

指数方面，全指已经摸到5238点，离去年10月的5252点就差一口气。要么突破刺激一波新高情绪，要么被解套盘砸下来，谁先动都不奇怪。

周期股开始退热，特别是上游方向，今天不少品种收盘飘绿，焦煤、玻璃、碳酸锂主力合约直接跌停，多晶硅也跌了快6%。修水电站的消息上周炒得太猛了，预期拉得过头，现在这波回调其实是个正常修正。

**红利低波**今天也值得提一句，我开始建仓了。它已经从高位回调了近3%，日线上回到支撑位。这类偏防守的资产，往往和高波动行情形成跷跷板。目前4.5%+的股息率，在当下利率环境下依然有上涨潜力，适合慢慢布局。

**我的看法：**

现在这行情，是热，但也危险。杠杆顶了、新资金贵了、情绪还激进——只要一个环节出问题，回调可能就来得很快。短期别激进，仓位控制好，有波动时倒是可以留意一些防守型机会。
