# 大跌如期而至

## 今日复盘

今天这波跌，其实我是挺欣慰的。不是说看空，而是该跌了，情绪之前冲得太猛了。

这次没像前几天那样盘中硬拉，反倒更像是主动给市场降温。甚至尾盘那点小拉升，说实话，不如不拉，明天搞不好还少点反弹的劲儿。

成交量今天又刷高了，1.9万亿，说明热钱还在。但情绪这块，已经有点透支了。前几天那帮融资冲进去的，今天开始“反杀”自己人了也不奇怪——典型的韭菜换防，上一波刚割完，新一波就排着队站岗了。

最典型的就是周期那边，上周不少人觉得水电站这个预期能带起一波大牛市，一窝蜂冲进去了，结果回头一看全被挂在了山顶。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250731164334309.png)

这波回调，我觉得挺健康，底下均线也撑着，不太会深跌。但市场在调整，红利、债这类低波资产可能又会开始有表现。比如30年国债今天就大涨了；红利低波早上跌得凶，尾盘还拉起来一点。

说到底，拉估值拉久了，总得回归下基本面。经济三季度大概率会掉到5%以下，但也不用一听这就慌。回头看，美股历史上几轮大牛市，都是在GDP增速下降时启动的。当做生意越来越难，资金自然会更愿意抱紧好公司、好资产。

牛市节奏就这套路：业绩改善是二级火箭，情绪和流动性才是第三阶段点火的关键。节奏错了，就容易被反复收割。

什么叫节奏错了？就是高位还要愣冲，低位反而不敢加仓只想割肉——这个状态，最容易被教做人。

转债中位数今天回撤了不少，已经从129多掉到了127，继续半仓轮动。

3、盘点一下涨停的正股们：

1）**辅助生殖概念**，理由：北京将辅助生殖纳入医保支付范围

2）**东数西算/算力概念**，理由：微软、Meta等业绩和资本开支超预期；OpenAI或将发布GPT5

大元泵业/大元转债：公司表示已完成在储能及数据机房液冷领域的产品储备，已开始小批量接受客户订单，公司与华为、中兴在IDC温控领域推进屏蔽齿轮泵项目

3）**医药概念**，理由：行业龙头再签大单；第11批药品集采启动，不再以简单的最低价作为参考

奇正藏药/奇正转债：公司是国内藏药龙头企业，主要从事新型藏药的研产销，奇正消痛贴膏是主打产品；目前已与钟南山院士团队合作开展科研

4）**雅下水电概念**，理由：雅鲁藏布江下游水电工程开工，总投资约1.2万亿

文科股份：据2025年7月23日互动易：公司前期已经在跟进雅鲁藏布江项目的环境生态工程业务。今天上涨应该是资金高切低，低位相关概念股补涨
