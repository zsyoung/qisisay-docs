## 科创债
昨天的文章提到国债最近跌了很多，有群友拿着科创债ETF的截图，问我这玩意儿是什么？值不值得买？今天刚好也给大家科普一下。
7月17日确实上新了一批科创债ETF，拿规模最大的科创债ETF华夏（551550）举例，先讲清楚定位：科创债ETF不是投股票，更不是科创板ETF，它是投一篮子**央国企科技龙头发的信用债**。比如中芯国际、中国电科系、阳光电源这些公司——你不是买它们的股，而是“借钱给它们赚利息”。
因为底层资产是高信用评级的央国企债（大多AA+及以上），叠加科技产业扶持政策，整体安全性比很多人想象中要高不少。但同时，它也承担了一定**行业波动和信用利差风险**，回撤比利率债略大。举个例子：科创债里半导体占比38%、新能源近30%，赛道涨跌，会直接反映在债券估值上。
![数据来源：wind](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/IMG_4166.jpg)
表现怎么样？和政金债ETF比一下：
| 指标     | 科创债ETF       | 政金债ETF |
| -------- | --------------- | --------- |
| 年化收益 | 4.6%左右        | 4.3%左右  |
| 波动率   | 稍高            | 极低      |
| 最大回撤 | -5.3%（2022年） | -2.9%     |
| 夏普比率 | 稍弱            | 更稳      |
拿历史数据来说，年化收益4.6%左右，比货基、短债略高；最大回撤约5%，波动比纯利率债大一些，但比股票温和很多。结论就是：**收益略高、波动稍大、回撤可控，但远低于权益资产。**
**它适合啥人？**
1. 不想碰股市，但想赚比货基多点利息的；
2. 想在债里配置一点“科技弹性”的；
3. 作为股市高位“缓冲带”来做防守的。
**不过也要注意几点风险：**
1. **信用风险虽小，但不是0**：毕竟是企业债，如果某行业爆雷（比如半导体突遭海外制裁），估值还是会动。
2. **行业集中度高**：新能源+半导体合计占比超65%，周期性风险依然存在。
3. **流动性不如传统债ETF**：目前规模百亿级，T+0可交易，但大额操作时注意溢价/折价。
一句话总结：科创债ETF是一个兼顾安全性和产业收益的信用债工具，适合稳中带点进攻的配置需求。
如果要投资的话，建议优先选择规模最大的，比如**科创债ETF华夏（551550）**，上市仅两个交易日，规模便从期初的29.61亿元增长至152.59亿元，规模增长超4倍，目前领跑同类ETF，资金抢筹特征显著。
