# 中证全指突破去年最高点！

## 今日复盘

今天又是熟悉的剧本：早盘吓你一下，午后拉回来收个稳稳的。最关键的是，中证全指终于突破了去年10月那个高点，干到了5252上方，技术上算是真·二级火箭启动了。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250729205405871.png)

但说句实话，这火箭也就技术上看着像，情绪和结构上完全不是那么回事。看今天指数红得挺亮眼的，创业板直接飙了快2%，但实际上3000多只个股是跌的。也就是说，**指数在飞，个股在哭**，典型的“看着热，其实没你啥事”。

现在的市场，很明显是热钱抱团趋势股，CRO、新能源、先进封装这些方向继续爆冲，资金扎堆进攻型资产；反过来，红利、债券这些稳健方向，完全没人搭理，股息再高也被冷落在角落里。

成交额干到了1.8万亿，换手率高，典型的强结构轮动——涨多的在撤，新的接着冲。但你只看量，看涨幅，不看个股分布，**很容易误判为全面行情**，其实不是，大部分股票根本没怎么涨。

**总结**：

1、当前市场节奏依然是成长和科技在“开跑”，防御性方向偏佛系，钱都往“牛股趋势”聚集。

2、这样的行情像“局部二级火箭”：指数可以创高、热点能接力，但底层资金并不均衡，主要是短线资金快速轮动。

3、操作上，我在逐步减仓成长科技ETF，逐步调入红利低波这类防御资产。

4、这情绪维持下去还可以，但不宜盲目扩仓，重点还是防守优先、继续轮动。

5、牛市不会一蹴而就，一定有回头的机会，求胜之前先虑败。
