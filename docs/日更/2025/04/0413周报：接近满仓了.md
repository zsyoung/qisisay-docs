# 周报：接近满仓了

## 下周观察

上周一开盘暴跌，但是之后低开高走，周K线画出一条长长的下影线，市场情绪惊疑不定。

但是启四确信随着政策的必然加码，以及PPI和ROE在底部抬头，今年还是能走出牛市。股债性价比也显示出当前位置处于钻石坑。

短期内技术面有压力，但长期确定性很强。

对全球大类资产的看法，可以看看[昨晚的文章](https://mp.weixin.qq.com/s/gYYGbQ2f3ikwcJkk6oWnkw)。

……

梳理一下周末利好利空消息：

**利好消息：**

1. 政策支持与流动性注入。央行等四部门联合发布指导意见，提出加大体育产业金融供给力度；平准基金再度入场托市。
2. 关税压力部分缓解。美国海关宣布豁免部分中国进口科技产品（如智能手机、电脑、半导体相关产品）的“对等关税”。
3. 消费刺激加码。国家税务总局推广“离境退税即买即退”服务，叠加多地发放消费券，饮料乳品、商业零售板块受益明显。
4. 外资信心与市场回购潮。富达、景顺等外资机构集体发声看好A股，认为估值优势显著且政策托底预期强，科技、消费板块受关注。
5. 回购潮。A股上市公司回购及重要股东增持规模井喷。

**利空消息：**

1. 外部贸易摩擦持续。中美关税战不利于出口型企业。
2. 美国商务部制裁11家中国实体，涉及稀土、科技等领域。
3. 市场技术性回调压力。上证指数连续三日缩量至1.35万亿元，技术面呈现见顶信号，叠加外部风险与内部获利盘抛压，短期或面临中阴线级别调整。

总体看，下周上半周技术面有压力，但长期看好。
