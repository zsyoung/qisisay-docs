## 周末利好利空消息
**利好：**
1. 跨境支付有新动作：内地和香港“跨境支付通”上线了，支持手机号直接汇款，速度快还免费，银行和金融科技公司有望受益；
2. 医疗器械政·策加码：国家药监局出了新规，鼓励医疗机器人、AI医疗等高端器械发展，对国产替代是个大利好；
3. 并购消息不断：像芯联集成、渝三峡这些公司在搞并购扩张，意味着行业整合和转型动作加快，容易带动一波炒作；
4. 旧换新真金白银来了：中央将下发1380亿资金支持消费升级，重点是新能源车和智能家居，这两块短期有催化；
5. 创业板改革又进一步：证监会发布“1+6”新规，搞了个“科创成长层”，对半导体、AI等硬科技算是明显利好。
**利空：**
1. 地缘局势紧张升级：美国突然对伊朗动手，轰炸了三个核设施，霍尔木兹海峡可能有封锁风险，油价蹭蹭涨，避险情绪飙升；
2. 大股东套现不断：有十多家公司发布减持公告，有的减持比例高达3.8%，这对市场信心会有明显压制；
3. 海外政·策打击来了：美国可能要取消台积电等企业在中国的芯片豁免资格，欧盟还限制中企参与医疗招标，外部压力增加；
4. 市场供需失衡苗头：今年IPO受理节奏加快，已经快赶上去年总量，美债流动性也在恶化，整体流动性紧张感在加剧。
**总结：**
这一周末，政·策面偏暖，有扶持科技和消费的利好，但外部冲击（中东打起来了、美欧政·策压制）带来明显不确定性。大盘可能走分化行情。
建议多关注政·策扶持方向，比如AI、半导体、医疗器械，也可以适当防守一下能源、黄金板块。
市场情绪偏悲观，下周一大概率会跌，不过有危才有机。
