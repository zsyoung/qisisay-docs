# 周报：下周热点转债梳理，军工板块再传利好

## 周末利好利空消息

**利好消息：** 

1. 中美关系缓和预期升温：双方·领·导人6月5日通话释放暖意，确认将举行新一轮贸易谈判，叠加商务部放松稀土出口管制，市场对关税摩擦缓解的预期增强，刺激外资回流（美股、中概股、离岸人民币同步拉升）。
2. 央行万亿级流动性支持：6月6日央行开展1万亿元买断式逆回购操作，提前对冲节后资金压力，释放流动性宽松信号，利好大金融板块和权重股。
3. 陆家嘴论坛政·策·预期：6月18日陆家嘴论坛将发布跨境金融、债券市场“科技板”等重大政·策·，外资准入松绑等预期吸引长期资金布局。
4. 产业催化密集：印尼拟采购中国歼-10战斗机（军工出海突破），数据要素试验区落地（利好算力、AI科技），机器人、跨境电商等政策加码。 

**利空消息：**

1. 解禁潮压力骤增：6月A股解禁市值达2531亿元，中国电建、中无人机等个股解禁规模超百亿，短期抛压风险高。
2. 高位股减持潮：潮宏基、玉禾田等热门股大股东密集减持，中小盘股套现压力加剧。
3. 消费板块承压：飞天茅台批发价跌破2000元，白酒股持续萎靡；家电补贴阶段性暂停，政策退坡压制消费复苏预期。 

**总结：**  

这周末消息挺分裂的，一边是中美打电话、央妈撒钱这些“强心针”，另一边是解禁潮和减持潮的“冷水澡”。

大方向看，金融、科技、军工这些有政策撑腰的板块还能蹦跶，但白酒、家电这些消费股估计还得躺平。

下周重点盯两个事儿：一是周一伦敦的中美会谈能不能放出实锤利好，二是市场量能能不能顶住解禁压力。

操作上别追高，军工、数据这些有事件催化的可以低吸。
