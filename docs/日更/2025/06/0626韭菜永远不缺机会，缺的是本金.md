# 韭菜永远不缺机会，缺的是本金

## 今日复盘

今天大盘冲高回落，成交量依然维持住了1.6万亿，量能很不错。

但说实话，看着挺热闹，其实挺虚的。很多人一看到成交量1.6万亿，就觉得“是不是又有大行情要来了？”但你仔细想想，这波上涨背后有没有什么确定性的逻辑？基本面有没有实质变化？利率有没有大幅下行？政·策有没有明显发力？都没有。说白了，就是情绪拉了一下，估值修复一下而已。

股价=业绩×估值，业绩没什么提升，现在指数上涨纯粹就是估值修复。现在顶多是从低估修复到中估，**真要拔到高估，还得再等逻辑配合——要么业绩提升，要么宏观放水。**

行情逻辑分5种：

1. **利率上行+业绩起飞** → 估值拔升（经济复苏）
2. **利率上行+业绩无力** → 估值掉（通胀加压）
3. **利率下行+经济配合** → 股债双牛（最爽那种）
4. **利率下行+经济差强人意** → 估值有缓冲（典型日本宽松）
5. **利率下行+经济继续滑坡** → 熊市（最差那种）

现在这个行情，说实话也就第四种，估值可以拔点儿，但别太指望大涨。

真正能推一波的点在下半年，一是业绩预期可能改善，二是全球都在等美联储降息，我们这边也可能跟着松一松，到时候可能还有一小波机会，但现在嘛，别追高，稳住仓位更重要。

记住一点：**韭菜永远不缺机会，缺的是本金。**

千万不要去盲目预测谁谁谁不会强赎，事实证明那都是瞎猜和意淫。

我总说：**只应对，不预测，这才是投资的正确姿势。**

什么叫应对？你再喜欢再舍不得一只转债，在它马上就要强赎且溢价率还很高的时候，就应该果断避开，不强赎再考虑接回。

什么叫预测？比如这次的金陵转债，有网友信誓旦旦说它肯定不会强赎，在它强赎计数只剩2天且溢价率还有12%的时候坚持持有。结果人家到底还是强赎了，两天暴跌13%，而这个风险完全是可以提前规避掉的。
