# 白酒，还没到入场的时候

## 今日复盘

A股三大指数今日集体调整，截至收盘，沪指跌0.07%，深成指跌0.47%，创业板指跌0.84%，北证50指数跌1.34%。全市场成交额10916亿元，较上日缩量1892亿元。全市场超3600只个股飘绿。

![](https://files.mdnice.com/user/7588/2d9b740e-8822-4e1c-ace0-d37a5484de27.png)

市场成交量又缩到1万亿了，说实话今天的跌幅我还挺意外，没有期待的那么大。

防御类资产已经开始顶上了，国债、红利指数、银行，还有食品饮料板块都红了，白酒反弹不少，这波白酒大跌是因5月18日政·策限制公务饮酒而起，本身只是限制工作期间不能喝酒，公务接待不得提供酒类。但是落实到地方，出于内卷和自保，却变成了层层加码，严禁任何时间喝酒，又从禁酒令发展到了禁止去饭店，直接一刀切。

这次消费反弹，说白了是官媒发了篇文章撑了一下，告诉大家不要恶意曲解政·策。但是别高兴太早，这种层层加码不是一篇文章就能松动的。现在的白酒，虽然业绩还行，但还在杀估值阶段，风险还远没释放完全，等杀到逻辑崩了，杀到人人喊打，杀到大量白酒股进入红利指数了，那才是真正的入场机会。

本周从指数表现来看，北证、中证2000、恒生科技、黄金跌幅居前，都在2%+；中证1000、创业板、科创跌幅在1.5%+；红利上，依然是红利低波表现最好，涨幅1.02%，红利指数涨0.24%，中证红利跌0.42%；30年国债本周也不错，涨了0.84%，上周清仓过早了。国债的玩法还是应该根据市场热度和降息预期来判断，不能用短线思维。

整体来看，大盘其实就是在震荡，没方向感，中证全指距离下方的缺口4678还有1.6%的空间，距离200日均线4664也只有1.9%的空间。只要不是遇上特别恶劣的利空，大概率撑得住。

周末就看中东那边的消息了，老美到底会不会下场干伊朗。

### 套利不参与

今天的原油基金套利，我没参与，虽然溢价率高，但都是场内游资拉起来的，大概率进去后就是连续跌停跑不出来，而且现在原油位置太高了，日线、周线、月线全都突破了布林通道上轨，现在入场风险太大。

尾盘果然两只原油基金开始暴跌，南方原油LOF的溢价直接跌没，嘉实原油LOF的溢价也从20%+跌到了7.5%。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20250620213126485.png)

那几个商品LOF更不套利了，溢价都是今天拉上去的，平常成交额几十万，今天成交额3个亿，进去铁定被埋。

由于商品LOF的套利都是T+3到账，所以只看当天的溢价率去套利是不靠谱的，我们需要预测3天后市场价能高于我们申购的净值，所以不光要看溢价，还要看成交量，更要分析趋势，没那么无脑简单。
