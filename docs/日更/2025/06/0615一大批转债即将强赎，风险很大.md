## 周末利好利空消息
**利好消息：** 
1. 央行10天投放1.4万亿流动性：继6月6日1万亿逆回购后，6月16日再投放4000亿，创阶段性新高，缓解银行资金压力，直接利好金融股（如银行、券商）和利率敏感板块。
2. 中美经贸缓和预期：中美元首通话确认推进贸易谈判，商务部放松稀土出口管制，市场对关税摩擦缓解的预期升温，刺激外资回流。
3. 产业政策催化：陆家嘴论坛（6月18日）临近，市场预期跨境金融、债券市场“科技板”等政策将落地；工业机器人出口前5月增长55.4%，低空经济成新增长点。
4. 中东冲突：推升油价，石油、黄金及军工板块受益，中石油等权重股可能护盘。  
**利空消息：**
1. 中东局势与外围拖累：伊朗导弹袭击引发原油暴涨7%，欧美股市普跌（纳指跌1.3%），避险情绪升温，A股开盘或承压。
2. 解禁潮与减持压力：6月解禁市值达2531亿元（如中国电建、中无人机），叠加恒瑞医药、山东章鼓等多家公司高管减持，短期抛压风险高。
3. 经济数据疲软：5月M2增速7.9%、人民币贷款仅增0.62万亿，反映经济复苏乏力，消费板块（白酒、家电）受冲击（茅台批发价跌破2000元）。
4. 技术面破位风险：A股放量跌破3400点，短期均线支撑失效，要警惕进一步回调。
**总结：**  
这周末消息像坐过山车，央妈狂撒钱1.4万亿和中美缓和算是两针强心剂，但架不住中东闹腾、解禁潮压顶，还有经济数据拖后腿。
周一估计得先挨一锤子（低开），能不能爬起来得看金融股能不能顶住，外加成交量给不给力。
板块方面，政策撑腰的科技、机器人、新能源还能蹦跶，但消费电子、白酒这些得小心。
操作上别头铁追高，手里有现金的可以关注超跌的成长股，等量能放大再说。
