# 突然跳水！

## 今日复盘

A股三大指数午后集体走低，截至收盘，沪指跌0.44%，深成指跌0.86%，创业板指跌1.17%，北证50指数跌1.01%。全市场成交额14512亿元，较上日放量1386亿元。全市场超4000只个股飘绿。

![](https://files.mdnice.com/user/7588/de69e358-a100-4dcd-b687-2ef027b86070.png)

今天下午市场突然跳水，传闻是中美谈判出了点小插曲——说是中方态度比较强硬，临时离场了。

不过不用慌，这种“拍案而起，爷不玩了”的操作，在谈判里其实挺常见的，属于常规博弈手法。

谈判还在继续进行中，没破裂，只是双方在细节上拉扯而已。这种级别的博弈，从来都不是你来我往几轮就能搞定的，反复推翻、重来，才是常态。

跳水一出来，盘面上很快就出现护盘动作，300ETF的资金马上进场，感觉现在维稳节奏是越来越快。

虽然最后市场拉回来了，但这并不代表短期调整就结束了。毕竟之前大家对谈判预期太乐观了，现在可能正好借着这事修正一下情绪，那6月涨上来的那部分，也许要回吐点。

说白了，现在市场对经济数据反应平平，对谈判消息却特别敏感，说明行情还是在靠情绪和消息面带节奏。基本面上，业绩还是偏弱，通缩趋势也没缓解，反弹缺乏弹性，远远没到“强复苏”的时候。

未来市场可能会在红利价值股和小盘科技之间来回切换：如果政·策给力、没啥风险事件，那小盘科技就有戏；但要是谈判真谈崩了，黑天鹅飞出来，那还是乖乖回到红利+债券的老路上去。至于大盘股，全看国·家·队脸色。
