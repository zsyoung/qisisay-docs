# 尾盘突然拉涨，这两只又要炒？

## 今日复盘

A股三大指数今日涨跌不一，截至收盘，沪指涨0.01%，深成指跌0.11%，创业板指涨0.26%，北证50指数跌0.25%。全市场成交额13035亿元，较上日放量169亿元。全市场超2300只个股上涨。

![](https://files.mdnice.com/user/7588/48114680-7cab-4f90-a6b0-595cf28c2524.png)

最近微盘股真的是太疯了，天天涨、天天新高，年内已经涨了34%多，看得人都麻了。但你要问这些股票为啥涨？其实跟业绩一点关系都没有——纯靠情绪、靠钱堆出来的。

微盘指数咋编的？就是从全市场挑出市值最小的那400只股票，不管公司好不好，直接等权放进去，每月轮换。这些票基本都是20亿以下的市值，盘子小、流动性差，几千万就能砸出涨跌停，大资金进不去，公募也做不了，只有量化策略可以玩一玩（类似我们的转债轮动策略），普通人基本靠运气。

那为啥最近这么猛？一方面，今年经济说不上强，但流动性很宽松，IPO少了、减持收紧、融券不好搞，钱出不去，就只能在存量市场里炒来炒去。另一方面，今年散户大量涌入，上交所开户数暴涨，创21年以来新高，韭菜多了，市场热度自然就上来了，而微盘股天然就是培养和收割韭菜的土壤。

但说实话，这类风格不太健康。成分股很多都是亏钱的“问题股”，涨全靠估值拉上去。只要哪天情绪变了、监·管出手、或者出现黑天鹅，可能几天就直接跌崩。

比如今年3月中旬到4月上旬，短视频荐股被严查、关税战黑天鹅等一系列事件，导致微盘股半个月就跌去20%+。

短期看，这波不适合追涨，最好等等回调。长远看，微盘股策略很适合在A股生存，但是必须通过量化策略的方式参与。普通散户想参与，最好通过中证1000、2000这些宽基ETF，更稳一些。
