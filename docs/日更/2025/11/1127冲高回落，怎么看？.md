# 冲高回落，怎么看？

## 今日复盘

11月27日，A股三大指数涨跌不一。

盘面上，造纸印刷、电池、消费电子等板块涨幅居前；文化传媒、互联网服务、水泥建材等板块跌幅居前。

成交额方面，沪深两市成交额1.71万亿，较上一个交易日缩量736亿。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20251127201036944.png)

量能不够，市场就依然是调整状态，今天的创业板就很明显，早盘冲高后没多久就一路回落。

从60分钟线看，刚好在MA120处遇阻，11/20也是在这里遇阻后持续大跌。这条之前的支撑线接下来会成为不小的阻力线，直到利好消息到来，多头才有望冲过去。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/image-20251127210545829.png)

过去这几个月的市场，其实就是两条线在前面拉着走：科技在上面猛冲，周期在后面稳稳托着。但十月底之后，这个节奏突然有点变味了。

**1、三季报给了市场一个很关键的信号**

今年第一次出现——科技和周期的业绩一起回暖。这一下，就把资金的注意力从“只盯科技”拉回到“两条腿一起看”。

再加上科技涨得时间不短、机构仓位也堆得够高，市场自然开始找第二条腿来分担节奏。

**2、于是我们进入了一个明显的“再平衡阶段”**

现在的行情，不是科技掉头，也不是周期接棒，而是两边重新找位置。科技依旧是主线，但不会再像前几个月那样一家独大；周期的存在感明显提高了。

这种再平衡大概率要持续一两个月，其实在任何牛市里，你都会看到类似的小幅风格切换，挺正常的。

如果把时间拉长到2026年去看，科技还是中长期逻辑最清晰的那条线，但周期的机会相比上半年已经更明朗。

**3、其实之前几轮牛市也都走过类似的路**

05-07年是金融在唱主角；14-15年谁都能蹦跶几下；19-21年核心资产占着C位。不同的年代，不同的主线，但共同点很一致：上涨过程中，一定会穿插调整，也一定会有风格切换。只要趋势没坏，就不必因为几天的情绪波动过度反应。

**4、所以接下来怎么看比较合理？**

我更倾向把现在当成：主线科技没变，只是行情会慢慢扩散到更多板块。像电力设备这种同时沾点科技、又踩着周期逻辑的方向，也开始陆续走强。

如果要拆得更清楚些，我现在的理解是：

1）**周期看PPI和反内卷链条**：钢铁、有色、化工、电力、光伏——都属于利润改善、供需趋稳的方向。

2）**科技看AI、硬件、自主可控**：半导体、通信、软件、军工，这些还是长期确定度更高的一侧。

3）**底仓看红利和黄金**：红利、黄金这种能扛回撤、能守节奏的，依旧是组合里的压舱石。

成长风格后面会不会继续？大概率还是会。但不急，等这一轮再平衡走完，市场会自己给出答案。

整体来看，科技依旧是长期主角，只不过镜头开始给到更多板块了。
