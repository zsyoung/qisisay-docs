# 乐歌转债进入前5

大家好，我是启四。

今晚讲两件事。

## 一、指数回撤

周日我在三要素可转债名单中提示了，可转债等权指数以及双低指数已经达到历史最高点，溢价率小于20%的转债已经极其稀少，今天两个指数已经有了回撤的迹象。

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20211116223054136.png)

目前按照既定策略保持低仓位，耐心等待杀溢价结束。

其实近两个月以来，低溢价转债策略的收益并不尽如人意，原因已经在之前的文章[低溢价转债策略回撤严重，问题找到了！](https://mp.weixin.qq.com/s/RrzBz4C1WWQvGlY_Ox-Q1Q)中提及：

> 低溢价策略的主要超额收益来自于对溢价率的高抛低吸，而正股趋势则贡献负的收益。

而高频率的调仓是导致正股超额收益减少的主要原因，经测算，2021年之后，20天轮动要比1天、5天、10天轮动更好。

所以，**没有永远正确的策略**，优秀的策略需要在运行中不断修正和调整。

## 二、偏离策略

最近，精力主要放在了[偏离策略](https://mp.weixin.qq.com/s/AZUgoPtm1S1FsEMG3iEYPg)上，原因是这个策略可以捕捉到一些热点转债，埋伏到一些上涨概率较大的转债。

而且捕捉到的一些转债即使溢价率很高或者价格很高，依然持续上涨，比如康隆转债、东缆转债、清水转债、中天转债等等，这个策略波动大，但是进攻能力确实强，转债日内5%以上的涨幅简直是家常便饭。

有位名人说过：「**不管黑猫白猫，能抓到老鼠就是好猫。**」收益平庸的低溢价策略和收益激进的偏离策略到底用哪个，就见仁见智了。没有必要对130元以上的转债天然畏惧，只要策略能带来收益，风险可控，就可以大胆用。

今日偏离策略前20：

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20211116225213242.png)

宏川转债掉出名单，乐歌转债正股大涨5.1%，进入名单前5。
