# 近期哪些转债还有可能下修？

大家好，我是启四，长期分享套利机会。

有小伙伴让我分析一下近期哪些转债有可能下修，话不多说，开始吧：

## 哪些转债有可能下修

根据之前总结的[可转债的下修套利方法](https://mp.weixin.qq.com/s/Z7ERqwjIwJV7ftqb_F2tpg)，我们可以知道：

下修套利发生的情况一般有三种：**避免回售、促进转股、大股东减持**。

其中容易把握和潜伏的是第一种和第三种情况。

### 1、先说第三种情况：大股东减持

这种情况需要**关注大股东配售比例较高且发行前正股走势不好的转债。**

看了下集思录的待发转债：

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20211120224905059.png)

近期即将上市的转债中，股东配售比例高的不少，但是正股现价比转股价都在90%以上，意味着这些转债的正股走势都不算拉胯。

所以这种情况pass。

### 2、然后分析第一种情况，避免回售

避免回售是为了防止触发回售条款，回售条款一共有两个，一个是有条件回售条款，一个是附加回售条款。

![img](https://cdn.nlark.com/yuque/0/2021/png/2369864/1629296332631-e133c33d-2d60-4830-8e60-86e0a799572d.png)

![img](https://cdn.nlark.com/yuque/0/2021/png/2369864/1629296332642-36ae9ea5-a283-4566-b76f-a5e312405ff6.png)

后者一般是改变募集资金用途，如涉及回购注销股份并减资，由于数据不好找，不在我们今天讨论的范围。

前者则需要**潜伏低价高溢价且即将或者已经到回售期的转债**，博弈下修机会。

我在集思录筛选出了**价格<115元，剩余年限<3年**的可转债，并按照剩余年限从小到大排序，共19只转债，如下图：

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20211120230018344.png)

值得一提的是，这些转债中有多达10只转债有过一次或者多次下修经历：

![](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20211120230037826.png)
