# 晶科转债意味的惊喜

大家好，我是启四。

今晚讲三件事。

#### 晶科转债套利成功

上周在文章[019 套利方法 晶科转债折价套利赚了5.21%](https://mp.weixin.qq.com/s/NhwY5yLnsUccMqAdwEgeEA)中给大家总结了折价套利的技术方法，并以我买入的[晶科转债](https://xueqiu.com/S/SH113048?from=status_stock_match)为例，具体讲解了原理和我的买卖点。

上周五，根据沪市统计，[晶科转债](https://xueqiu.com/S/SH113048?from=status_stock_match)尾盘总共转股了53万手，5个多亿。我判断周一正股获利盘抛压巨大，一定会低开。

果不其然，正股今天4%低开，以上周末折价安全垫3.3%计算，尾盘转股套利党反被当成了韭菜。让我没想到的是正股盘中走势反转，加上折价收敛，直接带动[晶科转债](https://xueqiu.com/S/SH113048?from=status_stock_match)冲到了最高价178.5元。

我的回落条件单随后也被触发。本次套利，建仓价163.6元，止盈价176.39元，共持仓4天，盈利7.79%。

折价套利属于套利中的高阶操作，靠认知差赚钱，如果对可转债和正股没有充分的认知，很容易镰刀变韭菜，你想套利别人，反而中了别人的套。

建议大家去回顾一下上面的教程。

#### 希望转2记得申购

明天[希望转2](https://xueqiu.com/S/SZ127049?from=status_stock_match)发行，这是自8月中旬以来，发行规模最大的可转债，达到了81.5亿，中签率会相对高一些。
