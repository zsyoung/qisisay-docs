# 冲！来不及了，嘉美转债！

2021年9月1日，我们参与了天合转债夜市委托套利，成功盈利，今天机会又来了。

套利逻辑在此：[如何夜市委托，套利快人一步？](https://mp.weixin.qq.com/s/nfRbcEU2wRX39u9csBzRnQ)

这次嘉美转债风险更小，收益更大，下面我讲一下怎么操作，仔细看：

## 1、预估嘉美转债上市价

嘉美转债(127042)，深市转债，根据我们之前总结的可转债交易规则图，可以看到这样几个信息：

1、上市首日开盘价最高130元；

2、涨停30%后，会临停至14:57，可以委托和撤单；

3、收盘集合竞价可以再涨10%，到143元。

![image-20210907131404135](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20210907131404135.png)

现在嘉美转债的转股价值是116.63元，结合近期新债动辄35%以上的溢价率，嘉美转债涨超130元是板上钉钉的，涨超143元的可能性也非常大。

**所以我计划以143元的价格夜市委托抢筹。**

## 2、准备套利资金

这就不用多说了，深市转债以10张为单位，所以最少要准备1430元，要多套利的资金倍乘即可。

下午15:00之前把套利资金转进证券账户。

## 3、关闭余额理财

这一点非常重要！！决定着你能否快人一步，成功抢筹。

为什么要关闭我就不细讲了，文章开头那篇套利逻辑文章里讲过了。

我以智投app举例，讲一下如何关闭余额理财：

**路径：**交易——余额理财——设置——余额理财解约——马上解约——确定解约

**注意：**必须在下午15:00之前操作好！

![WechatIMG121](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/WechatIMG121.jpeg)

## 4、夜市委托抢筹

然后就是夜市委托抢筹的操作了。

智投app的夜市委托一般在下午17点左右开启：

![image-20210907133008495](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20210907133008495.png)

具体操作方法：

1、在app里搜索嘉美转债，打开买入页面；

2、输入限价委托143元；

3、输入要买的张数，不能大于可买张数；

4、下午15:00之后，17:00左右，不断点击确认买入。

注意：因为夜市委托是17:00左右开启的，所以在15:00收盘之前提前点击买入是无效的申报。

![image-20210907133224365](https://zsyoung.oss-cn-hangzhou.aliyuncs.com/img/img/image-20210907133224365.png)

## 5、风险以及如何止盈？

风险在于，夜市委托抢筹成功之后，明天嘉美转债的价格能否达到143元。所以明天要密切关注正股嘉美包装的走势。如果明天正股大跌，我们就在明天14:57之前撤单即可。

关于如何止盈，推荐使用回落卖出条件单，这是智投app独有的功能，可以自动监控价格帮我们卖出，抢筹成功之后设置参数「大于143元，回落1%卖出」即可。
