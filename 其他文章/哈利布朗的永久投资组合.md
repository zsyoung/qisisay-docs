# 哈利-布朗的永久投资组合是什么

**哈利·布朗的永久投资组合（Permanent Portfolio）** 是一种投资策略，由美国金融作家和投资顾问哈利·布朗（Harry Browne）提出。这一策略的核心思想是，投资者无法预测未来经济环境，但可以通过合理分配资产，应对各种经济情景，保持财富的稳定增长。

### **永久投资组合的四大资产配置**

哈利·布朗将经济分为四种可能的状态：**经济繁荣、通货膨胀、通货紧缩、经济衰退**。对应这些情景，他设计了一个“永恒适用”的投资组合，每种资产针对一种经济状态。

#### **1. 股票（25%）**

- **应对经济繁荣**：经济繁荣时，企业盈利增加，股市通常表现良好。
- **投资选择**：优选标普500等宽基指数基金或蓝筹股ETF。
- **目标**：获取经济增长的红利。

#### **2. 长期国债（25%）**

- **应对通货紧缩**：经济衰退或通缩时，利率下降，长期债券价格上涨。
- **投资选择**：高质量的政府长期债券（例如，美国30年期国债）。
- **目标**：在利率下行周期中获得资本增值。

#### **3. 黄金（25%）**

- **应对通货膨胀**：通胀时期，货币贬值，黄金作为硬通货价值保值。
- **投资选择**：实物黄金、黄金ETF或黄金矿业基金。
- **目标**：对冲货币风险和通胀风险。

#### **4. 现金（短期国债，25%）**

- **应对经济衰退**：经济低迷时，现金为王，流动性最重要。
- **投资选择**：高流动性的短期国债（例如，美国3个月国债）。
- **目标**：为组合提供安全垫，并在市场低迷时捕捉机会。

------

### **永久投资组合的优点**

1. **抗经济波动**：四种资产各司其职，无论经济走向如何，总有一部分资产能跑赢。
2. **操作简单**：仅需定期（如每年）再平衡资产比例，维持25%分配。
3. **风险分散**：不同资产之间相关性低，降低了单一市场波动的风险。
4. **长期稳健**：历史数据表明，该策略在经济剧变中表现优异。

------

### **永久投资组合的不足**

1. **收益中规中矩**：因为过于强调风险控制，收益率通常低于风险较高的全股或偏股型组合。
2. **黄金争议**：部分投资者认为黄金不产生现金流，机会成本较高。
3. **不适应极端环境**：比如超级通胀或极度紧缩时期，该组合的表现可能不及针对性策略。

------

### **适合人群**

- 不擅长市场预测或没有时间频繁管理投资的人。
- 希望资产配置稳健，避免大幅波动的人。
- 对财富保值和长期收益有较高需求的人。

------

### **总结**

哈利·布朗的永久投资组合是一种简单、稳健的策略，强调分散风险，以应对不可预测的经济变化。尽管回报不一定很高，但其平衡的特点使它成为许多人长期投资的理想选择。如果你不想花太多精力盯盘或预测市场，这个策略可能非常适合你！









# 年化收益和回撤情况



哈利·布朗的永久投资组合（Permanent Portfolio）的年化收益和回撤情况，主要取决于资产类别的表现及其在不同经济环境下的相关性。根据过去几十年的历史数据（以美国市场为例），永久投资组合的年化收益和回撤大致可以总结如下：

------

### **1. 年化收益**

- 长期年化收益率：约6%-9%
  - 这一范围基于对过去40多年的美国市场数据的分析（假设配置为25%股票、25%长期国债、25%黄金、25%现金）。
  - 由于包含了黄金和现金，收益率会低于纯股权投资组合，但胜在稳定性。

#### **收益来源**

- **股票**：长期来看，股票是增长引擎，提供较高的年化收益（6%-10%，取决于市场）。
- **长期国债**：在利率下降或通缩时表现强劲，年化收益约3%-5%。
- **黄金**：对抗通胀的利器，年化收益约3%-6%，但波动较大。
- **现金**：短期国债提供安全性，年化收益约1%-2%。

------

### **2. 最大回撤**

- 长期最大回撤：约10%-15%（历史上在极端情况下，可能达到20%）。
  - 这一数值远低于纯股票组合（如标普500的最大回撤可达50%以上）。
  - 在2008年金融危机期间，永久投资组合的回撤约为**10%**，相比之下，标普500回撤超过**50%**。

#### **低回撤原因**

- 资产高度分散，且相关性低。例如：
  - **股市下跌时**，债券价格通常上涨（避险需求）。
  - **通胀高企时**，黄金通常表现强劲。
  - **市场恐慌时**，现金提供流动性。
- 这些特性使得组合整体波动较小。

------

### **3. 历史表现案例**

根据实际历史数据（以美国为例）：

- **1970-2020年**，永久投资组合的平均年化收益率约为 **8.5%**。
- 2008年金融危机：
  - 标普500下跌约**-37%**。
  - 永久投资组合仅下跌约**-8%-10%**。
- 2022年高通胀和加息环境：
  - 黄金在高通胀期间表现较强，股票和债券下跌时，永久投资组合回撤相对有限（约**-5%到-10%**）。

------

### **4. 对比其他组合**

| **投资组合**      | **年化收益率** | **最大回撤** |
| ----------------- | -------------- | ------------ |
| 标普500指数       | 9%-10%         | 50%以上      |
| 60/40股债平衡组合 | 7%-8%          | 25%-30%      |
| 永久投资组合      | 6%-9%          | 10%-15%      |

------

### **总结**

- **收益方面**：永久投资组合牺牲了一定的收益率，换取了稳定性和低波动。年化6%-9%的收益适合追求长期稳健增长的投资者。
- **回撤方面**：组合的最大回撤通常不会超过15%，远低于纯股权或传统60/40组合，非常适合风险厌恶型投资者。

如果你想追求“稳中有进”的财富增长，同时避免剧烈波动，永久投资组合是一种非常值得考虑的投资策略。





# 有没有优化方法

哈利·布朗的永久投资组合设计追求简单、稳定和抗风险，但这也意味着可能存在收益率偏低和通胀环境下现金拖累的问题。如果想进一步优化，可以根据自身需求、市场环境和投资目标进行调整。以下是一些常见的优化方法：

------

### **1. 调整资产类别权重**

永久投资组合的基础是每种资产25%的分配，但可以根据实际需求调整权重：

- **降低现金权重**：现金的长期收益最低，如果对流动性要求不高，可以减少现金权重至10%-15%，腾出的资金可增加到股票或债券。
- **增加股票权重**：适合更长投资期限和更高风险承受能力的人，比如将股票权重提高到30%-40%以提升长期收益。
- **调整黄金权重**：黄金表现与通胀和避险情绪相关，如果预计通胀环境较稳定，可以降低黄金权重至15%-20%。

### **2. 精选股票资产**

- 原版永久组合中，股票部分通常使用**标普500指数**或类似的大盘股指数。
- 优化方法：
  - **引入全球化配置**：添加国际市场（如MSCI全球指数）或新兴市场指数基金，分散风险。
  - **引入价值或高质量股票**：选择表现更稳定的价值股或高分红股票，可以增强回报和防御性。

### **3. 精选债券资产**

- 原版组合使用的是**长期国债**，但它对利率变化较敏感，在利率上升周期可能表现不佳。
- 优化方法：
  - **增加中短期债券**：用中期债券（如3-7年期）或债券ETF部分替代长期债券，降低利率波动风险。
  - **引入通胀保值债券（TIPS）**：在通胀压力较大的情况下，TIPS可以对冲实际购买力的下降。

### **4. 替换现金部分**

现金的长期回报率最低，可以用以下资产替代或补充：

- **短期债券基金**：收益率通常高于现金，且流动性较好。
- **货币市场基金**：类似现金，但收益率与利率挂钩，回报更高。
- **高收益储蓄账户**：特别是在高利率环境中，可以替代现金部分。

### **5. 黄金部分优化**

- 黄金的表现受市场情绪和通胀预期影响较大，长期回报有限。可以用其他抗通胀资产代替：
  - **大宗商品指数基金**：分散投资于多种商品，降低波动性。
  - **黄金+白银组合**：将黄金部分改为黄金和白银的组合，利用白银的高弹性来提升回报。
  - **REITs（房地产投资信托基金）**：在通胀环境下，房地产资产具有一定的抗通胀能力。

### **6. 动态调整策略**

永久组合的设计假定市场会经历各种经济环境，但如果你能提前捕捉宏观趋势，可以进行小幅动态调整：

- **通胀环境**：提高黄金和大宗商品权重，降低债券权重。
- **经济增长周期**：增加股票权重，适度减少现金和债券。
- **经济衰退**：提高债券权重，同时增加部分防御性股票（如消费必需品、医疗行业）。
- 注意：动态调整需要对市场有一定判断力，避免频繁交易。

### **7. 降低成本**

优化策略的同时，尽量控制投资成本。选择低费用的ETF或指数基金代替高费率的主动管理基金。

### **8. 再平衡频率**

- 原版策略建议每年一次再平衡，使各资产比例恢复到原定权重。
- 优化：可以根据波动范围设定触发再平衡的条件（如某类资产占比超过30%或低于20%时再调整），避免过度交易。

------

### **优化版示例组合**

以下是一个可能的优化版本：

- **股票（30%-40%）**：包括标普500、国际股票、新兴市场股票。
- **债券（25%-30%）**：长期国债+中期国债+TIPS。
- **黄金/大宗商品（15%-20%）**：黄金+白银或大宗商品指数基金。
- **现金/短期债券（10%-15%）**：高收益储蓄账户或短期债券基金。

------

### **总结**

优化后的永久投资组合仍保持了原版的核心特点——分散化和稳定性，但可以通过调整权重、引入多元资产和动态策略来提升收益潜力或适应特定市场环境。如果不确定如何调整，先从小幅改动开始，并随时间验证效果。









# 优化后的组合，收益和回撤大概有多少

优化后的永久投资组合收益和回撤表现取决于具体的调整方案以及所选资产的历史表现。以下以常见的优化方案为基础，结合历史数据进行大致评估：

------

### **1. 优化后的收益和回撤大致表现**

#### **优化组合假设：**

- **股票（35%）**：全球股票指数（MSCI全球指数+少量新兴市场）。
- **债券（30%）**：长期国债（20%）+中期国债（10%）。
- **黄金/大宗商品（20%）**：黄金（15%）+大宗商品指数（5%）。
- **现金（15%）**：短期债券基金或货币市场基金。

#### **历史数据（以过去50年为参考）：**

- **年化收益率**：7%-9%（相比原版略有提升，原版约6%-8%）。
- **最大回撤**：15%-18%（相比原版的10%-12%略高，但依旧低于纯股权组合的30%-50%）。
- **波动率**：8%-10%（原版约7%-9%）。

优化后的组合在增强股票和大宗商品权重的同时，收益有所提升，但回撤和波动略有增加。主要原因是股票和大宗商品的高波动性占比变大，而现金和债券的稳定性贡献减少。

------

### **2. 收益和回撤对比表**

| **组合类型**     | **年化收益率** | **最大回撤** | **波动率** |
| ---------------- | -------------- | ------------ | ---------- |
| 原版永久组合     | 6%-8%          | 10%-12%      | 7%-9%      |
| 优化后的永久组合 | 7%-9%          | 15%-18%      | 8%-10%     |
| 纯股权指数投资   | 9%-11%         | 30%-50%      | 15%-20%    |

优化后仍然是一种偏保守的策略，比纯股权投资风险小，但收益潜力稍弱。

------

### **3. 各类资产的贡献**

优化后的组合通过股票和大宗商品的提升，在经济扩张和通胀周期的收益更高；而债券和现金在经济下行时仍能提供对冲。具体来看：

- **股票**：贡献收益的主要来源，波动较大。提高股票权重增加了长期回报，但也提高了下行风险。
- **债券**：中长期国债在经济衰退时提供稳定回报，但在加息环境下表现不佳。
- **黄金/大宗商品**：对冲通胀和市场恐慌情绪，回报不稳定，但在危机期间（如2008年）表现突出。
- **现金**：流动性工具，拖累长期收益，但能在利率上升周期减少损失。

------

### **4. 回撤情景分析**

- 2008年金融危机

  ：

  - 优化组合回撤约15%-18%（原版约10%-12%，纯股权组合约40%-50%）。
  - 黄金和长期债券起到显著的对冲作用，减少组合的整体跌幅。

- 2020年疫情暴跌

  ：

  - 优化组合回撤约12%-15%（原版约8%-10%，纯股权组合约30%-35%）。
  - 由于黄金上涨和债券收益回升，组合在反弹中修复较快。

------

### **5. 长期视角的收益潜力**

优化后的组合在20-30年的时间尺度上，可能获得接近全球股市历史平均水平的回报（年化8%左右），同时降低极端市场条件下的风险。这对于寻求稳健增值的长期投资者是一个平衡收益与风险的好选择。

------

### **6. 风险和改进建议**

#### **潜在风险：**

- **通胀拖累**：现金部分长期表现不佳，在高通胀时期可能进一步拖累收益。
- **股票波动**：股票占比提高后，市场大跌时对整体组合的冲击可能加大。

#### **进一步改进建议：**

- **增加动态调整**：如在低估值时加大股票配置，高估值时提高债券权重。
- **分散股票投资**：增加小盘股、价值股的权重，平衡成长股的波动。
- **替代资产选择**：用通胀保值债券（TIPS）部分替代黄金或长期国债，在通胀周期表现更优。

------

**总结**：
 优化后的永久投资组合是一种更灵活、更适应现代市场环境的稳健策略。它适合希望获得较高长期收益、同时对回撤有一定承受力的投资者。在保持分散和安全的基础上，收益潜力略高于原版，但仍需根据自身风险承受能力和市场判断进行调整。
